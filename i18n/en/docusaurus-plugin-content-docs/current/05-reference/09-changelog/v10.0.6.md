# DesireCore v10.0.6

**Release Date**: 2026-03-09

## New Features
- **Unified Delegation Tool**: Introduced the `delegate` tool replacing three separate delegation tools, supporting sync, async, and handoff modes; async delegation runs tasks in background sub-sessions and automatically delivers results to the caller's next conversation
- **OpenClaw Import**: Import agents and skills from the OpenClaw platform with one click — scan, select via dual-tab interface, and track real-time import progress
- **Multi-Source Heartbeat**: A single agent can now have multiple heartbeat data sources, each with independent toggle controls and a management panel
- **Heartbeat Monitor Panel**: Agent detail page now shows heartbeat status cards, history records, and severity filtering, with consolidated reports after quiet hours
- **Registry Unification**: Registry directory structure unified to `entries/` format with Agent-driven installation, device-aware uninstall, and installation progress display

## Bug Fixes
- Fixed heartbeat scheduling sync issues: supports UUID-based schedule lookup, disabling now pauses instead of deleting, and panel updates in real-time after toggle changes
- Fixed CORS and cache invalidation issues in OpenClaw import
- Fixed hydration error caused by nested button elements
- Fixed registry entry format compatibility: properly derives legacy fields from the new format

## Improvements
- Delegated messages now show the sender agent's avatar and name in the target agent's chat, distinguishing human users from agent delegations
- CI release creation now uses idempotent mode to avoid conflicts during concurrent multi-job execution
