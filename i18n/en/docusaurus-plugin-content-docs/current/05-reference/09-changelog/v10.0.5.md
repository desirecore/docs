# DesireCore v10.0.5

**Release Date**: 2026-03-08

## New Features
- **Heartbeat System Enhancements**: Heartbeat checks now support timezone-aware scheduling, automatic retry for transient errors, a dual-layer suppression engine (hard/soft), and deep integration with the automation scheduler
- **SuperDoc Diff Panel**: SuperDoc now includes a Diff comparison panel with inline and side-by-side modes for easier document revision review
- **SuperDoc Immersive Mode**: SuperDoc supports full-screen immersive editing for focused content creation
- **Temperature Control**: Model temperature parameter is now fully connected across frontend and backend, allowing custom creativity levels per conversation; JSON output includes automatic repair for malformed responses
- **Mail Service Improvements**: Real-time unread count sync, automatic list refresh, and fetch error detection for a smoother email experience

## Bug Fixes
- Fixed text selection not working in packaged builds (inverted user-select strategy)
- Fixed apiFormat options being lost during editing and service compatibility issues, options are now data-driven
- Fixed immersive mode back button being unclickable
- Fixed email management component display issues in dark mode
- Fixed Electron drag region blocking SuperDoc toolbar button clicks
- Fixed unread count not refreshing when returning to email list
- Fixed IMAP UID range query errors

## Improvements
- Skills loading mechanism aligned with Claude Code's on-demand loading pattern, added S3 global skill
- Streamlined scheduler tool descriptions and cleaned up dead code in heartbeat templates
