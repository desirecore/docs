# DesireCore v10.0.5

**Release Date**: 2026-03-08

## New Features
- 心跳系统增强 — 时区支持、瞬态重试、抑制引擎与调度集成
- release notes 脚本增强与双语发布 (#198)
- temperature 打通 + JSON Repair + UI 修复 (#195)
- 邮件服务多项优化 - 未读计数、自动同步、fetch 错误检查
- SuperDoc 沉浸式模式实现
- SuperDoc 增加 Diff 对比面板（行内 + 并排模式）

## Bug Fixes
- CI fetch tags 加 --force 避免 tag 冲突 (#204)
- 授权显示问题、批量操作栏样式优化
- IMAPUID范围查询bug
- 修复 CI changelog 同步目标仓库错误与插入位置问题 (#200)
- 反转 user-select 策略，修复打包后文本无法选中 (#199)
- apiFormat 选项数据驱动化，修复编辑丢失与服务兼容性问题 (#196)
- 返回邮箱列表时刷新未读计数
- 修复沉浸式交互返回按钮无法点击
- 邮箱管理组件适配暗色模式 (#191)
- 修复 Electron 拖拽区域导致 SuperDoc/Cherry 工具栏按钮不可点击

## Improvements
- 精简调度工具描述 + 清理心跳模板死代码
- skills 按需加载对齐 Claude Code + 新增 s3 全局技能 (#202)
- CI changelog 改为子目录模式 (#201)
