# Naming Philosophy

**DesireCore** = **Desire** (will/wish) + **Core** (core)

---

## Desire: Will and Wish

"Desire" refers to **the user's intent and wishes**, which is the core focus of the product:

- The product's core philosophy is to transform the user's **intent, standards, boundaries, and preferences** into reusable behavioral norms
- The pain point of current AI is "intent fragmentation" — user experience and preferences cannot be沉淀, so AI remains just a "temporary worker without memory"

**Desire represents what the user wants — not just immediate requests, but long-term will and expectations.**

### Five Types of Core Assets

| Asset | Description |
|------|------|
| Persona | Who the Agent is — style, tone, value orientation |
