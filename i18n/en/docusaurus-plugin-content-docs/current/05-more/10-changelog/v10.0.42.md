# DesireCore v10.0.42

**Release Date**: 2026-04-28

![v10.0.42 更新概览](/img/changelog/v10.0.42-en.png)

## Fixes
- Front-end page refresh shortcuts have now been fully disabled in the desktop app to prevent accidental presses from unexpectedly refreshing the current screen or interrupting its state.
- Fixed an issue where the message list could scroll unstably in long conversations. When browsing message history, dragging the scrollbar, or switching back to a conversation, it is now less likely to jump, become misaligned, or lose the current position.

## Improvements
- **Performance optimization for long conversations**: Optimized message list rendering and virtualized loading for long conversation scenarios, making large conversations faster and smoother to open, switch between, and display on initial load.
- **Improved conversation preview experience**: Optimized the history preview and restore logic for long conversations, allowing you to more smoothly see recent content and continue from your previous browsing position when re-entering a conversation.
- **More stable message content rendering**: Complex messages, including long text, Markdown, and process messages, now display more smoothly, reducing the impact of rendering lag on the reading experience.
