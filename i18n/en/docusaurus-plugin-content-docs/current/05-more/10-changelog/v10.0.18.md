# DesireCore v10.0.18

**Release Date**: 2026-03-29

## New Features
- **Official launch of the desktop Agent operating system**: You can now create and manage agents, chat in natural language, call tools, use memory, extend skills, and coordinate across multiple services—all within a unified desktop application.
- **Agent marketplace and installation distribution**: You can browse agents and skill content in the marketplace and install them locally for immediate use.
- **Multi-model and compute resource management**: You can connect to and manage model services and compute nodes from multiple providers within the app, giving you more flexibility to choose the right model for different tasks.
- **Skills and Tools Center**: You can import, create, edit, and manage skills, MCP services, and tool configurations to expand your agents’ executable capabilities.
- **Long-term memory and knowledge resource system**: Agents now support combining long-term memory, knowledge content, and multiple types of resources to handle tasks, improving performance in ongoing conversations and complex workflows.
- **Approval and permission governance**: You can set up approval workflows and permission controls for high-risk operations, achieving a better balance between efficiency and security.
- **Mail Center**: You can connect your mailbox in DesireCore, view and search emails, compose replies, and handle attachments, allowing agents to participate in email-related work.
- **Automated scheduling and task orchestration**: You can create scheduled tasks and automated workflows so agents can carry out work on a plan and let you review execution history.
- **Code repository integration**: You can connect code repositories and view, manage, and assist with code-related tasks directly within the app.
- **Activity auditing and execution tracing**: You can view detailed records of how agents call tools, access services, and generate results, making it easier to trace execution.
- **Digital human interaction experience**: You can interact with agents through a digital human interface, voice playback, and real-time subtitles for a more immersive experience.
- **Team and relationship collaboration view**: You can manage team members, view relationship networks and collaboration status, and organize multi-agent collaboration more intuitively.
- **Documentation and versioning capabilities**: You can view diffs, export content, and manage versions in the built-in document interface, making it easier to preserve and reuse work成果.
- **Release notes and version update prompts**: The app now supports viewing version update details, helping you quickly understand what each upgrade brings.

## Improvements
- The overall experience of the conversation interface has been refined, making everyday workflows smoother, including streaming replies, message display, the input area, conversation history, and runtime status feedback.
- Agent profile pages, configuration pages, and import/publishing workflows are now more complete, making it more intuitive to create, edit, share, and sync agents.
- Search, filtering, and navigation capabilities have been enhanced, so you can find the content you need more quickly across conversations, resources, skills, and agents.
- The Settings Center is now more complete, with fuller desktop configuration options including appearance, language, notifications, shortcuts, privacy, and permissions.
- Core desktop app experiences such as windows, the title bar, notifications, screenshots, and environment checks have been filled out, bringing overall usability closer to that of mature desktop software.
- Multilingual content and interface copy are now fully covered, delivering a more consistent and complete experience in Chinese-language environments.
