# DesireCore v10.0.18

**Release Date**: 2026-03-29

![v10.0.18 更新概览](/img/changelog/v10.0.18.png)

## New Features
- **Update Notes Pop-up**: After downloading an app update, the app will now directly display the release notes and images for the versions you missed, with support for layered viewing by version and large image preview.
- **Team Organizational Chart**: A new "Team" view has been added to agents' social relationships, making it easier to visually inspect teams, team leads, members, ungrouped agents, and reporting relationships.
- **Enhanced Team Management**: You can now promote a team lead with one click in team settings, bulk add members, and bulk remove members, making team maintenance more efficient.
- **Tool Tag Interaction**: In agent profiles, hovering over a tool tag now shows its purpose, and clicking it takes you directly to the corresponding skill/tool management page.

## Fixes
- Fixed an issue where the "Available Tools" list in agent profiles was always empty. It will now correctly display the built-in and custom tools actually available to that agent.
- Fixed an issue where the default working directory was shared by all agents. Each agent now uses its own independent workspace and automatically connects to the shared directory of its team.
- Fixed an issue in the "Recent Activity" section of the runtime status panel where the duration of completed items would keep increasing, fail to appear before refresh, or show duplicates. Activity records and duration displays are now more accurate and stable.
- Fixed an issue with incorrect time display for background tasks. Completed tasks will no longer show continuously changing timers.
- Fixed an issue where the team organizational chart could crash under abnormal conditions such as when a team lead was deleted. The team view is now more stable.
- Fixed an issue where the privacy notice on the welcome page was hard to read against a light background. Key information is now easier to read.

## Improvements
- **Chain of Command Display**: The team view now includes a "User → DesireCore → Team Leads" chain-of-command hierarchy, making organizational relationships clearer. DesireCore is also no longer treated as a member that can join teams.
- In the conversation list, team leads are now surfaced in the general group for quicker access to key owners.
- The connector direction, styling, and legend in the team organizational chart have been optimized, making hierarchical relationships more intuitive than before.
- Team rules have been further tightened: the same agent can no longer serve as the lead of multiple teams at the same time, avoiding organizational configuration conflicts.
- The privacy notice on the welcome page has been rewritten into clearer bullet points and now explicitly states that logs are desensitized.
