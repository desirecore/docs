# DesireCore v10.0.7

**Release Date**: 2026-03-11

![v10.0.7 Release Overview](/img/changelog/v10.0.7-en.png)

## New Features
- NSIS 安装包启动时检测 Windows 版本，低于 Win10 提示不支持 (#232)
- WSL 环境检查改为按需触发（仅 Claude Agent SDK 运行时） (#231)
- describe 返回 commandDescriptors + invoke 改为通用引导
- Registry NodeInfo 透传 commandDescriptors
- Connector 解析 commandDescriptors + 发送 features
- 新增 commandDescriptorSchema 及 features/commandDescriptors 字段
- 将 Agent 活动状态指示器移至输入框上方显示
- 更新@azure/msal-node、优化群发邮件

## Bug Fixes
- 剥离 base64 图片 data URI 前缀，修复上游 API 500 错误 (#214)
- 修复环境检查 WSL/Linux 修复流程不可用的问题 (#228)
- 修复沉浸式交互返回按钮二次点击失效
- 智能体关机状态下拦截查询请求并隐藏输入框 (#230)
- 调度器增强 — 移除 node-cron + setTimeout 溢出修复 + 可选重叠保护 (#225)
- 编辑模型弹窗、图片发送、outlook授权再次优化
- 修复 PR #222 review 反馈的三个问题 (#223)
- release notes 双语修复 + 单实例提示对话框 (#222)
- 修复 release notes 生成的 GH_TOKEN 缺失、AI_MODEL 读取和 docs 英文 fallback 问题

## Improvements
- unify agent service client usage
- 合并 system.run + system.which 为 system.bash
