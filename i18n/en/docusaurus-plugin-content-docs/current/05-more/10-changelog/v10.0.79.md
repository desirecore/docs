# DesireCore v10.0.79

**Release Date**: 2026-06-28

![v10.0.79 更新概览](/img/changelog/v10.0.79-en.png)

## New Features
- **Official Compute Model Access and Consumption Display**: Restored access to official compute models. The model selector now shows a comprehensive consumption multiplier badge and unit price details compared to the default model, with capability tags automatically collapsing when there are too many.
- **Conversation Model Smart Filtering**: In the conversation model selection list and resource management panel, image/video generation models are now automatically disabled with a prompt to prevent conversation issues caused by accidental selection.

## Fixes
- **Streaming Response Stability**: Fixed an issue where streaming conversations through some compatible proxies could crash the app due to prematurely closed connections. Such truncations now correctly display an error prompt and automatically retry.
- **Team Remote Repository Dialog**: Fixed the team remote repository connection dialog where the background displayed abnormally and some fields were untranslated, and refined the closing interaction experience.
