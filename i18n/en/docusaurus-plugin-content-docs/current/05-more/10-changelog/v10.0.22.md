# DesireCore v10.0.22

**Release Date**: 2026-03-31

![v10.0.22 更新概览](/img/changelog/v10.0.22-en.png)

## Fixes
- In Windows environments, `.md` files in AI responses can now be correctly displayed as clickable links and opened properly, avoiding display issues or false matches caused by path parsing errors.
- When importing skills on Windows and Linux, the file selection method has been adapted to system limitations. You can now choose either an `.md` file or a skill folder to complete the import.
- During automatic updates, duplicate prompts will no longer pop up repeatedly, and the app will no longer keep rechecking while downloading updates, resulting in a more stable and smoother update experience.

## Improvements
- Button display in the SuperDoc toolbar is now more stable on narrow screens, preventing the action bar from being squeezed and making horizontal viewing smoother.
