# DesireCore v10.0.78

**Release Date**: 2026-06-16

![v10.0.78 更新概览](/img/changelog/v10.0.78-en.png)

## Fixes
- Fixed an issue on Windows where the app could fail to launch when opened via `dc://` deep links, desktop/Start menu shortcuts, or associated files, making app startup more reliable.
- When generating Markdown documents with the Write tool, new documents will now open automatically only after the entire round of content has been written, avoiding the display of unfinished partial drafts.
- SuperDoc will temporarily disable diff accept/reject actions while an agent is editing a document in streaming mode, preventing content from being written back too early and causing subsequent edits to become inconsistent.

## Improvements
- SuperDoc will continue to display real-time diff changes while a document is being generated in streaming mode, so you can clearly track content modifications even while waiting for generation to finish.
