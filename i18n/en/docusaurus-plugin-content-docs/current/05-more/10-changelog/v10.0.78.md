# DesireCore v10.0.78

**Release Date**: 2026-06-28

![v10.0.78 更新概览](/img/changelog/v10.0.78-en.png)

## Fixes
- Fixed an issue where the application might fail to launch on Windows when opened via `dc://` protocol links, desktop shortcuts, or the Start menu.
- Fixed an issue where the editor would pop up too early and display incomplete content while AI was writing a document. It now waits until the full reply is completed before opening automatically. Also fixed an issue where the accept/reject buttons in the document diff view could end editing prematurely and cause content corruption during AI generation. These buttons are now automatically hidden during streaming.
