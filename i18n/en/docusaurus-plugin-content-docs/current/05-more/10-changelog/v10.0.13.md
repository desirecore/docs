# DesireCore v10.0.13

**Release Date**: 2026-03-19

![v10.0.13 Release Overview](/img/changelog/v10.0.13-en.png)

## Fixes
- Sub-agents executing delegated tasks will no longer be interrupted early by the default 10-call tool usage limit. Complex tasks can now continue running until completion, while the 5-minute timeout safeguard remains in place.
