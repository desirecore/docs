# DesireCore v10.0.84

**Release Date**: 2026-07-13

![v10.0.84 更新概览](/img/changelog/v10.0.84-en.png)

## Fixes
- Fixed an issue where the built-in runtime was rejected on macOS due to notarization scanning, and could be falsely flagged by antivirus software on Windows: the built-in runtime is now encrypted and packaged to ensure it works out of the box without relying on online downloads.
