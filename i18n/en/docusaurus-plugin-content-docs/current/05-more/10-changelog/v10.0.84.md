# DesireCore v10.0.84

**Release Date**: 2026-07-13

## New Features
- **Runtime Encrypted Packaging**: The built-in runtime is packaged in an encrypted format. On macOS, the built-in runtime can be used directly upon first launch without an online download, and it has passed Apple notarization. On Windows, this also reduces the risk of false positives from antivirus software.

## Fixes
- Fixed an issue where large temporary files could be left behind and consume disk space if a file rename failed during runtime decryption.
