# DesireCore v10.0.80

**Release Date**: 2026-06-28

## Fixes
- **Duplicate messages after restart following retry**: Fixed an issue where manually retrying a message and then closing and reopening the client would cause the same message to appear duplicated.
- **Unstable local user identity**: Fixed an issue where the local user profile ID was incorrectly switched during login, logout, or session invalidation, causing user data to be scattered across multiple directories.

## Improvements
- **Save API quota**: When retrying or editing a message, the system automatically cancels the background request of the previous session to avoid further unnecessary consumption of the API quota.
