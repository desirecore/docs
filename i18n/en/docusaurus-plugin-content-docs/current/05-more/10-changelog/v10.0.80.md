# DesireCore v10.0.80

**Release Date**: 2026-06-27

![v10.0.80 更新概览](/img/changelog/v10.0.80-en.png)

## Fixes
- After retrying a message and restarting the client, the message no longer appears multiple times; when retrying or editing a message, old background requests are now automatically aborted to avoid wasting conversation turns.
- Fixed an issue where logging in, logging out, or session expiration incorrectly created a new local user profile folder, ensuring user data is always associated with a stable local identity.
