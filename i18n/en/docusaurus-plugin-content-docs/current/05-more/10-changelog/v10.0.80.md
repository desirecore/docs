# DesireCore v10.0.80

**Release Date**: 2026-06-28

## Fixes
- Fixed an issue where retried messages could appear twice after restarting the client, and optimized automatic cancellation of old background requests during retries to reduce token consumption.
- Fixed an issue where logging in, logging out, or session expiration could cause duplicate creation of the local user configuration directory. The local user identity is now more stable.
