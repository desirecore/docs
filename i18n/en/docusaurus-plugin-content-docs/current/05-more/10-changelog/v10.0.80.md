# DesireCore v10.0.80

**Release Date**: 2026-06-28

## Fixes
- Fixed an issue where restarting the client after retrying messages caused duplicate messages to appear, and optimized the retry mechanism to reduce unnecessary API token consumption.
- Fixed an issue where the local user identity could drift and result in redundant user data directories when logging in, logging out, or when a session expired; local user configuration now remains stable.

## Notes
> This update primarily addresses two issues related to chat retry and user identity, and refines the underlying build environment for the release process to ensure stable operation of the desktop client.
