# DesireCore v10.0.80

**Release Date**: 2026-06-28

## Fixes
- **Message Retry**: Fixed an issue where retrying a message and then restarting the client caused the same message to appear multiple times. Also optimized request handling during retries and edits to automatically cancel stale requests, saving tokens.
- **User Identity**: Fixed an issue where the local user identity could unexpectedly change after logging in, logging out, or when a session expired, ensuring the user data directory remains stable and preventing duplicate local data folders from being created.
