# DesireCore v10.0.80

**Release Date**: 2026-06-28

![v10.0.80 更新概览](/img/changelog/v10.0.80-en.png)

## Fixes
- Fixed an issue where retried messages would reappear after retrying them and restarting the client. Also optimized backend requests during manual retries and message edits to reduce unnecessary token consumption.
- Fixed an issue where the local user identity would incorrectly switch during cloud login, logout, or session expiration, ensuring user data is always saved in a fixed local directory.
