# DesireCore v10.0.23

**Release Date**: 2026-04-01

![v10.0.23 更新概览](/img/changelog/v10.0.23-en.png)

## New Features
- **ZIP Agent Import**: In addition to folders, you can now directly select ZIP archives when importing agents, making the import process more convenient.

## Fixes
- Fixed an issue on Windows where data exports could produce incomplete archives due to files being in use, preventing successful restoration.
- Fixed an issue where data imports could overwrite cache or runtime files, affecting the current running state.
- Fixed an issue where some import failure messages were unclear or inaccurate, making it easier to identify the cause when an import exception occurs.

## Improvements
- Optimized the contents of data exports by automatically excluding cache, logs, and other regenerable files, significantly reducing backup package size and making transfers and storage more efficient.
- Improved integrity checks for exported backups, reducing the risk of only discovering a corrupted archive after it has been generated and found unusable.
