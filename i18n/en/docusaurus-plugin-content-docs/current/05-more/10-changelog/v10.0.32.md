# DesireCore v10.0.32

**Release Date**: 2026-04-15

![v10.0.32 更新概览](/img/changelog/v10.0.32-en.png)

## Fixes
- Fixed issues during Agent delegation handoff where background information from user messages could be lost in the target conversation, messages could be swallowed, or the source Agent’s streamed content might not be fully preserved. Now, task instructions, context, and generated content before and after the handoff are displayed more reliably.
- Fixed an issue where conversation order could become incorrect after refreshing or loading message history. Messages are now correctly sorted by their actual timestamps, and new content in long conversations will no longer appear among older messages.
- Fixed an issue in the compute service where model capability labels were read and displayed inaccurately. Manually updated capability configurations now remain consistent across model details, edit dialogs, and capability validation for features such as image support.
- Fixed an issue where some text color hierarchy in the Notification Center was inconsistent. Time information and filter tags are now clearer and easier to read across different themes.

## Improvements
- Optimized the overall color scheme and readability of the Mail Center. Information hierarchy is now clearer across the mail list, details, preview, quote blocks, and editor toolbar, while read avatars and error states are also easier to distinguish.
