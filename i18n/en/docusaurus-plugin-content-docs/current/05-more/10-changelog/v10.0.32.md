# DesireCore v10.0.32

**Release Date**: 2026-04-16

![v10.0.32 更新概览](/img/changelog/v10.0.32-en.png)

## Fixes
- Fixed an issue in handoff scenarios where messages were displayed incompletely. The source agent’s streamed content before the handoff, tool call cards, and the contextual information received by the target agent are now all fully preserved and displayed correctly.
- Fixed an issue where messages could occasionally go missing or appear out of order after a handoff and page refresh. Conversation messages are now stably sorted based on their actual message timestamps, and historical messages missing timestamps can also be displayed in the correct position.
- Fixed an issue in the compute service where manually edited model capability labels could appear inconsistent with actual detection. Custom capability configurations are now displayed correctly and accurately recognized in scenarios such as sending images.
- Fixed an issue where some text color hierarchy in the Notification Center was inconsistent. Notification timestamps and filter labels are now more readable across different color themes.

## Improvements
- Optimized the overall color scheme and text hierarchy of the Mail Center. Timestamps, addresses, summaries, and read avatars in the mail list, detail view, and preview are now clearer and easier to read.
- Improved interface consistency in the mail composer and quick reply area. The visual style of the toolbar, quote blocks, and error messages is now more unified.
