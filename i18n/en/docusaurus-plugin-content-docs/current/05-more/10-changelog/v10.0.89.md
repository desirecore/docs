# DesireCore v10.0.89

**Release Date**: 2026-07-17

## New Features
- File access whitelist changed to per-agent opt-in (no interception by default) (#1212)
- Add checkpoint configuration and cleanup governance (#1204)

- Fix relative path Write being counted toward both recovered artifacts and uncovered warnings (#1213)

- Unblock startup Keychain sync probing + self-healing local certificate validation (#1209)
- Outbound tool schema always explicitly completes the top-level required array (#1208)
- Fix runtime tool permission lifecycle (#1205)
