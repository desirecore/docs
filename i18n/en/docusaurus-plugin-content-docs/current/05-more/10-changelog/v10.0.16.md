# DesireCore v10.0.16

**Release Date**: 2026-03-24

![v10.0.16 Release Overview](/img/changelog/v10.0.16-en.png)

## New Features
- **Cloud Publishing**: You can now publish an Agent directly from within the app to a linked GitHub, Gitee, or Gitea repository with one click, and view the repository link, sync status, and version history.
- **What-You-See-Is-What-You-Get Markdown Editing**: The Markdown editor now defaults to a more complete WYSIWYG mode, with richer visual editing capabilities and a Ribbon toolbar.
- **Upgraded Delegation Collaboration Experience**: Agent task delegation now supports clearer task handoff, asynchronous result callbacks, and temporary subtask Workers, making multi-Agent collaboration more intuitive.

## Fixes
- After sending an image, the model can now recognize image content more reliably, distinguish between new and previous images when multiple images are sent in succession, and images will no longer disappear after refreshing.
- Fixed an issue on the social relationship page where Agent online status was displayed inaccurately; it now stays consistent with the conversation list and updates in real time.
- Fixed an issue on Windows/Linux where the Markdown editor toolbar overlapped with window control buttons in fullscreen mode, and added draggable regions.
- Fixed an issue where delegation cards had unclear borders in the light theme; avatar display and subtask status presentation are now also more accurate.
- Fixed various authentication and connection compatibility issues in cloud publishing and syncing, including cases where the first publish did not actually push, or syncing failed when an old connection became invalid or was missing.

## Improvements
- The Markdown editor now automatically follows the app’s global theme, providing a more consistent interface when switching between light and dark modes.
- Delegation results are now automatically written back to the original conversation and status updates appear in real time; you will also receive desktop notifications upon completion, making task follow-up easier.
- The cloud sync button now intelligently chooses whether to publish updates directly or sync first before publishing based on the current state, reducing unnecessary steps.
