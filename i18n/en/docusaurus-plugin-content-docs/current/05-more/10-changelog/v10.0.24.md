# DesireCore v10.0.24

**Release Date**: 2026-04-06

![v10.0.24 更新概览](/img/changelog/v10.0.24-en.png)

## New Features
- **Error Log Panel**: A new "Error Logs" tab has been added to Activity Records, where you can centrally view system error statistics, filter errors by level, and browse historical issues with pagination.
- **Tool Search**: AI agents can now use a new tool search capability to find available tools more quickly and view complete parameter descriptions, improving tool selection efficiency in complex tasks.

## Fixes
- Relative `.md` links in Agent replies are now correctly resolved based on the corresponding working directory, making 404 errors less likely when clicking document links.
- Fixed issues caused by automatic retries, including duplicate retries, incorrect error classification, and unexpected interruptions, making conversations more stable during rate limits, network fluctuations, or upstream service failures.
- Fixed an issue in WYSIWYG editing mode where dragging to adjust width could trigger abnormal split-screen behavior, resulting in a more consistent editor layout.

## Improvements
- The Grep search tool can now search hidden files, automatically exclude version control directories, correctly display context content, and provide clearer count, sorting, and pagination information, making code and file search results more accurate and easier to read.
- Retry status messages in conversations have been refined, so when rate limits, network issues, or temporary service unavailability occur, you can see clearer reasons for the wait.
- Underlying system dependencies and security have been updated, improving overall runtime stability, compatibility, and security protection.
