# DesireCore v10.0.21

**Release Date**: 2026-03-31

![v10.0.21 更新概览](/img/changelog/v10.0.21-en.png)

## Fixes
- Fixed an issue on Windows where abnormal resolution of the user directory path could cause Agent handoff, routing, and Hook execution to fail.
- Fixed an issue where the HttpRequest tool could sometimes throw errors or behave unexpectedly when handling redirects, making related network requests more stable and reliable in Windows environments.
- Fixed intermittent network request errors across multiple scenarios, including code repositories, web scraping, web search, Ollama synchronization, and tool downloads, reducing issues such as “the request ended but no content was returned.”
