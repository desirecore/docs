# DesireCore v10.0.34

**Release Date**: 2026-04-17

![v10.0.34 更新概览](/img/changelog/v10.0.34-en.png)

## Fixes
- When an error occurs midway through a streaming reply, the error message will now reliably appear after the current AI response, instead of overwriting the reply content, missing the prompt, or appearing out of order.
- After refreshing or restarting the app, the order of error messages in conversations will remain correct and will no longer fall back below the user message.
- The core agent can now explicitly specify a team leader when creating a team, avoiding creation failures caused by using itself as the default identity.
- Fixed an issue in light mode where the “Context Isolated” divider was almost invisible, making the separator in the message stream clearer.
- Optimized the header layout of the tool execution area so that when multiple tool names and file badges appear at the same time, the title no longer gets squeezed into awkward line breaks.

## Improvements
- When creating a team, the explanation of team leader restrictions is now clearer, making related actions easier to understand when filling in parameters and using the feature.
