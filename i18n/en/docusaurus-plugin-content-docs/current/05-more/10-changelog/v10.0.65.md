# DesireCore v10.0.65

**Release Date**: 2026-05-19

![v10.0.65 更新概览](/img/changelog/v10.0.65-en.png)

## New Features
- **Chat formula rendering**: Markdown in chat messages now supports both inline and block-level formulas, with mathematical content displayed in a clearer, more readable layout.

## Fixes
- Fixed an issue where some agent tool-calling scenarios would incorrectly report a "tool concurrency issue" and cause requests to fail. Complex multi-turn tool interactions are now more stable.
- Fixed inaccurate error messages related to tool calls. When request format errors occur, the upstream reason is now displayed more faithfully, making troubleshooting easier.

## Improvements
- Improved the display of formulas inside chat bubbles. Long formulas are now smoother to browse within messages and less likely to overflow the layout.
