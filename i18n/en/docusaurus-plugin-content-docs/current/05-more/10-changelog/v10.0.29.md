# DesireCore v10.0.29

**Release Date**: 2026-04-13

![v10.0.29 更新概览](/img/changelog/v10.0.29-en.png)

## New Features
- **Team Import/Export**: You can now directly import or export an entire team from multiple entry points, and choose exactly what to transfer by category, such as agents, skills, memory, and workspaces.
- **Automatic Update Checks for Built-in Skills**: Built-in skills now automatically check Market for new versions every 10 minutes, so you can get the latest updates without restarting the app.
- **Copy Audit Details**: When viewing system instructions on the audit details page, you can now copy the corresponding content with one click for easier troubleshooting, record-keeping, and reuse.

## Fixes
- Fixed issues in chats where messages could be lost, duplicate separators could appear, or message order could become incorrect during retry, resend after editing, or message merge scenarios, making chat history display more stable and reliable.
- When the model service encounters rate limiting, gateway errors, or temporary network fluctuations, the system now retries more accurately and shows clearer error reasons and retry status.
- Fixed an issue where messages could still be sent while images were uploading, which could cause images to be lost, making send timing more reliable.
- Fixed an issue where image and file references were not recognized correctly in some environments; image content can now be passed to the model more reliably as visual context.
- Fixed an issue on Windows where exported ZIP files could become corrupted or invalid files could be left behind after an export failure, making the export process more reliable.
- Fixed inconsistent recognition of agent packages and team packages at import entry points; if you choose the wrong entry point, the system will now provide clearer guidance.
- Fixed an issue where the list did not refresh in time after importing a team, so imported content now appears more quickly and accurately after completion.
- Fixed a UI issue that could trigger exceptions when updating skills, making the skill update process more stable.
- Fixed some exceptions in topic completion and history processing, so a failure in a single topic no longer affects continued processing of other content.
- Fixed the invalid operation of setting the DesireCore core agent as a team leader, preventing incorrect team structure configuration.

## Improvements
- User message cards now support copy and read-aloud, making it more convenient to review and reuse content you have sent.
- File attachments in chats can now be opened directly, and supported built-in formats will be displayed using more suitable document or code viewing methods.
- The “Needs Your Confirmation/Response” card no longer has a wait time on the Skip button, and button visibility has been improved for more direct interaction.
- Proactive agent notifications and user messages sent to you are now displayed in a way that better matches trigger order, making conversation flow feel more natural.
- Global built-in skills have been adjusted to participate in calls only when needed, reducing irrelevant interference and helping agents stay more focused when executing tasks.
