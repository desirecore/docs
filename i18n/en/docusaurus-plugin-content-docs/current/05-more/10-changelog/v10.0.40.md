# DesireCore v10.0.40

**Release Date**: 2026-04-28

![v10.0.40 更新概览](/img/changelog/v10.0.40-en.png)

## Fixes
- Fixed an issue in multi-turn conversations where follow-up questions could get stuck on “Thinking” and fail to continue; subsequent replies now return more reliably and normally.
- Fixed retry handling for streaming request failures caused by network disconnections, timeouts, and similar issues. Error messages after failures are now more accurate, and conversations resume more smoothly once the network is restored.
- Fixed compatibility issues when integrating custom models. When using some domestic model providers, output length limits and reasoning modes now take effect more accurately.

## Improvements
- Optimized compatibility handling for model requests. Compatible APIs such as Kimi now behave more stably and align better with the actual capabilities of the corresponding providers when deep thinking is enabled or disabled.
