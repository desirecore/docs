# DesireCore v10.0.41

**Release Date**: 2026-04-28

![v10.0.41 更新概览](/img/changelog/v10.0.41-en.png)

## New Features
- **Alpha Internal Testing Channel**: Added a new Alpha preview update channel in Settings. When switching to it, a two-step risk confirmation will be required, helping you try more cutting-edge versions after fully acknowledging the risks.

## Fixes
- Fixed an issue where some model services using separated API endpoints could encounter errors when calling media capabilities. Services like MiniMax, where chat, image, voice, and video APIs are separated, can now use the relevant generation features normally.
- Fixed an issue where some basic parameters were not synchronized after upgrading preset model service configurations. Updates to service addresses, API formats, display names, and similar settings can now be correctly applied to local configurations.
- Fixed an issue where configurations that included media API endpoints could fail to load and trigger API errors.
- Fixed an issue where some music generation services were displayed incorrectly in the interface. These capabilities will now be shown under the correct names and categories.
