# DesireCore v10.0.20

**Release Date**: 2026-03-30

## Fixes
- Fixed an issue where, when creating an agent from the plus menu, the popup could remain stuck on “Creating agent” without any success or failure feedback.
- Fixed an issue where, when using a streaming model through a compatible proxy, the input Token and output Token statistics in the audit details often displayed as 0 or were inaccurate.
- Fixed an issue where the audit details could not correctly record input Tokens when some gateways passed through Anthropic protocol responses directly.

## Improvements
- **Updated the update check experience**: Optimized the desktop update check mechanism so that it can return results more quickly and automatically fall back when the network is unstable or the service is unavailable, reducing update check failures and long waits.
