# DesireCore v10.0.72

**Release Date**: 2026-06-03

![v10.0.72 更新概览](/img/changelog/v10.0.72-en.png)

## Fixes
- Fixed an issue where the app could report an error on startup in some cases, preventing exceptions during computing service initialization from affecting normal system entry.
- Fixed an issue in Resource Manager where the "AI Services" list could incorrectly display official cloud computing resources. It now shows only locally configured available computing resources.
