# DesireCore v10.0.62

**Release Date**: 2026-05-19

![v10.0.62 更新概览](/img/changelog/v10.0.62-en.png)

## Fixes
- Fixed an issue where the Agent failed to keep the cloud version when handling Git conflicts involving files that had already been deleted remotely. These conflicts can now be resolved correctly and the commit process can continue.
- Fixed an issue where untracked files were mistakenly identified as merge conflicts, preventing an erroneous conflict resolution window from appearing in the conversation.
- Fixed an intermittent issue where automatic commits could fail after conflict resolution, preventing errors caused by deleted files being repeatedly added to the commit.
- Fixed an intermittent error that could occur when closing the conflict comparison interface, making the conflict handling process more stable.

## Improvements
- Optimized how certain dependencies are loaded in desktop and Web modes, reducing exceptions during startup or runtime caused by missing component resources.
