# Installation & Quick Start

## Requirements

- **Node.js** >= 18.0
- **pnpm** >= 10.0
- **Git**

## Install

```bash
# Clone the project
git clone https://github.com/desirecore/desirecore.git
cd desirecore

# Install dependencies
pnpm install
```

## Start

```bash
# Desktop app (Electron)
pnpm dev

# Web only
pnpm dev:web
```

Once started, you can begin interacting with your AI Companion.

## Build

```bash
# macOS
pnpm build:mac

# Windows
pnpm build:win

# Linux
pnpm build:linux

# Web
pnpm build:web
```
