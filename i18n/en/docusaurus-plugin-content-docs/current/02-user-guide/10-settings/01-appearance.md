# Appearance Settings

DesireCore supports three theme modes. You can switch between them in settings anytime to find the most comfortable visual experience.

## Opening Appearance Settings

1. Click the **gear icon** in the bottom-left corner to open the settings panel
2. Find the **Appearance** section

## Theme Modes

DesireCore offers three theme modes:

| Mode | Description |
|------|-------------|
| **Follow System** | Automatically matches the operating system's light/dark settings. When the system switches to dark mode, DesireCore will also darken. This is the default option. |
| **Light** | Always uses the light interface, suitable for well-lit environments. |
| **Dark** | Always uses the dark interface, suitable for nighttime or low-light environments, reducing eye strain. |

## How to Switch Themes

In appearance settings, click the dropdown menu to the right of **Theme** and select your desired mode. The switch takes effect immediately without restarting the app.

:::tip
If your usage environment differs significantly between day and night, we recommend using "Follow System" mode. This way, you only need to set automatic switching times at the system level, and DesireCore will automatically adapt.
:::

## Design Style

Regardless of which theme you choose, DesireCore adopts a Glassmorphism design language, with interface elements featuring semi-transparent and blur effects to create a sense of depth and modernity.

The glassmorphism effect is more pronounced in dark mode, suitable for immersive use.
