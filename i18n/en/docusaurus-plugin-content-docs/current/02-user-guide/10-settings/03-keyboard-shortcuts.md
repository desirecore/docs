# Keyboard Shortcuts

DesireCore provides rich keyboard shortcuts to help you operate efficiently. You can view all shortcuts and customize them according to your personal habits.

## Opening Keyboard Shortcut Settings

1. Open the **Settings** panel
2. Click **Keyboard Shortcuts** in the **Shortcuts** section
3. Enter the keyboard shortcut management sub-panel

## Editor Shortcut Modes

DesireCore has three built-in editor shortcut modes to adapt to different users' operating habits:

| Mode | Description | Suitable For |
|------|-------------|--------------|
| **Native Mode** | Retains each editor's built-in shortcuts without any overrides. This is the safest option. | Default choice when unsure |
| **Unified Mode** (Recommended) | Uses DesireCore's unified shortcut system for consistent key experience across all editors. | Users who frequently switch between multiple editors |
| **Custom Mode** | Fully takes over shortcuts, allowing you to freely modify all key bindings. | Advanced users |

## Common Shortcuts

### Conversation Input

| Function | Shortcut |
|----------|----------|
| Send Message | `Enter` |
| New Line | `Shift + Enter` |
| Cancel Input | `Esc` |

### Super Document

| Function | Shortcut |
|----------|----------|
| Accept Changes | `Cmd/Ctrl + Enter` |
| Reject Changes | `Cmd/Ctrl + Backspace` |
| Next Change | `Cmd/Ctrl + ]` |
| Previous Change | `Cmd/Ctrl + [` |

### Editor General

| Function | Shortcut |
|----------|----------|
| Save | `Cmd/Ctrl + S` |
| Save As | `Cmd/Ctrl + Shift + S` |
| Find | `Cmd/Ctrl + F` |
| Replace | `Cmd/Ctrl + H` |
| Go to Line | `Cmd/Ctrl + G` |
| Full Screen | `F11` |

### Application Global

| Function | Shortcut |
|----------|----------|
| Show/Hide Window | Customizable |
| Screenshot | Customizable |
| Undo | `Cmd/Ctrl + Z` |
| Redo | `Cmd/Ctrl + Shift + Z` |

:::tip
Shortcuts marked "Customizable" in the table above need to be manually bound in keyboard shortcut settings. macOS uses `Cmd`, Windows/Linux uses `Ctrl`.
:::

## Customizing Shortcuts

1. In the keyboard shortcut management panel, find the shortcut you want to modify
2. Hover over the item and click the **Edit** button
3. Press your desired new key combination
4. If it conflicts with an existing shortcut, the system will prompt the conflicting item
5. Click **Save** to confirm the modification

You can also use the search box at the top to quickly locate specific shortcuts.

## Resetting Shortcuts

If you want to restore default settings after customization, click the **Reset All** button at the top of the panel to restore all shortcuts to their default values.

:::warning
Resetting will clear all custom shortcut bindings and cannot be undone. Please confirm before proceeding.
:::
