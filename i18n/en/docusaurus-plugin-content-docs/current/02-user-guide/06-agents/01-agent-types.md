# Agent Types

Agents are the core of DesireCore. Each Agent has its own personality, memory, and skills, working for you like a true professional assistant.

## General Agents vs. Specialized Agents

Agents in DesireCore are divided into two categories:

### General Agents

General Agents have no specific domain restrictions and can handle various daily tasks:

- Answering questions, organizing materials
- Writing and polishing documents
- Code assistance and technical Q&A
- Schedule planning and information retrieval

**DesireCore Core Agent** is a General Agent—it's your default assistant, ready to use out of the box.

### Specialized Agents

Specialized Agents are deeply optimized for specific domains, with professional knowledge, behavioral guidelines, and exclusive skills in that field:

| Domain | Example | Characteristics |
|---|---|---|
| Legal | Legal Advisor | Familiar with legal provisions, rigorous wording, risk warnings |
| Real Estate | Property Consultant | Market knowledge, contract review, investment analysis |
| Translation | Translator | Multilingual proficiency, context awareness, terminology consistency |
| Writing | AI Scribe | Document structuring, style adaptation, version management |
| Data | Data Analyst | Python operations, visualization, insight extraction |

## Pre-built Agents Overview

DesireCore comes with multiple pre-built Agents that you can use directly or install more from the marketplace. Pre-built Agents are categorized by use case:

| Category | Agents | Use Cases |
|---|---|---|
| Productivity | Project Assistant, Schedule Manager | Project management, task tracking |
| Development | Programmer, Code Reviewer | Coding assistance, bug troubleshooting |
| Communication | Translator, Business Writer | Translation, email writing |
| Learning | Learning Coach, Knowledge Organizer | Learning planning, note organization |
| Creative | Copywriter, Brainstorming | Copywriting, inspiration generation |

:::info Agents Grow
Each Agent learns your preferences and habits through conversation. The more you use it, the more it understands you. These "growths" are saved as memories, which you can view and manage at any time.
:::

## How to Choose the Right Agent

Simple principles for choosing an Agent:

1. **Daily miscellaneous tasks** → Use DesireCore Core Agent (General type)
2. **Professional domain tasks** → Install or create a Specialized Agent from the marketplace
3. **Specific workflows** → Create a custom Agent with exclusive skills and behavioral guidelines

:::tip Not sure which one to use?
Just ask DesireCore Core Agent: "I need an assistant to help me review contracts," and it will recommend a suitable Agent or help you create one.
:::
