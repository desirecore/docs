# First Conversation

Now that your API Key is configured, let's have your first conversation with your AI companion!

## Select an Agent

In the conversation list, you'll see a pre-installed agent — **DesireCore**. It's your general-purpose AI assistant, capable of handling a wide range of everyday tasks.

Click its name, and the chat area on the right will switch to its conversation interface.

<!-- Screenshot placeholder: Select agent (select-agent.png) -->

:::info About the DesireCore Core Agent
DesireCore is the built-in general-purpose agent that comes with the app, featuring a blue circular avatar. It can handle tasks directly or delegate them to more specialized agents when needed.
:::

## Send Your First Message

1. Click the input box at the bottom
2. Type what you'd like to say, for example:

   > Hello! Can you briefly introduce yourself?

3. Press **Enter** to send (or click the green send button on the right)

<!-- Screenshot placeholder: Send message (send-first-message.png) -->

:::tip Quick Actions
- **Enter**: Send message
- **Shift + Enter**: New line (without sending)
:::

## Waiting for a Reply

After sending, you'll see a "DesireCore is thinking" indicator. After a few seconds, the agent's reply will appear in the conversation.

Replies are displayed as left-aligned bubbles, while your messages appear on the right.

## Understanding Reply Formats

DesireCore's agents don't just reply with text — they may also display different types of content cards:

| Card Type | Appearance | Meaning |
|-----------|-----------|---------|
| Regular Reply | White bubble | The agent's text reply |
| Action Receipt | Green/blue/orange card | The agent performed an action and returned results |
| Teaching Confirmation (Learned) | Purple card | The agent memorized a rule you taught it |
| Delegation Notification (Delegate) | Blue card | The agent delegated the task to another specialized agent |

When you first start using DesireCore, you'll mainly see regular text replies. As you go deeper — teaching it how to do things and assigning tasks — you'll encounter the richer card types.

## Try More Conversations

Here are some useful things you can try with DesireCore:

- **Write an email**: "Help me write a meeting invitation email for a Friday afternoon project retrospective"
- **Analyze a problem**: "I'm considering changing jobs — please help me create a pros and cons analysis"
- **Learn something**: "Explain what RAG (Retrieval-Augmented Generation) is in simple terms"
- **Translate content**: "Translate this into French: The weather is lovely today, perfect for a walk outside"

## Complete Flow Recap

```
Select agent → Type in the input box → Press Enter to send → Wait for reply → Continue the conversation
```

That's it! DesireCore's core experience is natural conversation — you talk, it acts. The more you use it, the better it understands your preferences and habits.
