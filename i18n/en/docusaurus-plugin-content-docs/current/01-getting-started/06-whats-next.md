# What's Next

Congratulations on completing the getting started guide! You can now have normal conversations with your AI companion. Next, choose an exploration path that suits your needs.

## Choose a Path

### Want to learn the interface

If you'd like a deeper understanding of DesireCore's interface and functional areas:

- [Interface Navigation](../02-user-guide/01-interface/01-layout-overview.md) — Detailed overview of every panel, button, and interaction

### Want to teach your agent

DesireCore's core feature is "teaching" your AI companion. You can define rules and set preferences through conversation:

- [Teach Your Agent](../02-user-guide/03-teaching/01-delegation-interaction.md) — Learn how to train your AI companion through conversation
- [Delegate Tasks](../02-user-guide/04-delegation/01-giving-tasks.md) — Learn how to let agents execute tasks autonomously

### Want to accomplish specific tasks

If you have concrete tasks you'd like to complete, check out these use cases:

- [Use Cases](/use-cases) — Rich real-world examples covering daily office work, writing, analysis, and more

### Want to manage and customize agents

If you'd like to create new specialized agents or manage your existing companions:

- [Agent Management](../02-user-guide/06-agents/01-agent-types.md) — Create, configure, and manage your AI companions
- [Capabilities & Tools](../02-user-guide/09-capabilities/01-tool-system.md) — Explore the tools and skills available to agents

### Want to understand the philosophy

If you're interested in DesireCore's design philosophy and core concepts:

- [Concepts](../04-concepts/index.md) — Deep dive into AgentFS, delegation-based interaction, the receipt system, and other core concepts

### Want to adjust settings

If you'd like to customize the appearance, manage data, or configure more AI providers:

- [Settings](../02-user-guide/10-settings/01-appearance.md) — Application settings, compute services, data management
