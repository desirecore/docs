# Install DesireCore

The DesireCore desktop client is available for macOS, Windows, and Linux.

## Download

Head to the DesireCore website and click the download button to get the latest version:

> **Download**: [https://www.desirecore.com](https://www.desirecore.com)

The website will automatically detect your operating system and CPU architecture and recommend the most suitable installer. If you need a different version, click "More download options" on the page to expand the full list.

## Available Packages

| Platform | Format | Filename Example | Guide |
|----------|--------|-----------------|-------|
| macOS (Apple Silicon) | `.dmg` | `DesireCore_arm64_x.y.z.dmg` | [macOS Installation](./02-macos.md) |
| macOS (Intel) | `.dmg` | `DesireCore_x64_x.y.z.dmg` | [macOS Installation](./02-macos.md) |
| Windows (x64) | `.exe` | `DesireCore_x64_x.y.z.exe` | [Windows Installation](./01-windows.md) |
| Windows (ARM64) | `.exe` | `DesireCore_arm64_x.y.z.exe` | [Windows Installation](./01-windows.md) |
| Linux (x64) | `.AppImage` | `DesireCore_x86_64_x.y.z.AppImage` | [Linux Installation](./03-linux.md) |
| Linux (ARM64) | `.AppImage` | `DesireCore_arm64_x.y.z.AppImage` | [Linux Installation](./03-linux.md) |

:::tip Not sure which one to pick?
In most cases, simply click the download button on the homepage — the website will automatically detect your system. If the auto-detection is inaccurate:
- **macOS users**: Click  > **About This Mac** in the top-left corner and check the "Chip" field to determine whether it's Apple Silicon or Intel.
- **Windows users**: In most cases, the `x64` version is the right choice.
:::

:::info More Versions
Web, Android, iOS, and HarmonyOS versions are under development. See [More Versions](./04-more-versions.md).
:::

Once the download is complete, click the corresponding installation guide above to continue.
