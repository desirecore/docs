# More Versions

In addition to the desktop client (macOS, Windows, Linux), DesireCore plans to support more platforms.

:::info Under Development
The following versions are actively being developed:

- **Web** — Use directly in your browser, no installation required
- **Android** — Support for Android phones and tablets
- **iOS** — Support for iPhone and iPad
- **HarmonyOS** — Support for HarmonyOS devices
:::

We will announce the latest progress through the [Changelog](/more/changelog) and the official website.

For now, please use the desktop client for the full experience:

- [macOS Installation](./02-macos.md)
- [Windows Installation](./01-windows.md)
- [Linux Installation](./03-linux.md)
