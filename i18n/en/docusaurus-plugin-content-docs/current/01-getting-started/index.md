# Getting Started

Welcome to DesireCore! This guide will walk you through the entire process from installation to your first conversation in **5 minutes**.

:::tip What is DesireCore?
DesireCore is a "delegation-based interactive Agent OS" — you can train your AI companion like mentoring an apprentice, making it understand you better over time and helping you accomplish all kinds of tasks.
:::
