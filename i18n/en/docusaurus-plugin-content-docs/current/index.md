# DesireCore User Manual

Welcome to the DesireCore User Manual! DesireCore is a delegation-based interactive Agent OS — you can teach AI agents your way of working, and they will plan, execute, and report back like capable assistants.

Unlike traditional "Q&A" style AI assistants, DesireCore's core philosophy is: **I teach you → You plan → You execute → You report → You evolve**.
