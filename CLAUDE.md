# CLAUDE.md

本文件为 Claude Code（claude.ai/code）在本仓库中工作时提供说明和约定。

## 项目概览

DesireCore Docs 是 DesireCore（一个“委托操作系统”桌面应用）的官方文档站点。项目基于 **Docusaurus 3.9**（启用了 v4 future flags）、React 19 和 TypeScript 构建，部署到 `docs.desirecore.com`。

## 常用命令

```bash
pnpm install          # 安装依赖
pnpm start            # 启动开发服务器，地址为 localhost:3000（中文，默认语言）
pnpm start -- --locale en   # 启动英文版本开发服务器
pnpm build            # 生产环境构建，输出到 build/
pnpm serve            # 在本地预览生产构建结果
pnpm typecheck        # 执行 TypeScript 类型检查（tsc）
pnpm clear            # 清除 Docusaurus 缓存（.docusaurus/）
pnpm write-translations -- --locale en  # 生成英文翻译文件
```

## 架构说明

### 文档挂载在根路由

`docusaurus.config.ts` 中配置了 `routeBasePath: '/'`，因此文档直接挂载在站点根路径下，没有 `/docs/` 前缀。博客功能已禁用。

### 内容结构（docs/）

通过带编号前缀的目录控制排序。每个部分都在 `sidebars.ts` 中通过 `autogenerated` 生成独立侧边栏：

- `01-getting-started/` → `gettingStarted` 侧边栏（也包含 `index.md`）
- `02-user-guide/` → `userGuide` 侧边栏（共 11 个子章节：界面、对话、教学、委托、超级文档、智能体、记忆、自动化、能力、设置、安全）
- `03-use-cases/` → `useCases` 侧边栏（通用、专业、解决方案）
- `04-concepts/` → `concepts` 侧边栏
- `05-reference/` → `reference` 侧边栏
- `06-faq/` → `faq` 侧边栏
- 顶层页面：`index.md`、`privacy.md`、`terms.md`

### 国际化（i18n）

默认语言为 `zh-Hans`（简体中文）。英文翻译位于 `i18n/en/docusaurus-plugin-content-docs/current/` 下，目录结构与 `docs/` 保持镜像一致。分类元数据（`_category_.json`）用于提供对应的翻译标签。

### 设计系统 —— “Liquid Glass”

`src/css/custom.css` 实现了一个受 Apple 启发的玻璃拟态设计系统，主要包括：

- 以 `--dc-*` 为前缀的自定义 CSS 变量，用于玻璃材质表面和动画效果（Expo easing）
- 将响应式导航栏断点覆盖为 **1280px**（Infima 默认值为 996px），以容纳 8 个中文导航项

### Swizzle 后的组件

- `src/theme/Root.tsx`：使用 Vercel Analytics 包裹整个应用
- `src/theme/Navbar/MobileSidebar/index.tsx`：移除了 `shouldRender` 判断，使 `MobileSidebar` 的 DOM 始终保持挂载，从而支持在 997–1279px 区间内仅通过 CSS 控制响应式行为

## 开发约定

- 提交信息遵循 [Conventional Commits](https://www.conventionalcommits.org/) 规范（例如：`feat:`、`docs:`、`chore:`、`fix:`）
- 文档默认使用中文编写；英文翻译单独维护在 `i18n/en/` 目录下
- `onBrokenLinks: 'throw'`：一旦存在失效的内部链接，构建会直接失败
- 需要 Node.js >= 20，并使用 pnpm 作为包管理器
