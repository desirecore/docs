# 第一次对话

API Key 配置好了，现在来和你的 AI 同伴进行第一次对话吧！

## 选择一个智能体

在对话列表中，你会看到一个预置的智能体——**DesireCore**。它是你的通用 AI 助手，能够处理各种日常任务。

点击它的名字，右侧聊天区域就会切换到它的对话界面。

<!-- 截图占位: 选择智能体 (select-agent.png) -->

:::info 关于 DesireCore 核心智能体
DesireCore 是随应用自带的通用智能体，使用蓝色圆形头像。它可以直接帮你处理任务，也可以在需要时将任务委派（Delegate）给更专业的智能体。
:::

## 发送第一条消息

1. 点击底部的输入框
2. 输入你想说的内容，比如：

   > 你好！请简单介绍一下你自己。

3. 按 **Enter** 发送（或点击右侧的绿色发送按钮）

<!-- 截图占位: 发送消息 (send-first-message.png) -->

:::tip 快捷操作
- **Enter**：发送消息
- **Shift + Enter**：换行（不发送）
:::

## 等待回复

发送后，你会看到一个"DesireCore 思考中"的提示，几秒钟后，智能体的回复就会出现在对话中。

回复以左侧气泡的形式展示，你发送的消息在右侧。

## 理解回复格式

DesireCore 的智能体不仅会回复文字，还可能展示不同类型的内容卡片：

| 卡片类型 | 外观特征 | 含义 |
|---------|---------|------|
| 普通回复 | 白色气泡 | 智能体的文字回复 |
| 行动回执（Receipt） | 绿色/蓝色/橙色卡片 | 智能体执行了某个操作并返回结果 |
| 教学确认（Learned） | 紫色卡片 | 智能体记住了你教给它的规则 |
| 委派通知（Delegate） | 蓝色卡片 | 智能体将任务委派给了另一个专业智能体 |

在刚开始使用时，你看到的主要是普通的文字回复。随着你深入使用，教它做事、让它执行任务，才会看到更丰富的卡片类型。

## 试试更多对话

你可以尝试和 DesireCore 聊一些有用的事情：

- **写一封邮件**："帮我写一封会议邀请邮件，主题是周五下午的项目复盘"
- **分析问题**："我正在考虑换工作，请帮我列一个利弊分析"
- **学习知识**："用简单的语言解释一下什么是 RAG（检索增强生成）"
- **翻译内容**："把这段话翻译成英文：今天天气真好，适合出去走走"

## 完整操作流程回顾

```
选择智能体 → 在输入框输入内容 → 按 Enter 发送 → 等待回复 → 继续对话
```

就这么简单！DesireCore 的核心体验就是自然的对话——你说，它做。随着你使用得越多，它会越来越了解你的偏好和习惯。
