# 安装 DesireCore

DesireCore 桌面客户端支持 macOS、Windows 和 Linux 三大平台。

## 下载

前往 DesireCore 官网下载最新版本：

> **下载地址**：[https://www.desirecore.com/download](https://www.desirecore.com/download)

## 选择你的平台

| 平台 | 安装包格式 | 文件名示例 | 指南 |
|------|-----------|-----------|------|
| macOS (Apple Silicon) | `.dmg` | `DesireCore_arm64_1.0.0.dmg` | [macOS 安装指南](./02-macos.md) |
| macOS (Intel) | `.dmg` | `DesireCore_x64_1.0.0.dmg` | [macOS 安装指南](./02-macos.md) |
| Windows (x64) | `.exe` | `DesireCore_x64_1.0.0.exe` | [Windows 安装指南](./01-windows.md) |
| Windows (ARM) | `.exe` | `DesireCore_arm64_1.0.0.exe` | [Windows 安装指南](./01-windows.md) |
| Linux | `.AppImage` | `DesireCore_x64_1.0.0.AppImage` | [Linux 安装指南](./03-linux.md) |

:::tip 不确定选哪个？
- **macOS 用户**：如果你的 Mac 是 2020 年末或之后购买的，大概率是 Apple Silicon 芯片，选 `arm64` 版本。点击左上角  > 关于本机可以确认。
- **Windows 用户**：绝大多数情况下选 `x64` 版本即可。
:::

:::info 更多版本
Web、Android、iOS、鸿蒙版本正在开发中，详见 [更多版本](./04-more-versions.md)。
:::

选好平台后，点击上面对应的安装指南继续操作。
