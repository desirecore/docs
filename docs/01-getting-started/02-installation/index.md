# 安装 DesireCore

DesireCore 桌面客户端支持 macOS、Windows 和 Linux 三大平台。

## 下载

前往 DesireCore 官网，点击下载按钮即可获取最新版本：

> **下载地址**：[https://www.desirecore.cn](https://www.desirecore.cn)

官网会自动检测你的操作系统和 CPU 架构，推荐最合适的安装包。如果需要其他版本，可以点击页面上的"More download options"展开完整列表。

## 各平台安装包

| 平台 | 安装包格式 | 文件名示例 | 指南 |
|------|-----------|-----------|------|
| macOS (Apple Silicon) | `.dmg` | `DesireCore_arm64_x.y.z.dmg` | [macOS 安装指南](./02-macos.md) |
| macOS (Intel) | `.dmg` | `DesireCore_x64_x.y.z.dmg` | [macOS 安装指南](./02-macos.md) |
| Windows (x64) | `.exe` | `DesireCore_x64_x.y.z.exe` | [Windows 安装指南](./01-windows.md) |
| Windows (ARM64) | `.exe` | `DesireCore_arm64_x.y.z.exe` | [Windows 安装指南](./01-windows.md) |
| Linux (x64) | `.AppImage` | `DesireCore_x86_64_x.y.z.AppImage` | [Linux 安装指南](./03-linux.md) |
| Linux (ARM64) | `.AppImage` | `DesireCore_arm64_x.y.z.AppImage` | [Linux 安装指南](./03-linux.md) |

:::tip 不确定选哪个？
大多数情况下，直接点击官网主页的下载按钮即可——网站会自动识别你的系统。如果自动检测不准确：

- **macOS 用户**：点击左上角  > **关于本机**，查看"芯片"一栏确认是 Apple Silicon 还是 Intel。
- **Windows 用户**：绝大多数情况下选 `x64` 版本即可。
:::

:::info 更多版本
Web、Android、iOS、鸿蒙版本正在开发中，详见 [更多版本](./04-more-versions.md)。
:::

下载完成后，点击上面对应的安装指南继续操作。
