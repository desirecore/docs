# 使用条款

最后更新：2026年7月16日

## 1. 服务范围

DesireCore 是一款委派式交互的 Agent 操作系统，旨在为用户提供安全、私密、高效的 AI 助手服务。通过本软件，用户可以创建、管理和与个性化 AI 助手进行交互。

DesireCore 通过用户自行配置的第三方大语言模型（LLM）API、受支持的云算力或第三方账号连接提供 AI 推理能力。第三方服务的可用性、计费、内容处理和账号状态由相应服务商控制；相关费用和使用责任由用户承担。
