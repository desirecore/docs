# 详细系统要求

## 操作系统

各平台支持的操作系统版本和架构，请参阅 [系统要求](../01-getting-started/01-system-requirements.md)。

## 硬件要求

### 最低配置

| 资源 | 要求 | 说明 |
|------|------|------|
| **CPU** | 双核 | 支持 SSE4.2 |
| **内存** | 4 GB | 运行应用基本需要 |
| **存储** | 500 MB | 应用本身 + 基本数据 |
| **显示** | 1280x720 | 最低分辨率 |

### 推荐配置

| 资源 | 要求 | 说明 |
|------|------|------|
| **CPU** | 四核+ | 多任务并行更流畅 |
| **内存** | 8 GB+ | 多个对话窗口 + 超级文书 |
| **存储** | 2 GB+ | 应用 + 智能体数据 + 缓存 |
| **显示** | 1920x1080+ | 最佳界面体验 |

### 本地模型（Ollama）额外要求

如果你计划使用 Ollama 在本地运行 AI 模型，需要额外的硬件资源：

| 模型规模 | 内存要求 | GPU 显存 | 说明 |
|---------|---------|---------|------|
| 7B 参数 | 8 GB | 6 GB+ | 轻量模型，入门级 |
| 13B 参数 | 16 GB | 10 GB+ | 中等模型，效果更好 |
| 70B 参数 | 64 GB | 40 GB+ | 大模型，接近商用水平 |

## 软件依赖

### 桌面版

DesireCore 桌面版所有依赖已内置，通常不需要额外安装。

| 依赖 | 是否内置 | 说明 |
|------|---------|------|
| 运行时环境 | 是 | 已内置 |
| Git | 否（推荐安装） | 用于智能体版本管理（可选） |

## 网络要求

### 必需的网络连接

DesireCore 在以下场景需要网络：

| 场景 | 目标地址 | 说明 |
|------|---------|------|
| AI 模型调用 | 各供应商 API 地址 | 取决于你配置的供应商 |
| 更新检查 | `download.desirecore.net` | 检查客户端更新 |
| 智能体市场 | 市场 Github 仓库地址 | 浏览和安装智能体/技能 |

### 可离线的功能

以下功能不需要网络（使用 Ollama 本地模型时）：

- 已安装智能体的配置浏览和编辑
- AgentFS 文件管理
- 本地模型对话（需 Ollama）
- 历史对话查看
- 数据导出

### 端口使用

| 端口 | 用途 | 说明 |
|------|------|------|
| 任意可用端口 | Agent Service（Fastify + Socket.IO） | 应用内部通信，自动分配 |
| 11434 | Ollama 本地模型 | 仅使用 Ollama 时需要 |
| 8080 | 本地 Whisper | 仅使用本地语音识别时需要 |

## 存储空间分析

| 数据类型 | 预估大小 | 说明 |
|---------|---------|------|
| 应用安装 | 500-900 MB | 因平台而异 |
| 核心智能体 | 5-10 MB | DesireCore 内置智能体 |
| 每个自定义智能体 | 1-50 MB | 取决于技能和记忆 |
| 对话历史 | 10-100 MB/月 | 取决于使用频率 |
| 缓存 | 50-200 MB | 可安全清除 |
| 日志 | 10-50 MB | 可定期清除 |
