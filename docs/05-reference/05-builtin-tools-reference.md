# 内置工具参考

DesireCore 提供 **20 个内置工具**，随客户端版本自动更新。这些工具无需安装，智能体开箱即用。

## 文件操作

### Read（读取文件）

| 属性 | 说明 |
|------|------|
| **功能** | 从文件系统读取文件内容 |
| **风险等级** | 低 |
| **需要确认** | 否 |
| **使用场景** | 查看文档、读取配置、分析代码 |

### Write（写入文件）

| 属性 | 说明 |
|------|------|
| **功能** | 创建新文件或完整覆盖现有文件 |
| **风险等级** | 中 |
| **需要确认** | 默认是（ask 模式） |
| **使用场景** | 创建文档、生成报告、保存结果 |

### Edit（编辑文件）

| 属性 | 说明 |
|------|------|
| **功能** | 对文件进行精确的文本替换（仅发送差异部分） |
| **风险等级** | 中 |
| **需要确认** | 默认是（ask 模式） |
| **使用场景** | 修改配置、更新文档、代码修补 |

### Ls（列出目录）

| 属性 | 说明 |
|------|------|
| **功能** | 列出指定目录下的文件和子目录 |
| **风险等级** | 低 |
| **需要确认** | 否 |
| **使用场景** | 浏览项目结构、查找文件 |

### NotebookEdit（编辑 Jupyter 笔记本）

| 属性 | 说明 |
|------|------|
| **功能** | 替换、插入或删除 Jupyter Notebook 的特定单元格 |
| **风险等级** | 中 |
| **需要确认** | 默认是 |
| **使用场景** | 数据分析、编辑计算笔记本 |
