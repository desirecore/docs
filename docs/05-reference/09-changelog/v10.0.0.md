# DesireCore v10.0.0

**发布日期**：2026-03-06

## 新功能

- **邮件中心**：新增完整的邮件工作流，支持统一收件箱、会话视图、搜索筛选、附件下载、快速回复、批量操作以及 Gmail、Outlook、IMAP/SMTP 账户接入。
- **HostAgent 设备配对**：现在可以在应用内完成 HostAgent 设备发现、配对与认证，更方便地连接和管理外部设备能力。
- **审批系统升级**：新增更完整的审批体验，支持 AI 辅助审批、按 Agent 保存偏好，并让关键操作的确认流程更清晰。
- **调度系统**：引入任务调度能力，让 Agent 能更稳定地执行连续任务与自动化流程。
- **技能导入**：支持直接导入技能，并在配置时提供更智能的地址校验，降低接入门槛。
- **MCP 服务安装与管理**：新增服务安装、启用、禁用和移除流程，方便在资源管理中管理可用服务。
- **选择性清理数据**：支持按类别清除本地数据，用户可以更精细地管理本地缓存与隐私数据。
- **编辑器快捷键**：新增编辑器快捷键系统，并支持自定义 Markdown 快捷操作，提高内容编辑效率。
- **GUI 桌面自动化**：新增桌面自动化能力，可通过 Agent 执行更多图形界面操作。
- **邮件回复与标签管理**：支持直接回复邮件，并对邮件标签/分类进行管理，日常处理更高效。

## 修复

- 修复了资源管理面板中多项界面显示和数据展示问题，信息呈现更加准确稳定。
- 修复了分屏模式下聊天面板的布局错乱问题，双栏使用体验更稳定。
- 修复了审批卡片状态恢复和流式消息显示不一致的问题，长流程操作更连贯。
- 修复了重新授权后的邮件账户稳定性问题，邮箱连接更加可靠。
- 修复了 Outlook 分类、重新授权首页跳转以及 IMAP 邮箱中的多项问题，邮件使用体验更顺畅。
- 修复了设置弹窗样式与多个界面显示问题，整体界面表现更加一致。
- 修复了清空对话历史时可能出现的崩溃和残留数据问题，历史清理更彻底。
- 修复了授权提示、授权失效处理以及已读未读状态处理问题，邮件状态反馈更准确。
- 修复了 HostAgent 连接与发现过程中的稳定性问题，设备连接成功率更高。
- 修复了 PIM 权限检测可能误触发系统应用的问题，启动过程更加安静自然。
- 修复了 PIM 命令在无权限场景下长时间无响应的问题，权限不足时反馈更及时。
- 修复了智能体导出时的跨域错误，导出流程更加稳定。
- 修复了聊天记录、话题恢复和消息连续性中的多项问题，会话上下文更完整。
- 修复了默认引擎与 Agent 创建/删除后的状态同步问题，Agent 列表和运行状态更新更及时。
- 修复了模型切换后请求仍可能使用旧模型的问题，模型切换结果现在可以正确生效。

## 改进

- **关于页面更新**：优化了关于页面内容展示，版本与产品信息更清晰。
- 应用包体进一步优化，下载安装更轻量。
- macOS 版本的安全签名与公证体验得到改善，打开应用时更少遇到系统安全拦截。
- Gmail 现已支持同步系统标签，并支持一键下载全部邮件内容，提升邮箱整理效率。
- 邮件列表支持无限滚动、自动同步和账户状态展示，浏览大量邮件时更加顺畅。
- 邮件撰写与草稿同步体验得到提升，写信和续写更自然。
- 资源管理与小屏界面响应式表现进一步优化，在不同窗口尺寸下使用更舒适。
- 会话在重启后可自动恢复并静默重试，减少中断带来的影响。
- 新增空响应自动重试机制，在偶发无结果时能自动补救，提高对话成功率。
- PIM 权限请求与滚动保持体验得到优化，系统授权流程更加平滑。
- 聊天记录搜索、分页加载、话题恢复与记忆能力持续增强，长期使用下更容易找回上下文。
- 通知、运行状态、成本追踪和思考过程展示进一步完善，让 Agent 执行状态更透明。
- 首次使用欢迎界面与整体视觉细节得到优化，新用户上手更自然。
