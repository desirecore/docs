# DesireCore v10.0.6

**发布日期**: 2026-03-09

## 新功能
- delegate 异步委派后台处理 + 发送方头像显示 (#212)
- 前端 OpenClaw 导入功能（菜单项 + 弹窗 + 集成）
- 新增 OpenClaw 导入 API 函数与中英文翻译键
- 新增 OpenClaw 扫描与导入后端（scanner + routes）
- 多数据源心跳 + 数据源开关 + 心跳状态时间修复
- Agent 详情页心跳监控面板 + 安静时段后合并汇报
- 卸载按钮改为设备感知，移除无效的禁用/移除按钮
- 已安装注册表完整实现 — IPC + Store + UI 集成
- 安装按钮增加设备选择器和安装进度对话弹窗
- Registry Skill 统一 — entries/ 格式、Agent 驱动安装、共享 Skill

## 修复
- 修复 OpenClaw 导入的 CORS、缓存失效和成功反馈问题
- 心跳调度同步修复 — 支持 UUID 调度 + 禁用改为暂停 + 实时事件通知
- CI release 创建改为幂等模式 (#207)
- 完善 registry 改造 — 优先 entries/ 格式，补全 derive 函数字段
- 修复 button 嵌套导致的 hydration 错误
- 还原 InstallDeviceSelector 为 popover 形式，修复 prop 名称
- ServiceDetail 防御 capabilities/tags 为 undefined
- getRegistryData 从 entries/ 新格式回退派生 apps/services
