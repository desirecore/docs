# DesireCore v10.0.5

**发布日期**: 2026-03-08

## 新功能
- **心跳系统增强**：心跳检查新增时区感知调度、瞬态异常自动重试、软硬双层抑制引擎，与自动化调度系统深度集成
- **SuperDoc Diff 对比**：SuperDoc 新增 Diff 对比面板，支持行内高亮和并排对比两种模式，方便文档修订审阅
- **SuperDoc 沉浸式模式**：SuperDoc 支持进入全屏沉浸式编辑，专注内容创作
- **Temperature 控制**：模型温度参数完全打通前后端，支持在对话中自定义创造性程度；JSON 输出新增自动修复机制
- **邮件服务优化**：邮件未读计数实时同步、列表自动刷新、fetch 异常检查，整体使用更流畅

## 修复
- 修复打包后文本无法选中的问题（反转 user-select 策略）
- 修复 apiFormat 编辑时选项丢失和服务兼容性问题，选项改为数据驱动
- 修复沉浸式交互的返回按钮无法点击
- 修复邮箱管理组件在暗色模式下的显示异常
- 修复 Electron 拖拽区域导致 SuperDoc 工具栏按钮不可点击
- 修复邮箱列表返回时未读计数不刷新
- 修复 IMAP UID 范围查询错误

## 改进
- Skills 加载机制对齐 Claude Code 按需加载模式，新增 S3 全局技能
- 精简调度工具描述，清理心跳模板中的死代码
