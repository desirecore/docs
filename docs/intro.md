# 什么是 DesireCore

**DesireCore** 是一个 Delegation OS（委托操作系统）—— 一款跨平台桌面应用，让你拥有能**学习、执行、可审计、可回滚**的 AI 伙伴。

不同于传统的"你问我答"式聊天，DesireCore 的核心理念是：

> **我教你 → 你规划 → 你执行 → 你汇报 → 你进化**

## 核心能力

| 能力 | 说明 |
|------|------|
| **会学** | 通过对话捕捉你的偏好和习惯，形成持久记忆 |
| **会做** | 不只是聊天，能执行真实任务并汇报结果 |
| **可审计** | 每个动作都有完整记录：做了什么、为什么做 |
| **可回滚** | 随时撤销任何操作，不怕出错 |
