## 阅读建议

如果你是第一次接触 DesireCore，建议按照以下顺序阅读：

1. 先读 **数字同伴** -- 理解"为什么不一样"
2. 再读 **委派式交互模型** -- 理解"怎么用"
3. 然后读 **三层可控性** -- 打消"能不能信"的顾虑
4. 最后根据兴趣阅读其他概念

如果你是技术背景用户，可以直接跳到 **AgentFS 文件系统** 和 **回执系统**，这两个概念最能体现 DesireCore 的技术创新。
