# 通知中心

通知中心是 DesireCore 的统一消息提醒系统，集中展示来自智能体、系统和任务的各类通知。

## 入口位置

通知中心的入口是左侧导航栏底部的**铃铛图标**：

- 有未读通知时，铃铛右上角显示红色角标和未读数量
- 超过 99 条未读时角标变为红色小圆点
- 点击铃铛从右侧滑出通知面板

## 面板布局

通知面板从右侧滑入，宽度 320px，包含以下区域：

```
+----------------------------------+
|  通知中心     [全部已读] [清除全部]|  <- 顶栏
+----------------------------------+
|  [全部] [消息] [系统] [任务]      |  <- 分类 Tabs
+----------------------------------+
|                                  |
|  [图标] 标题            3分钟前   |  <- 通知项
|         正文内容描述...    (未读点)|
|                                  |
|  [图标] 标题            1小时前   |
|         正文内容描述...           |
|                                  |
+----------------------------------+
```

## 通知类型

通知分为三个类别：

| 类别 | 说明 | 示例 |
|------|------|------|
| **消息** | 来自智能体的对话消息通知 | "法律助手向你发送了合同审查结果" |
| **系统** | 系统级别的通知 | "DesireCore 已更新到 v2.1.0" |
| **任务** | 任务执行相关的通知 | "数据分析报告生成完成" |

每种类型有独特的图标和颜色标识，方便快速区分。

## 通知管理

### 查看通知

- 点击通知项可标记为已读，并跳转到相关内容
- 未读通知有浅色高亮背景，已读通知为普通背景

### 批量操作

- **全部已读**：一键将所有通知标记为已读
- **清除全部**：清空所有通知

### 分类过滤

顶部的分类 Tab 可以快速筛选特定类别的通知：
- 选中的分类会以绿色高亮
- "全部"Tab 显示所有类别的通知

## 关闭通知面板

以下操作会关闭通知面板：
- 点击面板外部的遮罩区域
- 按 **Esc** 键
- 点击通知项跳转到相关内容时自动关闭

## 时间显示

通知时间以相对格式展示，更贴近日常阅读习惯：

| 时间范围 | 显示格式 |
|---------|---------|
| 不到 1 分钟 | "刚刚" |
| 1-59 分钟 | "X 分钟前" |
| 1-23 小时 | "X 小时前" |
| 1-29 天 | "X 天前" |
| 30 天以上 | 具体日期（如"2026/2/15"） |

:::info 通知不会丢失
通知数据持久化存储在本地。即使关闭应用，下次打开时所有通知仍然保留，未读状态也会恢复。
:::
