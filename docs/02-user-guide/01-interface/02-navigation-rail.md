# 左侧导航栏

导航栏（NavRail）位于窗口最左侧，宽 62px，是 DesireCore 所有功能的统一入口。

## 结构概览

导航栏从上到下分为三个区域：

```
+--------+
| Traffic|  <- macOS 窗口控制按钮（仅 macOS）
| Lights |
+--------+
|  Logo  |  <- DesireCore 品牌标识
+--------+
|  对话   |  Tab 0
|  关系   |  Tab 1
|  资源   |  Tab 2
|  应用   |  Tab 3
|  市场   |  Tab 4
|        |
|  (空白) |  <- 可拖拽区域（桌面端可拖拽窗口）
|        |
+--------+
|  通知   |  铃铛按钮 + 未读角标
|  活动   |  Tab 5
|  设置   |  齿轮按钮（打开设置面板）
|  头像   |  用户头像（点击编辑个人资料）
+--------+
```

## 功能导航按钮

| 序号 | 图标 | 功能 | 说明 |
|------|------|------|------|
| Tab 0 | 气泡 | **对话** | 默认页面，显示对话列表和聊天区域 |
| Tab 1 | 人形 | **关系** | 智能体和联系人的关系图谱 |
| Tab 2 | 文件夹 | **资源** | 智能体的资源管理器（技能、工具、文件） |
| Tab 3 | 四宫格 | **应用** | 已安装的应用和服务 |
| Tab 4 | 购物袋 | **市场** | 智能体和技能市场，浏览并安装新能力 |
| Tab 5 | 脉冲线 | **活动** | 所有智能体的活动记录和执行日志 |

## 底部工具区

### 通知铃铛

导航栏底部的铃铛图标是通知中心（Notification Center）的入口。

- 有未读通知时，铃铛右上角会显示**红色角标**，标注未读数量
- 超过 99 条未读时角标变为红点（不显示数字）
- 点击铃铛会从右侧滑出通知面板，详见[通知中心](./06-notifications.md)

### 设置按钮

齿轮图标，点击后打开设置面板。你可以在这里配置外观主题、快捷键、语言、算力服务等。

### 用户头像

底部显示你的头像（32x32 圆形）。点击头像可以编辑个人资料。

- 访客模式下，头像右下角会有一个绿色小圆点标记
- 悬停时显示你的用户名或访客模式提示

## 交互状态

导航按钮有三种视觉状态：

| 状态 | 外观 |
|------|------|
| 默认 | 透明背景，图标为灰色 |
| 悬停 | 浅白色半透明背景 |
| 选中 | 白色半透明背景 + 阴影，图标变为绿色 |

当前选中的 Tab 会以绿色图标高亮，让你随时知道自己在哪个功能模块。

:::tip 拖拽窗口
在桌面端（Electron），导航栏中间的空白区域可以用来拖拽整个窗口。macOS 上红黄绿三个窗口按钮位于导航栏顶部；Windows 和 Linux 的窗口控制按钮位于右上角。
:::
