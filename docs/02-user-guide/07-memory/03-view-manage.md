# 查看与管理记忆

智能体的记忆完全透明可控。你随时可以查看智能体记住了什么，修改不准确的内容，或删除不再需要的记忆。

## 进入记忆管理

你可以通过以下方式访问智能体的记忆：

1. 打开智能体详情页
2. 切换到**记忆**标签页
3. 你会看到按域分类的记忆列表

记忆管理界面展示三个域的内容：

- **关系记忆** — 你和这个智能体之间的互动经验
- **共享记忆** — 跨智能体共享的知识
- **核心记忆** — 智能体的出厂知识（通常为只读）

## 查看记忆内容

每条记忆包含以下信息：

- **内容摘要** — 记忆的核心要点
- **类型标签** — preference / fact / decision / commitment / milestone / lesson
- **重要性评分** — 系统根据内容自动评定的重要程度
- **创建时间** — 记忆的生成时间
- **来源会话** — 这条记忆是从哪次对话中提炼出来的

:::tip 追溯来源
点击"来源会话"链接，可以回到产生这条记忆的原始对话上下文，了解完整的讨论背景。
:::

## 搜索记忆

当记忆条目较多时，你可以使用搜索功能快速定位：

**自然语言搜索** — 在搜索框中输入你想查找的内容：

- "关于报告格式的偏好"
- "项目 Alpha 的决策"
- "上次讨论的技术方案"

系统会在所有域中进行语义搜索，返回最相关的结果。高置信度的记忆会优先展示。

## 编辑记忆

如果发现某条记忆不够准确，你可以直接编辑：

1. 找到需要编辑的记忆条目
2. 点击编辑按钮
3. 修改记忆内容
4. 保存修改

:::info 哪些可以编辑
你可以编辑**关系记忆**和**共享记忆**中的内容。核心记忆由智能体开发者定义，通常不支持用户编辑。
:::

## 删除记忆

如果某条记忆不再需要或你不希望智能体记住：

1. 找到要删除的记忆条目
2. 点击删除按钮
3. 确认删除

删除后，这条记忆将不再影响智能体的行为。

:::warning 删除不可撤销
删除记忆后，智能体将无法再引用这条信息。请确认你不再需要它。不过，记忆的历史记录仍保留在版本控制中，技术上可以恢复。
:::

## 标记重要记忆

对于特别重要的记忆，你可以将其标记为**固定（Pin）**：

1. 找到重要的记忆条目
2. 点击固定/置顶按钮
3. 该记忆将获得"永不遗忘"保护

被固定的记忆不会因自动遗忘机制而被压缩或清理。适合用于：

- 关键的业务决策
- 重要的个人偏好
- 不能遗忘的承诺和约定

## 审阅候选记忆

每次对话结束后，系统可能会生成候选记忆供你审阅：

1. 对话结束时，界面底部弹出记忆审阅面板
2. 每条候选记忆显示：
   - 记忆内容预览
   - 建议写入的域
   - 记忆类型和重要性评分
3. 你可以对每条候选记忆进行操作：
   - **接受** — 确认记录这条记忆
   - **编辑后接受** — 修改内容再记录
   - **拒绝** — 不记录这条信息

:::tip 批量操作
如果候选记忆较多，你可以使用"全部接受"或"全部拒绝"来快速处理。
:::
