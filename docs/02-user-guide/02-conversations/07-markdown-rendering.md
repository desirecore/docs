# Markdown 与图表渲染

Companion 的回复会以 Markdown 富文本展示。你可以直接阅读标题、列表、表格、代码块、数学公式和图表，也可以把内容复制为纯文本或保留 Markdown 格式。

## 支持的内容

| 内容类型 | 支持方式 |
|----------|----------|
| 标题、列表、引用 | 按 Markdown 结构渲染 |
| 表格 | 自动保持列对齐，窄屏下可横向滚动 |
| 代码块 | 语法高亮，支持一键复制 |
| 数学公式 | 支持行内和块级 KaTeX / LaTeX |
| Mermaid 图表 | 在消息内渲染，可打开全屏预览 |
| 图片附件 | 在消息中展示缩略图，可点击预览 |
| `.md` 文件路径 | 可信助手回复中的本地 Markdown 路径可点击打开 |

## Mermaid 图表

当回复中包含 Mermaid 代码块时，DesireCore 会自动渲染为图表。点击图表可以打开全屏预览：

- 鼠标滚轮缩放
- 拖拽平移画布
- 双击或点击重置按钮恢复适配屏幕
- 关闭预览后回到原对话位置

如果 Mermaid 代码存在语法错误，消息会保留原始代码块，方便你让 Companion 继续修正。

## 数学公式

DesireCore 支持常见 LaTeX 写法：

- 行内公式：`$E = mc^2$`
- 块级公式：`$$\int_a^b f(x) dx$$`

长公式会在气泡内横向滚动，不会撑破消息布局。

## 文件路径和链接

助手回复中的本地 `.md` 文件路径会显示为可点击链接。点击后，DesireCore 会在应用内打开对应文件或跳转到系统文件位置。

:::info 安全边界
路径点击只对可信助手回复中的本地 Markdown 路径启用。普通文本、用户消息和不明确的路径不会自动执行任何操作。
:::

## 复制内容

消息操作栏提供两种复制方式：

| 操作 | 结果 |
|------|------|
| 复制 | 复制适合粘贴到聊天或普通文档的纯文本 |
| 复制 Markdown | 保留标题、列表、代码块、表格和图表源码 |

当你要把回复继续交给其他 Markdown 编辑器处理时，优先使用「复制 Markdown」。
