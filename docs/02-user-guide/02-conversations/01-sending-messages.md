# 发送消息

DesireCore 的输入区域支持多种消息发送方式，包括文本、图片上传、文件引用和屏幕截图。本章介绍你可以使用的所有发送方式和快捷操作。

## 文本消息

在聊天界面底部的输入框中直接输入文字，即可向 Companion（智能体）发送消息。

- 输入框会随着文字增多自动扩展高度，最大可展开到 120px
- 按下 `Enter` 发送消息（默认快捷键，可在设置中修改）
- 按下 `Shift + Enter` 换行

:::tip 快捷键自定义
发送和换行的快捷键可以在「设置 > 快捷键」中自定义。比如你可以将发送改为 `Ctrl + Enter`，将 `Enter` 设为换行。
:::

### 历史消息快速调用

类似终端的历史命令功能，在输入框中按 `Up` 方向键，可以快速调出你之前发送过的消息。按 `Down` 方向键可以前进到更近的消息，直到回到当前输入草稿。

## 图片上传

你可以通过以下方式向 Companion 发送图片：

1. **点击图片按钮** --- 输入框左侧的图片图标，点击后打开系统文件选择器，支持选择多张图片
2. **粘贴图片** --- 直接在输入框中使用 `Ctrl/Cmd + V` 粘贴剪贴板中的图片
3. **拖拽图片** --- 将图片文件拖拽到输入框区域，或拖到整个聊天区域出现高亮后松手

上传后的图片会以缩略图形式显示在输入框上方。你可以：

- 点击缩略图查看大图预览
- 悬停在缩略图上，点击右上角的关闭按钮移除图片
- 发送前查看上传进度和压缩后的文件大小
- 编辑已发送的用户消息时，删除原有图片或追加新图片

:::info 支持的图片格式
支持常见的图片格式，包括 JPEG、PNG、WebP、GIF 等。图片会以 Base64 编码发送给 AI 模型进行视觉理解。
:::

## 屏幕截图

点击输入框左侧的截图按钮，或使用截图快捷键，即可进入区域选择截图模式。选定区域后可以进行标注，完成的截图会自动添加到输入框的图片列表中。

:::tip
即使你不在聊天页面（比如在设置页面），截图完成后也会自动添加到当前对话的输入区域，并弹出提示告知你截图已添加。
:::

## 文件引用

点击输入框左侧的「+」按钮，可以打开系统文件选择器来引用文件或文件夹：

- **选择文件** --- 文件路径会以标签形式显示在输入框上方，Companion 能看到你引用了哪些本地文件
- **选择文件夹** --- 文件夹会被添加为工作目录（Working Directory），Companion 可以在该目录范围内操作文件

文件引用标签支持悬停查看完整路径，也可以点击关闭按钮移除。

### 编辑已发送的文档引用

对于已经发送的「文字 + 文档引用」消息，悬停消息并点击 **编辑**，可以同时修改文字和引用列表：

- 点击引用标签上的关闭按钮可以删除一个或全部文档引用
- 删除过引用后会出现虚线「+」按钮，可重新选择一个或多个文档
- 新选择的文件会按路径去重，文件夹不会作为文档引用加入
- 即使文字没有变化，仅增删或重新选择文档也可以确认编辑

确认后，DesireCore 会按编辑后的文字和文档引用重新发送这条消息。取消编辑则保留原消息，不会修改引用。当前入口仅用于编辑原本已经包含文档引用的消息；纯文字消息仍通过输入区添加新的文件引用。

:::warning 文件引用与图片上传的区别
「文件引用」只是告诉 Companion 你引用了哪个文件的路径，Companion 会自行读取。而「图片上传」会真正读取图片内容并发送给 AI 模型。如果你想让 AI 分析一张图片，请使用图片上传而非文件引用。
:::

### 拖拽到聊天区

当你把本地文件或文件夹拖到聊天区时，整个聊天框会进入高亮状态。松手后：

- 图片会作为图片附件加入输入框
- 普通文件会作为文件引用加入输入框
- 文件夹会作为工作目录引用加入输入框

这种方式适合把桌面、下载目录或文件管理器中的素材直接交给当前智能体处理。

## 智能预测

当你刚完成一轮对话、输入框为空且智能体处于空闲状态时，DesireCore 可能会在输入框上方显示一条「下一条消息」预测芯片。

- 点击预测芯片、按 `Tab` 或按 `→` 可以采纳预测内容
- 继续手动输入时，预测会自动隐藏
- 预测使用轻量模型生成，只用于补全你可能要发送的下一句话

如果你不需要这项能力，可以在设置中关闭智能预测。

## 斜杠命令

在输入框中输入 `/`（斜杠），会弹出命令自动补全菜单。斜杠命令让你快速调用系统功能和 Companion 技能。

### 命令导航

- `Up/Down` 方向键 --- 在命令列表中上下移动
- `Tab` 或 `Enter` --- 选择当前高亮的命令
- `Esc` --- 关闭命令菜单

### 命令类型

| 类型 | 语法 | 说明 |
|------|------|------|
| 系统命令 | `/命令名` | 内置功能，如 `/plan`、`/new`、`/compact`、`/rewind`、`/steer`、`/help` 等 |
| 技能调用 | `/skill:技能名` | 调用已安装的技能 |
| 快捷调用 | `/技能名` | 直接输入技能名称快速调用 |

输入 `/skill:` 后会自动显示当前 Companion 可用的技能列表，继续输入可过滤匹配。

### 常用系统命令

| 命令 | 用途 | 适合场景 |
|------|------|----------|
| `/plan` | 强制 Companion 先制定计划再动手 | 希望先确认方案，再允许智能体执行 |
| `/new` | 开启新的上下文边界 | 想换话题，但保留可见历史 |
| `/compact` | 手动压缩当前会话上下文 | 长对话接近模型上下文上限 |
| `/rewind` | 回撤到某条用户消息前的检查点 | 智能体走错方向，想回到之前状态 |
| `/steer` | 给正在运行的智能体追加方向指令 | 不想中断任务，只想调整执行方向 |
| `/skill:技能名` | 显式加载指定技能 | 已知道当前任务需要某个技能 |
| `/help` | 查看可用命令完整列表 | 不确定当前有哪些可用命令 |

:::tip `/plan` 的两种用法
- 单独发送 `/plan`：开启计划模式，不发任何消息
- `/plan 把所有 console.log 换成 logger.info`：开启计划模式 + 把任务作为消息发出去

计划被批准（或拒绝）后，强制模式会自动关闭。
:::

`/steer` 只在智能体运行中生效。发送后它会作为运行中的 steer 指令进入当前任务，并在历史中以专门标记展示。

## 快捷键速查

| 操作 | 默认快捷键 (macOS) | 默认快捷键 (Windows/Linux) |
|------|-------------------|--------------------------|
| 发送消息 | `Enter` | `Enter` |
| 换行 | `Shift + Enter` | `Shift + Enter` |
| 打开命令菜单 | `/` | `/` |
| 取消/关闭 | `Esc` | `Esc` |
| 截图 | 可在设置中配置 | 可在设置中配置 |
| 粘贴图片 | `Cmd + V` | `Ctrl + V` |
| 采纳预测 | `Tab` 或 `→` | `Tab` 或 `→` |
| 回撤检查点 | `Cmd + Alt + Z` | `Ctrl + Alt + Z` |

输入框底部也会显示当前的快捷键提示，方便你随时查看。
