# 导出对话

你可以把对话保存为 Markdown 或 PDF，用于归档、分享、提交报告或继续编辑。

## 导出入口

在聊天头部点击「更多」，选择「导出对话」。如果你已经进入多选模式，也可以只导出选中的对话范围。

导出前可以选择：

- 导出格式：Markdown 或 PDF
- 导出范围：当前会话、完整对话历史或选中消息相关的 run
- 工具调用：隐藏、摘要或完整展开
- 是否包含 thinking / system 类辅助信息
- 是否嵌入图片附件

:::info 多选导出的范围
多选导出会按选中消息所属的 run 聚合导出，确保上下文、工具结果和图片引用完整。它不是简单地逐条拼接消息。
:::

## Markdown 导出

Markdown 导出适合继续编辑或交给其他系统处理。它会保留：

- 消息角色和时间
- 标题、列表、表格、代码块
- Mermaid 图表源码
- 图片引用
- 工具调用摘要或完整内容

## PDF 导出

PDF 导出适合正式归档和分享。桌面端会使用统一的文档渲染链路生成 PDF，尽量保持 Markdown 的标题、表格、代码块、图片和公式样式。

:::tip 图片和 thinking
较新的导出流程会把对话中的图片嵌入 PDF，并正确处理 thinking 块。若图片来自本地文件，请确认文件仍然存在。
:::

## 导出建议

| 目标 | 建议 |
|------|------|
| 继续写作 | 导出 Markdown |
| 发送给非技术同事 | 导出 PDF |
| 调试智能体行为 | 导出完整工具调用 |
| 做会议纪要或报告 | 隐藏工具调用，仅保留摘要 |
