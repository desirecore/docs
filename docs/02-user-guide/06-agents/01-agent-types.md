# 智能体类型

智能体（Agent）是 DesireCore 的核心。每个智能体都有自己的人格、记忆和技能，像一个真正的专业助手一样为你工作。

## 通用智能体 vs 专业智能体

DesireCore 中的智能体分为两大类：

### 通用智能体

通用智能体没有特定的专业领域限制，能处理各种日常任务：

- 回答问题、整理资料
- 写作和润色文档
- 代码辅助和技术问答
- 日程规划和信息检索

**DesireCore 核心智能体**就是一个通用智能体——它是你的默认助手，开箱即用。

### 专业智能体

专业智能体针对特定领域做了深度优化，拥有该领域的专业知识、行为准则和专属技能：

| 领域 | 示例 | 特点 |
|---|---|---|
| 法律 | 法律顾问 | 熟悉法律条文，严谨措辞，风险提示 |
| 房产 | 房产顾问 | 了解市场行情，合同审核，投资分析 |
| 翻译 | 翻译官 | 多语言精通，上下文感知，术语一致 |
| 写作 | AI 文书 | 文书结构化，风格适配，版本管理 |
| 数据 | 数据分析师 | Python 运算，可视化，洞察提炼 |

## 预置智能体简介

DesireCore 预置了多个常用智能体，你可以直接使用或从市场安装更多。预置智能体按用途分类：

| 分类 | 智能体 | 适用场景 |
|---|---|---|
| 效率 | 项目助手、日程管家 | 项目管理、任务跟踪 |
| 开发 | 程序员、代码审阅 | 编码辅助、Bug 排查 |
| 沟通 | 翻译官、商务文书 | 翻译、邮件写作 |
| 学习 | 学习教练、知识整理 | 学习规划、笔记梳理 |
| 创意 | 创意文案、头脑风暴 | 文案撰写、灵感激发 |

:::info 智能体会成长
每个智能体都会通过对话学习你的偏好和习惯。使用越多，它越了解你。这些"成长"以记忆的形式保存，你可以随时查看和管理。
:::

## 如何选择适合的智能体

选择智能体的简单原则：

1. **日常杂事** → 使用 DesireCore 核心智能体（通用型）
2. **专业领域任务** → 从市场安装或创建专业智能体
3. **特定工作流** → 创建自定义智能体，配置专属技能和行为准则

:::tip 不确定用哪个？
直接问 DesireCore 核心智能体："我需要一个能帮我审合同的助手"，它会推荐合适的智能体或帮你创建一个。
:::
