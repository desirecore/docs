# 编辑人格设定

人格设定（Persona）定义了智能体"是谁"——它的语气、风格、专业领域和行为方式。这不是一段人设文案，而是一份**可执行的行为规范**。

## 什么是人格设定

人格设定保存在智能体目录下的 `persona.md` 文件中。它告诉智能体：

| 方面 | 说明 | 示例 |
|---|---|---|
| **回答策略** | 怎么回答问题 | "先给结论，再给理由；少客套" |
| **语气风格** | 用什么口吻说话 | "专业但不刻板，简洁但不敷衍" |
| **确认策略** | 什么时候要问用户 | "发消息、删除、付费操作前必须确认" |
| **禁区** | 什么不该做 | "不要使用模板腔，不要回避给结论" |
| **不确定处理** | 拿不准时怎么办 | "说明不确定点并给验证路径" |

:::info 人格 vs 行为准则
**人格设定**定义的是"怎么做"——语气、风格、回答方式。**行为准则**（Principles）定义的是"做什么和不做什么"——规则、边界、优先级。两者互补但不重叠。
:::

## 编辑名称、头像、描述

在智能体详情页面，你可以修改基础信息：

### 名称

智能体的显示名称。选择一个简洁明了的名字，便于在对话列表中识别。

### 头像

智能体的视觉标识。你可以：

- 使用文字头像（一个汉字或 emoji）
- 上传自定义图片

### 描述

一句话介绍智能体的用途。好的描述能帮你快速识别智能体的定位，例如：

- "专注于合同审阅和法律咨询的专业助手"
- "帮你规划日程、追踪任务的效率管家"

## 角色定位和性格特征

在人格编辑器中，你可以配置智能体的角色和性格：

### 角色定位

角色定位决定了智能体的专业方向。例如：

- **法律顾问**：专注法律领域，措辞严谨，注重风险提示
- **技术助手**：专注编程和技术，善于给出代码示例
- **写作教练**：专注文字表达，注重结构和措辞

### 性格特征

性格特征影响智能体的交互风格。你可以为智能体添加多个标签：

| 特征 | 效果 |
|---|---|
| 严谨 | 回答注重准确性，会标注不确定的地方 |
| 简洁 | 回答言简意赅，不废话 |
| 友好 | 语气温和，适当使用鼓励性表达 |
| 果断 | 倾向给出明确结论，避免模棱两可 |
| 谨慎 | 涉及敏感操作时主动提醒风险 |

## 问候语设置

问候语是智能体在新对话开始时的第一句话。一个好的问候语应该：

- 简短自然，不要过度客套
- 让用户知道这个智能体能做什么
- 引导用户开始对话

例如：

> "你好，我是你的法律顾问。合同审阅、法律咨询，有什么可以帮你的？"

:::tip 通过对话修改人格
你也可以直接在对话中教智能体调整风格：

> "以后回复简洁一点，不要超过三段。"

> "不要再用'首先...其次...最后'这种套路了。"

智能体会把你的偏好记录下来，在后续对话中应用。
:::
