# 智能体市场

智能体市场（Agent Market）是发现和获取智能体的中心。你可以在这里浏览官方和社区提供的智能体，一键安装到本地使用。

## 进入市场

点击主导航栏中的"市场"图标即可进入。市场主界面包含三个标签页：

| 标签页 | 内容 |
|---|---|
| **Agent** | 智能体模板，安装后成为你的专属助手 |
| **Skill** | 技能包，扩展智能体的能力 |
| **应用** | 与 DesireCore 集成的第三方应用 |

## 浏览和搜索

### 分类浏览

市场中的项目按用途分类：

| 分类 | 说明 |
|---|---|
| 效率 | 提升工作效率的工具型智能体 |
| 开发 | 软件开发辅助 |
| 沟通 | 语言和沟通相关 |
| 学习 | 知识和学习辅助 |
| 创意 | 创意和设计相关 |

### 搜索和过滤

- **关键词搜索**：在名称、描述、标签中搜索
- **过滤条件**：按分类、安装状态、来源仓库筛选
- **排序方式**：按更新时间、名称或热度排序

### 项目卡片

每个智能体以卡片形式展示，包含：

- 头像和名称
- 简短描述
- 版本号和更新时间
- 标签
- 安装状态按钮

点击卡片进入详情页，查看完整介绍、功能特性和变更日志。

## 安装智能体

### 安装步骤

1. 找到你想要的智能体
2. 点击"安装"按钮
3. 等待下载和配置完成
4. 安装成功后，在对话页面切换到新智能体即可开始使用

:::info 安装过程
安装时，DesireCore 会下载智能体的配置文件并写入本地的 AgentFS 文件系统。整个过程通常只需要几秒钟。
:::

### 高风险技能确认

如果智能体包含高风险技能（如文件写入、命令执行），安装时会弹出安全确认对话框，告知你涉及的敏感操作。你需要明确确认后才能继续安装。

### 依赖检查

部分智能体依赖特定的工具或连接。如果缺少依赖，系统会提示你先安装所需的组件。

## 更新和卸载

### 更新

系统会自动检测已安装智能体的可用更新：

- 启动时自动检查
- 市场页面会显示"更新"角标
- 点击"更新"按钮查看变更日志后确认更新

:::tip 更新不会丢失你的修改
如果你手动修改过智能体的配置，更新时系统会检测到偏离并跳过覆盖，保护你的自定义内容。
:::

### 卸载

1. 在已安装的智能体上点击菜单按钮
2. 选择"卸载"
3. 可选择是否保留配置文件
4. 确认后完成卸载
