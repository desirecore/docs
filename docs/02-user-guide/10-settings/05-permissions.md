# 权限设置

DesireCore 的部分功能需要操作系统授予相应权限才能正常工作。权限设置页面集中展示了所有需要的系统权限及其用途，方便你按需授权。

## 打开权限设置

1. 打开 **设置** 面板
2. 点击 **权限管理** 分区中的 **系统权限**

## 权限列表

| 权限 | 用途 | 何时需要 |
|------|------|---------|
| **屏幕录制** | 截屏、屏幕共享等功能 | 使用截图或屏幕分享功能时 |
| **麦克风** | 语音输入、语音通话 | 使用语音对话功能时 |
| **摄像头** | 视频通话、拍照 | 使用视频相关功能时 |
| **辅助功能** | 自动化操作、全局快捷键 | 使用自动化工具或全局热键时 |
| **通知** | 桌面通知推送 | 需要接收后台任务完成提醒时 |

:::info
上表中列出的是 macOS 系统的权限分类。Windows 和 Linux 的权限机制有所不同，实际显示的权限项可能会根据操作系统而变化。
:::

## 如何授权

1. 在权限列表中，找到需要授权的权限项
2. 点击该权限项
3. 系统会弹出一个引导对话框，说明为什么需要此权限
4. 点击 **去授权**，系统设置会自动打开到对应页面
5. 在系统设置中为 DesireCore 勾选权限

:::tip
你不需要一次性开启所有权限。DesireCore 遵循最小权限原则——只在你使用到相关功能时才会请求权限。如果暂时不需要某项功能，可以选择"暂不授权"。
:::

## 权限状态

每个权限项会显示当前状态：

- **已授权**：该权限已开启，相关功能可以正常使用
- **未授权**：该权限尚未开启，相关功能不可用
- **受限**：系统部分授予了权限（例如麦克风只允许在应用前台时使用）

## Web 模式说明

在 Web 模式下运行 DesireCore 时，权限由浏览器管理。浏览器通常会在你首次使用麦克风、摄像头等硬件时自动弹窗询问。你可以在浏览器的站点设置中查看和管理已授予的权限。
