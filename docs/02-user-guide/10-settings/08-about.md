# 关于与更新

在"关于"页面你可以查看当前版本、检查更新，以及查阅使用条款和隐私政策。

## 查看当前版本

1. 打开 **设置** 面板
2. 滚动到最底部的 **关于** 分区
3. **版本** 一行显示当前安装的版本号

## 检查更新

:::info
检查更新功能在桌面客户端（Electron）中可用。
:::

### 手动检查

1. 在关于分区中，点击 **版本** 一行
2. 右侧会显示"检查更新"，点击后开始检查
3. 检查结果：
   - **已是最新版本**：你正在使用最新版
   - **新版本 vX.X.X**：有可用更新，系统会自动开始下载
   - **检查失败，点击重试**：网络异常，稍后再试

### 更新流程

当检测到新版本后：

1. 系统自动下载更新包，右侧显示下载进度百分比
2. 下载完成后，显示 **立即安装**
3. 点击即可安装更新并重启应用

更新服务器地址：`download.desirecore.net`

## 法律文档

在关于分区中，你还可以查阅：

- **使用条款**：DesireCore 的使用协议
- **隐私政策**：数据收集和处理说明

点击对应项即可查看完整文档。
