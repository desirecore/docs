# 用户档案

用户档案是你在 DesireCore 中的身份信息。智能体会根据你的档案来称呼你，你也可以自定义头像让界面更有个人特色。

## 编辑个人信息

1. 点击应用左上角的 **头像** 或 **名称** 区域
2. 进入个人资料编辑面板

在这里你可以修改以下信息：

- **名称**：智能体称呼你时使用的名字
- **头像**：你在对话界面中显示的头像

## 设置头像

DesireCore 支持自定义头像：

1. 在个人资料面板中，点击头像区域
2. 选择一张本地图片上传
3. 支持的图片格式：JPEG、PNG、WebP

头像会保存在本地的 `~/.desirecore/config/avatar/` 目录中。

:::tip
推荐使用正方形图片作为头像，系统会自动裁剪为圆形显示。
:::

## 显示名称

你的显示名称会出现在：

- 对话界面中你的消息旁
- 智能体回复中对你的称呼

修改名称后点击 **保存** 即可生效。

## 游客模式与登录

DesireCore 支持免登录使用，所有核心功能在游客模式下都完全可用。

- **游客模式**：数据保存在本地，无需注册账号
- **登录用户**：未来登录后可将数据同步到云端（功能开发中）

:::info
当前版本的登录功能正在开发中，所有功能均可在免登录状态下正常使用。你的数据安全存储在本地设备上。
:::
