# 语言设置

DesireCore 支持多语言界面，你可以随时切换。

## 支持的语言

| 语言 | 语言代码 |
|------|---------|
| 简体中文 | `zh-CN` |
| English | `en-US` |

后续版本将根据社区反馈增加更多语言支持。

## 如何切换语言

1. 打开 **设置** 面板
2. 找到 **语言** 分区
3. 点击 **语言 / Language** 右侧的下拉菜单
4. 选择目标语言

切换后，界面所有文案会立即更新为所选语言。

## 语言切换的影响范围

切换界面语言只改变 DesireCore 应用本身的 UI 文案，包括：

- 菜单、按钮、提示文字
- 设置项名称和描述
- 系统通知和状态提示

:::info
语言切换**不会**影响智能体的对话语言。智能体使用什么语言回复，取决于它的角色设定（Persona）和你使用的提示语言。即使界面切换为英文，智能体仍然可以用中文与你对话。
:::

## 贡献翻译

DesireCore 的界面文案存储在 `app/data/i18n/` 目录下。如果你希望帮助改进翻译或贡献新语言，欢迎通过 GitHub 提交 Pull Request。
