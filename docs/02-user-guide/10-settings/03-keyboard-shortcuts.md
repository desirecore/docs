# 快捷键设置

DesireCore 提供了丰富的键盘快捷键，帮助你高效操作。你可以查看所有快捷键，也可以根据个人习惯进行自定义。

## 打开快捷键设置

1. 打开 **设置** 面板
2. 点击 **快捷键** 分区中的 **键盘快捷键**
3. 进入快捷键管理子面板

## 编辑器快捷键模式

DesireCore 内置了三种编辑器快捷键模式，适应不同用户的操作习惯：

| 模式 | 说明 | 适合人群 |
|------|------|---------|
| **原生模式** | 保留各编辑器的内置快捷键，不做任何覆盖。这是最安全的选项。 | 不确定时的默认选择 |
| **统一模式**（推荐） | 使用 DesireCore 统一的快捷键系统，在所有编辑器中保持一致的按键体验。 | 经常在多种编辑器间切换的用户 |
| **自定义模式** | 完全接管快捷键，允许你自由修改所有按键绑定。 | 高级用户 |

## 常用快捷键

### 对话输入

| 功能 | 快捷键 |
|------|--------|
| 发送消息 | `Enter` |
| 换行 | `Shift + Enter` |
| 取消输入 | `Esc` |

### 超级文书

| 功能 | 快捷键 |
|------|--------|
| 接受修改 | `Cmd/Ctrl + Enter` |
| 拒绝修改 | `Cmd/Ctrl + Backspace` |
| 下一个修改 | `Cmd/Ctrl + ]` |
| 上一个修改 | `Cmd/Ctrl + [` |

### 编辑器通用

| 功能 | 快捷键 |
|------|--------|
| 保存 | `Cmd/Ctrl + S` |
| 另存为 | `Cmd/Ctrl + Shift + S` |
| 查找 | `Cmd/Ctrl + F` |
| 替换 | `Cmd/Ctrl + H` |
| 跳转到行 | `Cmd/Ctrl + G` |
| 全屏 | `F11` |

### 应用全局

| 功能 | 快捷键 |
|------|--------|
| 显示/隐藏窗口 | 可自定义 |
| 截图 | 可自定义 |
| 撤销 | `Cmd/Ctrl + Z` |
| 重做 | `Cmd/Ctrl + Shift + Z` |

:::tip
上表中标注"可自定义"的快捷键，需要在快捷键设置中手动绑定。macOS 使用 `Cmd`，Windows/Linux 使用 `Ctrl`。
:::

## 自定义快捷键

1. 在快捷键管理面板中，找到你想修改的快捷键
2. 悬停在该项上，点击 **编辑** 按钮
3. 按下你想要的新按键组合
4. 如果与已有快捷键冲突，系统会提示冲突项
5. 点击 **保存** 确认修改

你也可以使用顶部的搜索框快速定位特定快捷键。

## 重置快捷键

如果自定义后想恢复默认设置，点击面板顶部的 **重置所有** 按钮，即可将所有快捷键恢复为默认值。

:::warning
重置操作会清除所有自定义的快捷键绑定，且无法撤销。请确认后再操作。
:::
