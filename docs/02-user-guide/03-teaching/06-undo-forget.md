# 撤销与遗忘

人会犯错，教学也一样。也许你教了一条不太合适的规则，或者智能体理解偏了。DesireCore 提供了完善的"后悔药"机制，让你可以安全地修正教学内容。

## 撤销最近的操作

### 在学习反馈卡片中撤销

这是最简单的方式。当智能体刚刚学到某个内容时，反馈卡片上有"撤销"按钮，点击即可：

```
你：[点击 ✗ 撤销]
智能体："已撤销。这条内容不会影响我之后的行为。"
```

### 在对话中撤销

如果你已经确认了学习但后来发现不对，可以在对话中直接撤销：

```
你："撤销你刚才学的那条关于违约金的规则。"
智能体："你确定要撤销以下内容吗？

        规则：审查合同时，违约金超过 20% 标记为高风险
        学习时间：10 分钟前
        来源：本次对话

        确认撤销？"

你："确认。"
智能体："已撤销。"
```

## 让智能体遗忘特定规则

如果要删除更早之前教的某条规则：

```
你："忘掉你之前学的'邮件必须抄送经理'这条规则。"

智能体："找到了这条规则：

        规则：发送外部邮件时必须抄送经理
        层级：强制
        学习时间：2 周前
        来源：3 月 1 日的对话

        确认要遗忘这条规则吗？"

你："确认。"
智能体："已遗忘。以后发邮件时不会再自动抄送经理了。"
```

:::warning 遗忘是永久操作
与撤销不同，遗忘是永久删除这条规则。如果以后又需要，你需要重新教一遍。在确认遗忘之前，请确认这确实是你想要的。
:::

### 批量查看和管理规则

如果你想系统地清理智能体的规则库：

```
你："列出所有关于邮件的规则。"

智能体："找到 4 条关于邮件的规则：

        1. [强制] 外部邮件使用正式语气
           学习时间：1 个月前 | 状态：生效中

        2. [建议] 邮件篇幅控制在 300 字以内
           学习时间：3 周前 | 状态：生效中

        3. [偏好] 邮件结尾用'此致'
           学习时间：2 周前 | 状态：生效中

        4. [强制] 给客户发邮件前需要确认
           学习时间：1 周前 | 状态：生效中

        你想对哪条规则进行操作？"

你："第 3 条删掉，第 2 条的字数限制改为 500 字。"

智能体："确认以下操作：
        - 遗忘规则 3：邮件结尾用'此致'
        - 修改规则 2：邮件篇幅从 300 字改为 500 字

        确认？"
```

## 重置智能体到初始状态

:::danger 这是不可恢复的操作
重置会清除智能体的所有学习内容（规则、示例、偏好），将其恢复到刚安装时的状态。执行前请确认你真的需要从零开始。
:::

如果你想让智能体"从零开始"：

```
你："重置你的所有学习内容。"

智能体："⚠️ 这将清除我学到的所有内容：
        - 42 条规则
        - 15 个示例
        - 8 条偏好设定

        重置后，我将恢复到初始状态，之前所有的教学
        内容都将丢失。

        请输入'确认重置'以继续。"

你："确认重置。"

智能体："已重置完成。所有学习内容已清除。
        你可以重新开始教我了。"
```

### 部分重置

你也可以只重置某个领域的知识：

```
你："只重置关于合同审查的规则，其他的保留。"

智能体："将重置以下内容：
        - 合同审查相关规则：12 条
        - 合同审查相关示例：5 个

        其他领域的知识将保留。确认？"
```

## 操作对照表

| 操作 | 影响范围 | 可恢复 | 适用场景 |
|------|---------|--------|---------|
| **撤销** | 最近一次学习 | 是（重新教即可） | 刚教完发现不对 |
| **遗忘** | 指定的某条规则 | 否（需重新教） | 删除过时或错误的规则 |
| **修改** | 指定的某条规则 | 是（有版本记录） | 调整已有规则的细节 |
| **部分重置** | 某个领域的所有知识 | 否 | 彻底重做某个领域的教学 |
| **完全重置** | 全部学习内容 | 否 | 从零开始培养智能体 |

:::tip 善用版本历史
在智能体设置中，你可以查看规则的变更历史。每次修改都有记录，你可以对比不同版本，必要时恢复到之前的版本。这比完全重置安全得多。
:::
