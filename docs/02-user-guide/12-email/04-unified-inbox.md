# 统一收件箱

当你绑定了两个或以上邮箱时，**统一收件箱**让你无需在不同账户之间来回切换，即可在一个列表中浏览所有邮件。

## 进入方式

在账户管理首页，点击顶部的「统一收件箱」卡片即可进入。这张卡片只在你有两个以上邮箱时才会出现。

## 页面全貌

![统一收件箱](/img/user-guide/email/unified-inbox.png)

### 顶部栏

顶部显示所有账户的汇总信息：

- **未读/总数统计** — 所有账户的邮件汇总
- **账户数** — 当前聚合了几个邮箱
- **同步状态** — 同步进行中时显示脉冲动画
- **撰写** — 打开写信弹窗
- **同步全部** — 同时触发所有账户的邮件同步

### 文件夹标签页

通过水平标签页切换文件夹（收件箱、已发送、草稿、回收站），切换后邮件列表自动刷新。

### Provider 过滤

文件夹标签页下方提供按邮箱类型筛选的按钮：

![Provider 过滤](/img/user-guide/email/provider-filter.png)

- 默认显示「全部」，即所有邮箱的邮件混合排列
- 点击某个 Provider 后，只显示该类型邮箱的邮件
- 每个按钮上的数字是对应类型的邮件数量

### 授权失败提示

如果某些账户的授权已过期，页面顶部会浮动显示醒目的提示横幅，点击「重新登录」返回账户管理首页重新授权。

## 邮件列表

所有邮箱的邮件按时间倒序混合排列。每封邮件带有 **Provider 徽章**用于区分来源：

- Gmail — 红色徽章
- Outlook — 蓝色徽章
- IMAP — 紫色徽章

未读邮件的主题以加粗样式显示，左侧有未读指示点。

向下滚动到列表底部时，会自动加载更多邮件。

## 查看邮件详情

点击邮件列表中的任意一封，会进入邮件详情视图，显示完整的邮件正文、附件、标签等信息。打开邮件时会自动标记为已读。

在详情视图中，你可以直接回复或转发邮件。
