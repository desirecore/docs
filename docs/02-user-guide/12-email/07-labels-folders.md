# 标签和文件夹

邮件多了之后，找到特定邮件会越来越难。DesireCore 提供了灵活的标签系统，帮助你分门别类地整理邮件。

## 文件夹

每个邮箱账户都有一组文件夹，在邮件视图的水平标签页中展示。默认显示四个最常用的文件夹：

| 文件夹 | 说明 |
|-------|------|
| 📥 收件箱 | 收到的邮件 |
| 📤 已发送 | 你发出的邮件 |
| 📝 草稿 | 未发送的草稿 |
| 🗑️ 回收站 | 已删除的邮件 |

文件夹结构与邮箱服务商的原生结构对应，添加账户时自动从服务器同步。

## 标签的类型

DesireCore 中有三种标签：

| 标签类型 | 来源 | 说明 |
|---------|------|------|
| **系统标签** | 邮箱服务商 | Gmail 的收件箱、已加星标、重要等，由邮箱自带 |
| **账户标签** | 用户创建 | 绑定在特定邮箱账户下的自定义标签 |
| **全局标签** | 用户创建 | 跨所有邮箱通用的标签，在统一收件箱中特别有用 |

## 创建和管理标签

### 创建标签

你可以在标签管理中创建新标签，设置：

- **标签名称** — 如「待回复」、「项目 A」、「重要客户」
- **颜色** — 从 10 种预设颜色中选择（红、橙、黄、绿、青、蓝、靛、紫、粉、灰）
- **范围** — 仅当前邮箱，或全局通用
- **可见性** — 是否在标签列表中显示
- **排序权重** — 调整在列表中的显示顺序

### 编辑和删除标签

你可以随时修改标签的名称、颜色和排序。删除标签不会删除邮件本身——邮件只是不再带有这个标签。

## 为邮件添加标签

在邮件预览面板的工具栏中，有标签/分类下拉菜单：

- **Gmail 账户** — 显示标签多选器，可以为邮件勾选多个标签
- **Outlook 账户** — 显示分类选择器，功能与标签类似
- **IMAP 账户** — 不支持标签操作

一封邮件可以同时带有多个标签。

## 标签同步

### Gmail 标签

DesireCore 会在添加 Gmail 账户时自动从服务器同步你已有的标签。在 DesireCore 中修改标签也会同步回 Gmail 服务器。你也可以手动触发标签同步。

### Outlook 分类

Outlook 使用「分类」（Categories）的概念，功能与标签类似。DesireCore 会自动同步你的 Outlook 分类，并将其作为标签展示。

:::tip 全局标签的妙用
如果你有多个邮箱，可以创建全局标签（如「紧急」、「本周处理」），在统一收件箱中跨邮箱筛选邮件。
:::
