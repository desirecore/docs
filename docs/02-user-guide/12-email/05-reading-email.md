# 阅读和管理邮件

在账户管理首页点击某个邮箱卡片后，进入该账户的邮件视图。这是你日常处理邮件的主要界面。

## 页面全貌

![邮件视图 - 消息模式](/img/user-guide/email/email-view-messages.png)

### 顶部栏

- **返回** — 回到账户管理首页，同时刷新未读计数
- **搜索栏** — 实时搜索邮件（按 `/` 快捷键可聚焦）
- **同步状态** — 显示「同步中」（蓝色脉冲）、「同步失败」（红色）或上次同步时间
- **同步** — 手动触发一次邮件同步
- **撰写** — 打开写信弹窗

### 文件夹标签页

顶部栏下方是水平排列的文件夹标签（收件箱、已发送、草稿、回收站）。点击切换文件夹，邮件列表自动更新。活跃的标签以渐变色高亮显示，带有未读数徽章。

### 视图模式切换

文件夹标签页右侧提供两种浏览模式的切换：

- **消息模式**（默认） — 每封邮件独立显示
- **会话模式** — 按邮件线程聚合，同一主题的往来邮件归为一个会话

### 两栏布局

页面下方分为左右两栏：左栏是邮件/会话列表（280-320px 宽），右栏是预览面板（自适应宽度）。

## 消息模式

### 邮件列表（左栏）

![邮件列表项与快速操作](/img/user-guide/email/email-list-hover.png)

每封邮件以列表项展示，包含未读指示点、发件人、主题、预览文字和时间。未读邮件的主题以加粗样式显示，选中的邮件左侧有蓝色竖线。

邮件列表支持**虚拟滚动**，即使有大量邮件也能保持流畅。向下滚动时会自动加载更多。

#### 快速操作

鼠标悬停在邮件上时，右侧出现快速操作按钮：

| 按钮 | 功能 |
|------|------|
| 已读/未读 | 切换已读/未读状态 |
| 归档 | 归档邮件 |
| 删除 | 删除邮件 |

所有操作都采用**乐观更新**——界面立即反映变更，如果 API 调用失败则自动回滚。

#### 批量操作

鼠标悬停时，邮件左侧的未读圆点会变为复选框。勾选一封或多封邮件后，列表顶部出现批量操作栏：

![批量操作栏](/img/user-guide/email/batch-actions.png)

支持全选/取消全选、批量标记已读、标记未读和删除。

### 邮件预览（右栏）

选中一封邮件后，右栏显示完整内容。

![邮件预览面板](/img/user-guide/email/email-preview.png)

#### 顶部工具栏

提供关闭、回复、全部回复、转发、标签/分类管理、归档和删除等操作。

#### 邮件头信息

显示发件人头像和名称、邮箱地址、收件人、抄送、时间、邮件大小、优先级和已有标签。

#### 邮件正文

HTML 格式的邮件以富文本渲染，纯文本邮件以等宽字体展示。

#### 附件列表

如有附件，显示文件名、大小和下载按钮。

#### 快速回复

预览面板底部提供快速回复功能，无需打开完整的写信弹窗。支持切换「回复」和「全部回复」模式。

![快速回复](/img/user-guide/email/quick-reply.png)

:::tip 未选中状态
在没有选中任何邮件时，右栏显示空状态提示：「选择一封邮件 — 在左侧列表中点击邮件以查看详情」。
:::

## 会话模式

切换到会话模式后，同一线程的邮件会被聚合为一个会话。

![邮件视图 - 会话模式](/img/user-guide/email/email-view-conversations.png)

### 会话列表（左栏）

每个会话显示主题、参与者列表、邮件数量、未读数和最新邮件时间。

### 会话详情（右栏）

选中一个会话后，右栏展开显示该会话中的所有邮件，每封以可折叠的卡片形式呈现：

- 点击邮件头部可展开/折叠
- 展开时自动标记该邮件为已读
- 会话底部提供快速回复

## 搜索与筛选

### 关键词搜索

在顶部搜索栏输入关键词，邮件列表会实时过滤，匹配邮件的主题、正文和发件人信息。

| 快捷键 | 功能 |
|-------|------|
| `/` | 聚焦搜索栏 |
| `Enter` | 触发搜索 |
| `Esc` | 清空搜索 |

### 高级筛选

点击搜索栏旁的筛选按钮，展开高级筛选面板：

![高级筛选面板](/img/user-guide/email/advanced-filter.png)

支持按发件人、日期范围、是否有附件、是否未读等条件组合筛选。

## 授权失败处理

如果当前账户的 OAuth 授权已过期，页面顶部会浮动显示醒目的提示横幅。点击「重新授权」打开重新授权弹窗，完成授权后恢复正常。

![授权过期提示](/img/user-guide/email/auth-expired.png)

## 操作反馈

所有邮件操作执行后，右下角会弹出 Toast 提示（如「已标记为已读」或「删除失败，请重试」），4 秒后自动消失。
