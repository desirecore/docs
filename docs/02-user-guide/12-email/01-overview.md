# 邮件管理概览

**DesireCore 邮件管理**将你的所有邮箱聚合在一个界面中统一管理。

## 支持的邮箱类型

| 邮箱类型 | 认证方式 | 典型场景 |
|---------|--------|--------|
| **Gmail** | OAuth 2.0 授权 | 个人 Google 邮箱、Google Workspace |
| **Outlook** | OAuth 2.0 授权 | 个人 Outlook/Hotmail、Microsoft 365 |
| **IMAP/SMTP** | 账号密码 | QQ 邮箱、163 邮箱、企业邮箱等 |

:::tip 一句话理解
Gmail 和 Outlook 通过官方授权接入，安全且功能完整；IMAP/SMTP 是通用协议，几乎支持所有邮箱服务商。
:::

## 核心功能一览

- **多账户管理** — 添加多个不同类型的邮箱，在同一界面查看和切换
- **统一收件箱** — 将所有邮箱的邮件汇聚到一个视图中，按时间排序
- **收发邮件** — 阅读、回复、撰写新邮件，支持富文本编辑和附件
- **消息与会话** — 两种浏览模式：按单封邮件查看，或按会话线程聚合
- **搜索与筛选** — 关键词搜索和多维度高级筛选，快速定位邮件
- **标签与分类** — 自定义标签体系，同步 Gmail 标签和 Outlook 分类
- **邮件规则** — 设定条件自动分类、标记、归档或转发邮件
- **批量操作** — 多选邮件后批量标记已读、删除
- **草稿与签名** — 自动保存草稿，管理多个邮件签名
- **实时同步** — 后台自动轮询，新邮件即时推送通知
- **离线可用** — 已同步的邮件离线可查看，操作在恢复网络后自动执行
- **本地存储** — 邮件数据加密存储在本机，不经过第三方服务器

## 数据安全

你的邮件数据始终保存在本地设备上：

- **Token 加密**：OAuth Token 和 IMAP 密码使用 AES-256-GCM 加密存储
- **本地优先**：邮件内容和索引存储在 `~/.desirecore/mail/` 目录
- **无中转**：DesireCore 直接与邮箱服务商通信，不经过任何第三方服务器

:::info 关于隐私
DesireCore 不会上传你的邮件内容到云端。所有同步操作都在你的设备与邮箱服务商之间直接完成。
:::
