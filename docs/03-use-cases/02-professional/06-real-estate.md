# 房产顾问

## 场景描述

你准备买套房，看了好几个楼盘，每个中介都说自己推荐的最好。你不确定哪个楼盘真正适合自己，也不知道报价是否合理。需要一个客观、专业的分析来辅助决策。

房产顾问（Real Estate Advisor）智能体可以帮你分析楼盘优劣势、评估价格合理性、解读购房政策变动，以及系统对比多个备选楼盘，让你的购房决策有数据支撑。

## 推荐智能体

**房产顾问** -- 房产领域专业顾问，提供楼盘分析、价格评估与购房决策支持。

## 完整对话示例

```
你: 我在 A 楼盘和 B 楼盘之间纠结，能帮我对比一下吗？
    A 楼盘：8.5 万/平，89 平三居，地铁口 500 米
    B 楼盘：7.8 万/平，95 平三居，地铁 1.2 公里

房产顾问: 好的，我来帮你做全面对比分析。

        基础信息对比
        | 维度       | A 楼盘      | B 楼盘      |
        |-----------|------------|------------|
        | 单价       | 8.5 万/平   | 7.8 万/平   |
        | 面积       | 89 平三居    | 95 平三居    |
        | 总价       | 757 万      | 741 万      |
        | 地铁距离   | 500 米      | 1.2 公里    |

        核心优劣势

        A 楼盘优势：
        - 地铁更近，通勤便利
        - 学区资源更好
        - 开发商品牌知名度高

        B 楼盘优势：
        - 单价低 8%，性价比更高
        - 面积大 6 平，三居更宽敞
        - 周边商业配套更成熟

        购房建议
        - 重视学区和通勤 → 选 A
        - 重视性价比和居住舒适度 → 选 B

        你买房主要用途是什么？自住还是投资？
        有没有学区需求？

你: 买婚房自住，计划 2 年内要孩子，上班在 CBD。

房产顾问: 基于你的情况，我更推荐 A 楼盘，理由：
        1. 学区需求：A 楼盘对口学校排名靠前
        2. 通勤：地铁 500 米，到 CBD 约 35 分钟
        3. 保值：学区房在市场调整期更抗跌
        4. 总价差异仅 16 万，但学区和通勤优势明显

        月供参考（首付 30%，贷款 30 年）：
        A 楼盘：月供约 28,500 元
        B 楼盘：月供约 27,800 元
        差额仅 700 元/月
```

## 关键步骤

1. **明确需求** -- 告诉智能体预算、用途、偏好（学区、通勤等）
2. **楼盘分析** -- 智能体分析区位、配套、开发商背景
3. **价格评估** -- 参考周边成交数据，评估报价合理性
4. **对比决策** -- 系统对比多个候选楼盘，给出推荐

## 最终成果

一份客观的楼盘对比分析报告，包含价格评估、优劣势对比和个性化购房建议。

:::tip 最佳实践
- 尽量提供完整的购房需求（预算、用途、家庭情况、通勤地点），推荐会更精准
- 对中介的报价不确定时，让智能体查询周边同类房源的成交价格作参考
- 政策变动时（如利率调整、限购放松），及时咨询智能体了解对你的具体影响
:::
