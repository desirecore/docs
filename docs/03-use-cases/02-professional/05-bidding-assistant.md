# 招投标助手

## 场景描述

你公司拿到了一个项目的招标文件，投标截止日期还有 18 天。你需要搞清楚资格要求、编写技术方案、制定报价策略，最后还要确保投标文件万无一失。任何一个环节出纰漏，都可能导致废标。

招投标助手（Bidding Assistant）智能体可以帮你系统化地完成整个投标流程：解读招标文件、自审投标资格、生成技术方案框架、分析最优报价，最后逐项检查投标文件的完整性。

## 推荐智能体

**招投标助手** -- 招投标全流程管理专家，负责标书编制、资格审查与报价策略。

## 前期教学：让智能体了解你的公司实力

> 招投标最耗时的是反复整理公司资质和过往业绩。教一次，之后每个新项目都能自动匹配。

### 教规则

```
你: 我来教你我们公司的基本情况，以后投标用得上：
    [上传 公司资质汇总.pdf]
    [上传 近三年中标项目清单.xlsx]
    补充几条规则：
    1. 报价利润率底线：不低于 15%，低于这个数不投（强制）
    2. 技术方案引用业绩时，优先用同行业、同规模的项目（偏好）
    3. 投标文件统一用公司模板格式，封面必须有公司 LOGO 和资质编号（强制）

招投标助手: 📝 学到了新内容
            来源：公司资质汇总.pdf + 近三年中标项目清单.xlsx + 本次说明
            类型：规则（投标策略）
            层级：强制 × 2 项，偏好 × 1 项

            我的理解：
            投标时自动匹配公司相关资质和同类业绩案例。报价环节
            确保利润率 ≥15%，否则建议放弃。技术方案优先引用同
            行业案例。文件格式统一用公司模板。

            已录入资质：营业执照、ISO 9001/14001/45001、
            信息系统集成三级、软件著作权 12 项...
            已录入业绩：中标项目 47 个，涉及 6 个行业。

            [✓ 正确]  [✎ 修改]  [✗ 撤销]

你: [✓ 正确]
```

教学完成后，招投标助手已掌握你公司的资质和投标策略。之后拿到新的招标文件直接发过去分析即可。

## 完整对话示例

```
你: 帮我分析一下这个招标文件的重点内容。
    [粘贴招标文件]

招投标助手: 我来帮你深度解读这份招标文件。

        项目基本信息
        项目名称：智慧城市数据中心建设项目
        预算金额：2,500 万元
        招标方式：公开招标

        关键时间节点
        投标截止：6月20日 09:30（剩余 18 天）
        答疑截止：6月10日 17:00（剩余 8 天）

        资格审查要求（缺一不可）
        - 独立法人资格
        - 电子与智能化工程专业承包一级资质
        - 安全生产许可证
        - ISO9001 质量管理体系认证
        - 近 3 年类似项目业绩 >= 3 个

        评分标准
        商务部分（30 分）：报价 20 分 + 业绩 10 分
        技术部分（50 分）：方案 25 分 + 实施 15 分 + 售后 10 分

        风险提示
        - 不接受联合体投标
        - 质保期要求 3 年（行业通常 2 年）
        - 履约保证金为合同金额的 10%

        建议行动
        1. 立即核实资质是否满足
        2. 3 天内完成现场踏勘
        3. 7 天内确定技术方案框架
        4. 15 天内完成标书初稿

你: 我们符合这个项目的投标资格吗？

招投标助手: 资格自审结果：基本符合（3 项需补充）
        符合项：8/11
        需补充：
        1. 业绩证明：已完工 2 个，需补充在建项目合同
        2. 项目经理：需更换为持机电工程专业证书的人员
        3. 信用等级：当前 AA 级，需说明或提升
        ...

        📋 回执摘要
        引用的资质：信息系统集成三级 · ISO 9001 等 4 项
        引用的业绩：同行业中标项目 3 个
        引用的规则：利润率底线 15% · 公司模板格式
        均来自你的教学（2 周前）✓
```

## 关键步骤

1. **招标文件解读** -- 智能体提取关键要求、时间节点和评分标准
2. **资格自审** -- 逐项检查是否满足投标条件，标注缺项
3. **技术方案生成** -- 根据招标要求生成方案框架，突出评分重点
4. **报价策略分析** -- 基于成本测算和竞争分析，推荐最优报价
5. **文件完整性检查** -- 投标前逐项检查所有文件是否齐全

## 最终成果

一套完整的投标材料，包含招标文件解读报告、资格自审清单、技术方案、报价分析和投标文件检查清单。

:::tip 最佳实践
- 拿到招标文件后第一时间让智能体分析，尽早发现资格缺项，留足补充时间
- 技术方案重点投入高分章节，智能体会根据评分标准提示哪些部分需要重点撰写
- 投标前一天务必做一次完整性检查，避免因形式问题（如缺签字盖章）导致废标
- **并行应标**：同时有多个招标项目时，一次发给招投标助手，它会并行分析每份招标文件，自动从你教过的资质库中匹配各项目所需的资质和业绩
:::
