# 财务助手

## 场景描述

季度末了，你需要编制下季度的部门预算。要参考历史数据、考虑业务增长计划、还要合理分配各项费用。同时还有一堆报销单据等着审核。

财务助手（Finance Assistant）智能体可以帮你基于历史数据科学编制预算、智能审核报销单据的合规性、生成标准化财务报表，并识别成本优化机会。

## 推荐智能体

**财务助手** -- 专精于财务管理领域，处理预算编制、费用审批、报表归档和成本分析。

## 前期教学：让智能体掌握你的财务制度

> 每家公司的报销标准都不一样。花几分钟把公司规定教给财务助手，之后它审核报销单时就会严格对照你的制度，而不是"通用标准"。

### 教规则

```
你: 我把我们公司的费用报销管理制度传给你学习。
    [上传 公司费用报销制度2024版.pdf]
    有几条特别注意的：
    1. 市内交通：只接受打车软件电子发票，不接受手撕出租车发票（强制）
    2. 餐费：工作日不超过 80 元/人，接待客户不超过 200 元/人（强制）
    3. 出差住宿：北上广深不超过 500 元/晚，其他城市不超过 350 元/晚（强制）
    4. 超标报销：需要附书面说明，经部门主管审批后才能报（偏好）

财务助手: 📝 学到了新内容
          来源：公司费用报销制度2024版.pdf + 本次补充说明
          类型：规则（费用报销标准）
          层级：强制 × 3 项，偏好 × 1 项

          我的理解：
          审核报销单时，交通费只认打车软件电子发票；餐费按工作日
          80 元/人、客户接待 200 元/人把控；住宿按城市等级分档
          （一线 500、其他 350）。超标需附说明并有主管签字。

          [✓ 正确]  [✎ 修改]  [✗ 撤销]

你: [✓ 正确]
```

教学完成后，财务助手已掌握你公司的报销标准。之后直接发报销单过去审核即可。

## 完整对话示例

```
你: 帮我编制下季度销售部门的预算。上季度预算总额 180 万，
    实际支出 182 万。下季度销售目标增长 20%，
    计划新招 2 名销售。

财务助手: 好的，我来编制下季度预算。

        历史数据参考（上季度）
        | 费用类别 | 预算  | 实际  | 执行率  |
        |---------|------|------|---------|
        | 人员成本 | 80 万 | 78 万 | 97.5%  |
        | 差旅费  | 25 万 | 32 万 | 128%   |
        | 业务招待 | 15 万 | 18 万 | 120%   |
        | 市场推广 | 50 万 | 45 万 | 90%    |
        | 办公费用 | 10 万 | 9 万  | 90%    |

        下季度预算草案：

        1. 人员成本：95 万
           - 现有团队：78 万
           - 新增 2 人（2 个月薪资）：12 万
           - 绩效奖金：5 万

        2. 差旅费：40 万
           - 说明：上季度超支 28%，本季度有 3 场展会

        3. 业务招待：20 万
        4. 市场推广：60 万（配合销售目标增长）
        5. 办公费用：12 万

        预算总额：227 万（环比 +24.7%）

        风险提示
        差旅费增长 60%，建议设置月度预算预警线。

        需要我生成正式的预算申请表吗？

你: 差旅费能控制在 35 万吗？另外帮我审核这几张报销单。

财务助手: 差旅费调整为 35 万。削减建议：
        - 部分远程会议替代出差，节约 3 万
        - 展会住宿选择经济型酒店，节约 2 万

        报销审核结果：
        通过 12 笔 | 退回 2 笔 | 拒绝 1 笔
        详细说明...

        📋 回执摘要
        引用的规则：餐费标准 · 住宿标准 · 交通发票要求
        均来自你的教学（上周）✓
```

## 关键步骤

1. **提供基础信息** -- 历史数据、业务计划、预算约束
2. **生成预算草案** -- 智能体基于数据生成分项预算
3. **调整优化** -- 根据你的反馈调整各项数额
4. **费用审核** -- 批量审核报销单据，检查合规性

## 最终成果

一份基于数据分析的科学预算方案，以及合规性审核报告。

:::warning 免责声明
财务助手提供的建议仅供参考，不构成专业财务或税务意见。涉及重大财务决策时，建议咨询专业会计师。
:::

:::tip 最佳实践
- 编制预算时提供尽可能完整的历史数据，预算准确性会显著提高
- 费用审核时可以配置你公司的报销标准（如差旅标准、招待限额），智能体会严格按照标准审核
- 月末、季末可以让财务助手自动生成财务报表，包含关键指标和异常预警
- **月末批量审核**：把整月报销单一次发给财务助手，它会逐条对照你教过的制度核查，超标项自动汇总，通常几分钟内完成全部审核
:::
