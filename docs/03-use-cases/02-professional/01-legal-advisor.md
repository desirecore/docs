# 法律顾问

## 场景描述

你收到了一份供应商合同，明天就要签约，但法务同事出差了。合同里密密麻麻的条款，你担心里面有不利于公司的"坑"。

法律顾问（Legal Advisor）智能体可以快速审查合同，逐条分析关键条款的合理性，标出风险点并给出修改建议。就像身边随时有一位经验丰富的法律助手。

## 推荐智能体

**法律顾问** -- 专精于法律合规领域，具备合同审查、法规检索、风险评估和合规建议等能力。

## 完整对话示例

```
你: 请帮我审查这份供应商合同，明天就要签约了。
    [粘贴合同内容]

法律顾问: 我来帮你快速审查这份合同。正在分析条款...

        审查结果摘要

        合同主体：清晰明确
        付款条款：存在风险
        违约责任：基本合理
        争议解决：建议修改

        重点关注事项

        1. 付款条款 -- 中风险
           问题：预付款比例过高（50%），且未约定进度节点
           建议：改为 30% 预付 + 40% 进度款 + 30% 验收款
           依据：《民法典》合同编相关规定

        2. 争议解决 -- 低风险
           问题：约定仲裁但未明确仲裁机构
           建议：明确约定具体仲裁委员会名称

        3. 知识产权 -- 需关注
           问题：未明确定制开发成果的知识产权归属
           建议：补充约定"定制成果归甲方所有"

        总体评估
        合同基本可用，建议修改上述 3 处后再签署。
        需要我生成修改建议稿吗？

你: 帮我生成修改建议稿，重点改付款条款。

法律顾问: 已生成修改建议稿。
        付款条款修改建议：
        原文："甲方应在合同签订后 5 个工作日内支付合同
              总价的 50% 作为预付款..."
        修改为："甲方按以下节点分期支付：
              (1) 合同签订后 5 个工作日内支付 30% 作为预付款；
              (2) 乙方完成系统部署并通过中期验收后 5 个工作日内
                  支付 40%；
              (3) 项目终验通过后 10 个工作日内支付剩余 30%。"
        ...
```

## 关键步骤

1. **上传或粘贴合同** -- 提供待审查的合同文本
2. **获取审查报告** -- 智能体逐条分析，标注风险等级
3. **查看修改建议** -- 针对每个风险点给出具体的修改文案
4. **生成修改稿** -- 可选择生成包含所有修改建议的合同版本

## 最终成果

一份详细的合同审查报告，标注了所有风险点和对应的修改建议，让你在签约前心中有数。

:::warning 免责声明
法律顾问智能体提供的建议仅供参考，不构成正式法律意见。涉及重大法律决策（如大额合同签署、诉讼应对、股权交易等）时，建议咨询执业律师。
:::

:::tip 最佳实践
- 上传合同时一并说明你最关心的条款（如付款方式、违约责任），智能体会重点分析这些部分
- 如果涉及特定行业（如建筑工程、IT 外包），告诉智能体行业背景，它会参考行业惯例给出更精准的建议
- 可以让法律顾问和财务助手协作，一个审条款一个算成本，全面评估合同
:::
