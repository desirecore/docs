# AI 招投标

## 场景描述

一个大型投标项目涉及方方面面：招标文件解读、资格审查、技术方案撰写、报价策略制定、合规性检查、文件格式审核......需要法律、财务、技术、商务多个专业领域的配合。传统做法是各部门分别作业再人工整合，耗时长、容易出错。

DesireCore 的 AI 招投标解决方案通过多智能体协作，让各专业领域的智能体并行工作、交叉审核，将投标周期压缩一半以上。

## 涉及智能体

- **招投标助手** -- 统筹投标全流程，编制标书
- **法律顾问** -- 审查投标文件的合规性和合同条款
- **财务助手** -- 成本测算、报价策略分析
- **数据分析师** -- 竞争对手分析、市场数据支撑
- **AI 文书** -- 文案润色、排版整合

## 端到端流程

### 阶段一：招标文件分析（第 1-2 天）

```
你: 我们收到了一个 2500 万的数据中心建设项目招标文件，
    请帮我全面分析。

招投标助手: 正在解读招标文件...

        已提取关键信息：
        - 投标截止：6月20日（18 天后）
        - 评分方式：综合评分（技术 50 + 商务 30 + 价格 20）
        - 必备资质：4 项
        - 业绩要求：3 个千万级项目

        同时已委派：
        → 法律顾问：审查合同条款和合规要求
        → 财务助手：开始成本预估
```

### 阶段二：资格审查与策略制定（第 3-5 天）

招投标助手完成资格自审，财务助手完成成本测算，法律顾问给出合规建议。各智能体的分析结果汇总为一份综合评估报告。

### 阶段三：技术方案编写（第 6-12 天）

```
招投标助手: 基于评分标准，技术方案重点分配：
        - 系统架构（高分项）-- 25% 评分权重
        - 安全设计（招标方特别强调）
        - 实施计划和售后方案

        已生成技术方案框架，6 大章节，28 个子项。
```

### 阶段四：报价与审核（第 13-16 天）

```
财务助手: 成本测算结果：
        直接成本 2,300 万 + 间接成本 401 万 = 2,701 万

        报价策略建议：
        推荐报价 2,480 万（平衡中标概率和利润）
        预计价格得分：18-19 分（满分 20）

法律顾问: 投标文件合规性检查：
        - 合同条款第 5 条有不利条款，建议在答疑时提出
        - 质保期 3 年超出行业惯例，注意成本预留
```

### 阶段五：最终检查与提交（第 17-18 天）

```
招投标助手: 投标文件完整性检查结果：

        总检查项：56 项
        已完成：53 项
        缺失：2 项（保证金凭证、分项报价表）
        待确认：1 项（签字盖章）

        紧急事项：
        1. 今日内上传保证金凭证
        2. 2 小时内完成分项报价表
```

## 质量控制

- **交叉审核**：每个智能体完成本职工作后，由其他智能体交叉检查
- **风险标注**：所有高风险项自动标注，必须经你确认
- **版本管理**：每次修改自动保存版本，可随时回溯
- **完整性检查**：提交前自动逐项核对文件清单

## 最终成果

一套完整的投标材料，经过多智能体交叉审核，涵盖技术方案、商务报价、资格证明和合规审查。

:::tip 最佳实践
- 拿到招标文件后第一时间启动分析，给后续编写留足时间
- 答疑环节很重要，法律顾问发现的不利条款可以通过答疑要求招标方澄清
- 最终提交前至少留半天做完整性检查，不要卡到最后一刻
- **提前上传公司资质库**：把公司营业执照、资质证书、历史中标业绩等提前教给招投标助手，之后每个新项目都会自动匹配相关资质，不需要每次重新整理
:::
