# 数据分析

## 场景描述

你手上有一份销售数据表格，老板要求下午之前给出趋势分析和下季度预测。手动整理数据、画图表、写分析结论，至少需要半天时间。

数据分析师（Data Analyst）智能体可以在几分钟内完成这些工作：自动识别数据格式、清洗异常值、生成统计摘要、绘制可视化图表，并给出可操作的分析结论。

## 推荐智能体

**数据分析师** -- 擅长数据清洗、统计分析、趋势预测和可视化报表生成，支持 CSV、Excel、JSON 等多种数据格式。

## 完整对话示例

```
你: 帮我分析这份 Q4 销售数据，我想了解各区域的表现和趋势。
    [上传 sales_q4.csv]

数据分析师: 正在分析你上传的销售数据...

        数据概览
        - 数据行数：15,234 条
        - 时间范围：2024-10-01 至 2024-12-31
        - 涉及产品：128 个 SKU
        - 销售区域：华东、华南、华北、西部

        关键指标
        ┌──────────────┬───────────┬──────────┬──────────┐
        │ 指标         │ Q4 实际    │ Q3 对比  │ 同比增长  │
        ├──────────────┼───────────┼──────────┼──────────┤
        │ 总销售额     │ 2,580 万  │ +18.5%   │ +32.1%   │
        │ 订单量       │ 15,234    │ +12.3%   │ +28.4%   │
        │ 客单价       │ 1,694 元  │ +5.5%    │ +2.9%    │
        └──────────────┴───────────┴──────────┴──────────┘

        区域表现排名
        1. 华东区：980 万（38%）-- 超额完成目标
        2. 华南区：720 万（28%）-- 达成目标
        3. 华北区：580 万（22%）-- 略低于目标 5%
        4. 西部区：300 万（12%）-- 低于目标 15%

        异常发现
        12 月 25 日销售额异常低（仅 8 万），建议核实数据完整性。

        需要我进一步分析某个区域或产品线吗？

你: 西部区为什么低于目标这么多？能深入分析一下吗？

数据分析师: 西部区深入分析结果如下：
        - 主力产品销量下降 22%，可能受竞品促销活动影响
        - 11 月新客户获取量环比下降 35%
        - 但客单价逆势增长 8%，说明现有客户忠诚度较高
        建议：加大西部区的获客投入，同时维护好存量客户...
```

## 关键步骤

1. **上传数据** -- 将 CSV、Excel 或 JSON 文件上传给智能体
2. **描述分析需求** -- 说明你想了解什么（趋势、对比、预测等）
3. **获取分析结果** -- 智能体自动生成统计摘要、图表和洞察
4. **深入追问** -- 针对发现的问题进一步追问，获取更深入的分析

## 最终成果

包含关键指标、区域对比、趋势图表、异常检测和改进建议的完整分析报告，可以直接用于团队汇报。

:::tip 最佳实践
- 上传数据前确保列名清晰（如"销售额"而非"col_1"），智能体能更准确地理解数据含义
- 分析需求越具体，结果越精准。"分析销售趋势"不如"对比 Q3 和 Q4 各区域的销售增长率，找出增长最快和最慢的区域"
- 如果数据量很大，可以先让智能体做数据概览，再针对感兴趣的方向深入分析
:::
