# DesireCore 用户手册

欢迎来到 DesireCore 用户手册！DesireCore 是一个委派式交互的 Agent 操作系统 —— 你可以教会 AI 智能体你的工作方式，让它们像得力助手一样为你规划、执行和汇报。

不同于传统的"你问我答"式 AI 助手，DesireCore 的核心理念是：**我教你 → 你规划 → 你执行 → 你汇报 → 你进化**。
