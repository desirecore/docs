# DesireCore 用户手册

欢迎来到 DesireCore 用户手册！DesireCore 是一个开放的 Agent OS（智能体操作系统），支持多 Agent 协作，帮助你快速打造能干活的数字员工超级团队。

它基于独创的 [AgentFS 文件系统](./concepts/agentfs) 将智能体的配置、记忆、技能统一文件化管理，通过 [智能任务编排引擎](./concepts/task-orchestration) 自动解析你的意图并分配给最合适的智能体执行。系统支持复杂任务拆解、多智能体并行协作、全程状态追踪与超时自动提醒，并具备能力自动匹配、安全边界管控、变更可审计可追溯等企业级特性。

不同于传统的"你问我答"式 AI 助手，DesireCore 的核心理念是：**我教你 → 你规划 → 你执行 → 你汇报 → 你进化**。DesireCore 的目标是让行业专家训练 Agent 的门槛降到最低，让每个人都能用上专业的 Agent 团队。
