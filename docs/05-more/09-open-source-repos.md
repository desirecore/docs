# 开源仓库

DesireCore 项目在 GitHub 上维护了多个官方仓库，欢迎关注和参与贡献。

> **GitHub 组织主页**：[github.com/desirecore](https://github.com/desirecore)

## 核心仓库

| 仓库 | 说明 | 状态 |
|------|------|------|
| [desirecore](https://github.com/desirecore/desirecore) | DesireCore 客户端软件主仓库 | 初步计划稳定后开源，根据公司融资情况和经营战略决定，不作为最终承诺。各可解耦模块会逐步开源 |
| [desirecore-hostagent](https://github.com/desirecore/desirecore-hostagent) | 跨平台 Host Agent，用于 Windows、macOS、Linux、Android、iOS、HarmonyOS 的 Host 常驻服务，实现操作系统的权限获取、GUI 操作与命令执行 | 初步计划稳定后开源，根据公司融资情况和经营战略决定，不作为最终承诺。各可解耦模块会逐步开源 |
