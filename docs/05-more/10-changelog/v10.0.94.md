# DesireCore v10.0.94

**发布日期**: 2026-07-23

![v10.0.94 更新概览](/img/changelog/v10.0.94.png)

## 新功能
- **手机验证码登录**：新增手机号验证码登录方式，验证即注册，登录后可在资料页查看已绑定的手机和邮箱状态。
- **统一调度管理工具**：将调度能力整合为单一的 ManageSchedule 工具，支持创建、查看、修改和删除定时任务。
- **LSP 语义代码导航**：新增代码智能导航能力，支持定义跳转、查找引用、悬停提示、文档符号等语义操作。
- **工作目录胶囊**：聊天输入框新增当前工作目录胶囊，点击可快速切换或添加工作目录，与资源面板实时同步。
- **RefreshAgentFS 工具**：智能体现在可以主动刷新自己的文件系统变更，无需等待目录监控延迟。
- **后台命令观测与停止**：完善后台执行命令的监控能力，支持查看输出、等待完成和主动终止。
- **设置面板「新建实例」入口**：在设置关于页面新增入口，可快速打开多实例选择器创建新实例。
- **WebSearch 搜索结果可信展示**：AI 引用搜索结果中的链接若不在真实来源中，会以虚线标记提醒用户核实，工具调用卡片直接展示可点击的真实来源。
- **新增实例入口**：在设置关于页面新增入口，可快速打开多实例选择器创建新实例。

## 修复
- 修复对话结束后「续写」活动药丸永久残留的问题，现在续写状态最迟 11 分钟内会自动清除。
- 消除 AI 流式回复中包含 Mermaid 图表时的反复闪烁，图表渲染后不再随后续文字生成而重画。
- 恢复消息中引用文件夹的能力，现在可以像引用文件一样引用整个目录发送给 AI。
- 修复 Agent 执行重操作时主进程短暂阻塞导致误报“本地服务连接中”的问题。
- 修复切换 Agent 时工作目录胶囊短暂显示旧 Agent 目录的问题。
- 修复后台分析会话重试时，聊天头部误显示“重试中”的问题。
- 修复升级进度窗口在部分显示分辨率下文字显示不全的问题。
- 修复更新弹窗中版本配图点击无法放大预览的问题。
- 修复沉浸态与服务审批浮层内容过长时无法通过鼠标滚轮查看底部内容的问题。
- 修复回撤与检查点设置弹窗背景透明、透过内容可见的问题。
- 修复 Windows 平台上团队成员导入时可能因文件监听句柄冲突而失败的问题。
- 修复终端输出净化错误删除部分语言文字和 Emoji 中必需的零宽连字符的问题。
- 修复升级重启提示在非 Windows 平台显示错误文案的问题。
- 修复 macOS/Linux 上点击升级后过渡文案暗示“安装程序”会出现的误导性问题。

## 改进
- 代码托管平台（GitHub/Gitee）的 OAuth 授权迁移至新鉴权服务，登录流程更稳定。
- 记忆系统能力提升暂缓发布，待内部审查完善后再行推出。
- 优化网络断连判断逻辑，减少因 Agent 繁忙导致的误报。
- 「检测更新」菜单项更名为「检测智能体更新」，避免与客户端软件更新混淆。
- 输入框内的工作目录胶囊在切换 Agent 时不再短暂显示旧数据。
- 优化设置面板中回撤与检查点弹窗的背景样式。
