# DesireCore v10.0.23

**发布日期**: 2026-04-01

![v10.0.23 更新概览](/img/changelog/v10.0.23.png)

## 新功能
- **ZIP 智能体导入**：现在导入智能体时除了文件夹外，也支持直接选择 ZIP 压缩包，导入方式更方便。

## 修复
- 修复了 Windows 环境下数据导出时可能因文件占用导致压缩包不完整、无法正常恢复的问题。
- 修复了数据导入时可能将缓存或运行时文件一并覆盖、影响当前运行状态的问题。
- 修复了部分导入失败提示不清晰或识别不准确的问题，导入异常时更容易判断原因。

## 改进
- 优化了数据导出内容，自动排除缓存、日志和可重建文件，使导出的备份包体积显著减小、传输和保存更轻便。
- 提升了导出备份的完整性校验能力，降低生成损坏压缩包后才发现无法使用的风险。
