# DesireCore v10.0.22

**发布日期**: 2026-03-31

![v10.0.22 更新概览](/img/changelog/v10.0.22.png)

## 修复
- 在 Windows 环境下，AI 回复中的 `.md` 文件现在可以正确显示为可点击链接并正常打开，避免路径解析错误导致的显示异常或误匹配问题。
- 在 Windows 和 Linux 上导入技能时，文件选择方式已适配系统限制，您现在可以分别选择 `.md` 文件或技能文件夹完成导入。
- 自动更新过程中不会再频繁弹出重复提示，也不会在下载更新时反复重复检查，更新体验更稳定流畅。

## 改进
- SuperDoc 工具栏在窄屏场景下的按钮显示更稳定，避免操作栏被挤压，横向查看也更顺畅。
