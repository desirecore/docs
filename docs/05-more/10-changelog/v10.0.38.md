# DesireCore v10.0.38

**发布日期**: 2026-04-27

![v10.0.38 更新概览](/img/changelog/v10.0.38.png)

## 新功能
- **Agent 云端发布与导入**：现在你可以在发布 Agent 时选择公开或私有、为已发布的 Agent 一键复制分享链接或转为公开，并支持直接从公开链接导入 Agent。
- **Agent 版本管理**：现在可以通过发布向导为 Agent 创建正式版本，自动递增补丁版本、生成更新记录并发布版本标签，同时面板会显示本地与远程版本状态。
- **媒体生成技能**：新增文生图、文生视频和语音合成等多种媒体生成能力，并会优先匹配与你当前模型供应商一致的技能来减少“用错能力”的情况。
- **SuperDoc 文本改写与检查**：在 SuperDoc 中选中文本后可直接进行简化、扩展、正式化、口语化或修正，还可一键检查 Markdown 结构问题并跳转到对应位置。
- **SuperDoc 引用定位**：当文档问答结果带有行号引用时，你可以直接点击引用跳转到文档对应位置并看到高亮提示。
- **SuperDoc Mermaid 预览**：SuperDoc 现已支持 Mermaid 图表渲染，文档中的图表内容可以更直观地查看。

## 修复
- 长时间未操作后再次发送消息时，不会再出现消息发不出去却一直卡在“思考中”的情况。
- 重试或编辑历史消息时，修复了可能误删大量聊天记录的问题。
- 清除对话历史时，现在会更彻底地清理各类历史运行记录，避免刷新后旧内容“又回来”。
- AskUserQuestion 不再以漂浮弹窗覆盖所有标签页和对话，而是只在对应会话内显示，并在跨会话或失焦时通过提醒引导你处理。
- 多个 Agent 并行工作时，后台文档或工作流更新不再突然抢占你当前正在查看的会话面板。
- 未配置 API Key 等配置错误出现后，“加载工具和技能”等状态会正常结束，不会一直停留在界面上。
- 手动暂停已结束的会话时，不会再误报“会话不存在或已停止”之类的错误。
- 重新发送失败消息后，之前残留的错误提示现在会被正确清除。
- 当同一协议下配置了多个同名模型来源时，模型请求会更准确地使用你实际选择的供应商凭据，避免串用错误的 API Key。
- 本地视频与其他本地媒体现在可以更稳定地在对话中内联播放，视频上传也已支持。
- 审计日志很多时，审计页面不再因数据量过大而明显卡顿甚至崩溃。
- SuperDoc 中部分功能在分屏或窄窗口下按钮不可见、检查无效、选区改写不可用、引用跳转报错等问题已修复。
- Agent 修改文档后偶发 diff 丢失的问题已修复，文档改动更容易被正确展示和审阅。
- 编辑文档时，YAML frontmatter 不会再被错误改写为分隔线格式。

## 改进
- 长会话的首屏加载和滚动渲染更加稳定，打开超长对话时会更流畅。
- 对话中的长内容、Markdown 和过程消息渲染性能得到优化，长聊天记录下的阅读体验更顺滑。
- SuperDoc 工具栏在窄屏下改为更紧凑的图标模式，常用操作更不容易被挤出界面。
- 媒体内容在消息中的展示与供应商兼容性进一步优化，生成结果更容易直接预览和使用。
