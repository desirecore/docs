# DesireCore v10.0.68

**发布日期**: 2026-05-27

![v10.0.68 更新概览](/img/changelog/v10.0.68.png)

## 新功能
- **云端账号登录与自动算力绑定**：现在你可以直接用邮箱注册和登录 DesireCore，登录后系统会自动绑定你的云端模型与免费额度，并在后续自动同步可用模型列表。
- **版本公告弹窗**：升级到新版本后，首次启动会弹出一次公告，帮助你快速了解云端账号上线和旧内测公共算力下线等重要变化。
- **SuperDoc 导出 DOCX**：SuperDoc 现在支持将内容直接导出为 DOCX 文件，可用 Word、WPS 等常见办公软件打开。
- **Agent 自动压缩当前会话**：当上下文变长时，Agent 现在可以主动整理和压缩当前对话，帮助更高效地继续任务。

## 修复
- 重试生成回复时，之前已输出的文本和已完成的工具调用现在会被保留，不会因为自动重试而从头再来。
- 修复了长时间会话在重试、中断、异常退出或恢复后可能丢失部分消息记录的问题，历史内容更完整可靠。
- 修复了重试消息时图片附件可能丢失的问题，带图对话的再次生成更稳定。
- 修复了导出 Agent 时可能遗漏对话记录的问题，导入后可更完整保留原有交流历史。
- 修复了 DeepSeek 推理模式下历史 reasoning 内容可能丢失的问题，连续推理对话更稳定。
- 修复了云端模型同步中部分智谱模型可能报错或中断的问题，相关模型的对话可用性更好。
- 修复了版本公告在部分场景下可能因版本号未注入而无法正常显示的问题。

## 改进
- 手动点击“压缩对话”时，现在即使对话轮数不多也会尽量执行压缩，处理长文本、长代码场景更实用。
- 未登录或模型不可用时，发送消息前会先给出明确提示并保留你的文字和附件草稿，避免内容白白丢失。
- 与登录、算力不足或模型不可用相关的系统提示更智能，会根据你的登录状态提供更合适的引导操作。
- 资源面板新增更清晰的登录引导和云端模型同步状态提示，让你更容易知道何时该登录以及模型是否正在更新。
- 登录弹窗与账号流程体验得到优化，包括注册/登录切换、邮箱验证提示、密码管理器支持和记住我逻辑，整体使用更顺手。
- 旧的内测公共算力入口已完成下线清理，界面与模型列表会更聚焦于正式可用的云端账号能力。
