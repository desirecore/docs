## v10.1.0（开发中）

DesireCore 的首个公开版本，包含以下核心功能：

### 新功能

- **委派式交互**：通过六原语（教规则、给示例、下达任务、确认、反馈、遗忘）与智能体互动
- **智能体系统**：支持创建、编辑、导入自定义智能体
- **三域记忆**：核心记忆、关系记忆、共享记忆
- **超级文书**：类似 Code Review 的文档协作模式
- **多供应商支持**：连接 OpenAI、Anthropic、DeepSeek 等 20+ AI 供应商
- **跨平台支持**：macOS、Windows、Linux 桌面应用
- **本地优先**：所有数据默认存储在本地，保护隐私

### 已知限制

- 部分高级调度功能待后续版本推出
