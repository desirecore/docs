# DesireCore v10.0.34

**发布日期**: 2026-04-17

![v10.0.34 更新概览](/img/changelog/v10.0.34.png)

## 修复
- 对话在流式回复中途报错时，错误提示现在会稳定显示在本轮 AI 回复之后，不再出现覆盖回复内容、缺失提示或顺序错乱的问题。
- 刷新或重启应用后，对话中的错误消息顺序会保持正确，不会再回退到用户消息下方。
- 核心智能体现在可以在创建团队时显式指定组长，避免因默认使用自身身份而导致创建失败。
- 修复了浅色模式下“上下文已隔离”分隔线几乎不可见的问题，消息流中的分隔标识更清晰。
- 优化了工具执行区域的头部排版，多个工具名与文件徽章同时出现时，标题不再被挤压得异常换行。

## 改进
- 创建团队时，组长限制说明更加明确，相关操作在填写参数和使用时更易理解。
