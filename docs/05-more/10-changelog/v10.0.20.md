# DesireCore v10.0.20

**发布日期**: 2026-03-30

![v10.0.20 更新概览](/img/changelog/v10.0.20.png)

## 修复
- 修复了从加号菜单创建智能体时弹窗可能一直停留在“正在创建智能体”且没有成功或失败反馈的问题。
- 修复了通过兼容代理使用流式模型时，审计详情中的输入 Token 与输出 Token 统计经常显示为 0 或不准确的问题。
- 修复了在部分网关直通 Anthropic 协议响应时，审计详情无法正确记录输入 Token 的问题。

## 改进
- **更新检查体验**：优化了桌面端更新检查机制，在网络波动或服务异常时能更快返回结果并自动降级，减少更新检查失败或长时间等待的情况。
