# DesireCore v10.0.95

**发布日期**: 2026-07-23

![v10.0.95 更新概览](/img/changelog/v10.0.95.png)

## 新功能
- **手动会话 Project 与工作树隔离**：手动会话现在支持创建和绑定 Project，可基于 Git 工作树为每个会话提供独立的文件环境，并确保会话结束时安全清理。
- **MCP OAuth 认证**：远程 MCP 服务现支持标准 OAuth 2.1 授权流程，无需手动填写令牌，可在界面中直接完成连接授权。
- **默认算力图像生成**：无需额外配置，可直接在对话中描述需求生成图片，生成结果实时显示在回复中。
- **图片全屏预览**：对话中的图片支持点击全屏查看，并可在预览模式下缩放、拖拽和下载原图。
- **消息列表快捷跳转**：当聊天消息较多时，右下角会自动显示“回到顶部”和“回到底部”的悬浮按钮，方便快速定位。
- **记忆自动沉淀与精修**：当用户说出“记住…”时，系统会自动将内容沉淀为记忆并显示记忆卡片；后台新增精修机制，可持续提升记忆分类的准确性。
- **Plan 文件协议与团队工作目录**：Agent 的计划以可追溯的文件形式保存在工作目录中；团队共享工作目录现在会真正被 Agent 在执行任务时使用，并支持“仅团队目录”模式以增强数据隔离。
- **CDP 远程调试**：生产打包版可通过环境变量显式开启 Chrome 调试端口，并支持绑定到局域网地址，方便远程诊断与测试。

## 修复
- 修复终端面板关闭后重新打开时出现乱码的问题。
- 修复市场详情页内容过长时安装按钮无法点击，以及安装弹窗无法滚动的问题。
- 修复因先前版本过度收紧 Agent 能力边界，导致邮件发送等内部服务不可用的 bug。
- 修复记忆系统将“不要记住”等否定语句错误沉淀为记忆的问题。

## 改进
- 聊天输入框旁的终端入口现在始终显示，无终端时呈灰色空闲态，多个终端时自动折叠为聚合入口，不再占用过多空间。
- 全屏图片预览支持鼠标滚轮缩放、拖拽平移和下载原图。
- 市场详情页改为两栏布局，安装按钮固定可见；未安装的卡片不再显示多余的安装状态徽章。
- 记忆系统在算力未就绪时不会丢弃整理任务，等算力恢复后继续执行，提高了可靠性。
- 手动会话话题列表支持按 Project 分组、置顶、优先级和多种排序方式整理。
