# DesireCore v10.0.15

**发布日期**: 2026-03-22

![v10.0.15 更新概览](/img/changelog/v10.0.15.png)

## 新功能
- **智能体文件夹导入**：现在在桌面版中可以直接选择本地文件夹导入智能体，无需先打包成 ZIP，导入流程更顺手且兼容 Windows。
- **系统默认 MCP 工具**：所有智能体默认可直接使用内置的浏览器开发工具能力，无需手动配置即可调用相关 MCP 服务。
- **SuperDoc 分栏调节**：SuperDoc 编辑器支持拖动调整编辑区与聊天区宽度，并会在空间不足时自动隐藏预览面板，分屏编辑更灵活。

## 修复
- 解决了切换话题后点击旧消息“重试”无反应的问题，同时修复了重试时媒体附件可能丢失的情况。
- 解决了重试或编辑后重发时旧消息未被正确清除的问题，避免刷新后历史中再次出现重复内容。
- 修复了聊天记录弹窗只能显示最近部分消息的问题，现在可以完整查看更早的对话内容。
- 修复了应用重启或刷新后，流式回复中的用量统计会从 0 重新开始显示的问题。
- 修复了暂停生成后顶部活动状态仍持续显示的问题，停止后界面状态会立即恢复。
- 修复了对话中的 `.md` 文件链接在刷新后无法点击打开的问题。
- 修复了市场页和市场详情页中智能体头像文字显示为黑色的问题，头像展示更清晰一致。
- 修复了 Windows 环境下智能体创建、导入和删除过程中出现的兼容性问题。

## 改进
- 优化了 MCP 服务的可用性，打包后的桌面版对命令行环境识别更完整，工具参数兼容性也更好，减少调用失败情况。
- 提升了大体积请求和文件上传的稳定性，处理大上下文内容或较大附件时更不容易出现请求过大报错。
