# DesireCore v10.0.18

**发布日期**: 2026-03-29

![v10.0.18 更新概览](/img/changelog/v10.0.18.png)

## 新功能
- **更新说明弹窗**：应用下载更新后，现在会直接展示你错过版本的更新日志和配图，并支持按版本分层查看与大图预览。
- **团队组织架构图**：智能体社会关系新增“团队”视图，可更直观地查看团队、组长、成员、未分组智能体以及上下级关系。
- **团队管理增强**：你现在可以在团队设置中一键提升组长、批量添加成员、批量移除成员，团队维护更高效。
- **工具标签交互**：在智能体档案中，鼠标悬停工具标签可查看用途说明，点击后还能直接跳转到对应的技能/工具管理页面。

## 修复
- 修复智能体档案中“可用工具”列表始终为空的问题，现在会正确显示该智能体实际可用的内置工具和自定义工具。
- 修复默认工作目录被所有智能体共用的问题，现在每个智能体都会使用各自独立的工作区，并自动接入所属团队的共享目录。
- 修复运行状态面板“最近活动”中已完成条目的耗时会持续增长、刷新前不显示或出现重复的问题，现在活动记录与耗时显示更准确稳定。
- 修复后台任务时间显示异常的问题，已完成任务不再出现不断跳动的计时。
- 修复团队组织架构图在组长被删除等异常情况下可能崩溃的问题，团队视图显示更加稳定。
- 修复欢迎页隐私说明在浅色背景下不易看清的问题，关键信息现在更易阅读。

## 改进
- **指挥链展示**：团队视图中新增“用户 → DesireCore → 各团队组长”的指挥链层级，组织关系更清晰，同时 DesireCore 不再被视为可加入团队的成员。
- 会话列表中，团队组长现在会被提升到通用分组中展示，便于你更快找到关键负责人。
- 团队组织架构图的连线方向、样式与图例说明得到优化，层级关系比之前更直观。
- 团队规则进一步收紧，同一位智能体不能同时担任多个团队的组长，避免组织配置冲突。
- 欢迎页的隐私说明改为更清晰的分点展示，并明确说明日志会经过脱敏处理。
