# DesireCore v10.0.18

**发布日期**: 2026-03-29

![v10.0.18 更新概览](/img/changelog/v10.0.18.png)

## 新功能
- **桌面端 Agent 操作系统正式上线**：你现在可以在一个统一的桌面应用中完成智能体创建与管理、自然语言对话、工具调用、记忆使用、技能扩展和多服务协作。
- **智能体市场与安装分发**：你可以浏览市场中的智能体与技能内容，并将其安装到本地快速使用。
- **多模型与算力资源管理**：你可以在应用内接入和管理多家模型服务与算力节点，更灵活地为不同任务选择合适的模型。
- **技能与工具中心**：你可以导入、创建、编辑和管理技能、MCP 服务及工具配置，扩展智能体的可执行能力。
- **长期记忆与知识资源体系**：智能体现已支持结合长期记忆、知识内容及多类资源进行任务处理，提升连续对话和复杂任务表现。
- **审批与权限治理**：你可以为高风险操作设置审批流程与权限控制，在效率与安全之间获得更好的平衡。
- **邮件中心**：你可以在 DesireCore 中接入邮箱、查看和搜索邮件、撰写回复并处理附件，让智能体参与邮件相关工作。
- **自动化日程与调度**：你可以创建定时任务和自动化流程，让智能体按计划执行工作并查看历史记录。
- **代码仓库集成**：你可以连接代码仓库并在应用内查看、管理和协助处理代码相关任务。
- **活动审计与运行追踪**：你可以查看智能体调用工具、访问服务和生成结果的过程明细，更方便地追踪执行情况。
- **数字人交互体验**：你可以通过数字人界面、语音播报和实时字幕与智能体进行更具沉浸感的互动。
- **团队与关系协作视图**：你可以管理团队成员、查看关系网络与协作状态，更直观地组织多智能体协作。
- **文档与版本能力**：你可以在内置文档界面中查看差异、导出内容和管理版本，便于沉淀与复用工作成果。
- **更新日志与版本提示**：应用现已支持查看版本更新内容，帮助你快速了解每次升级带来的变化。

## 改进
- 对话界面整体体验得到完善，包括流式回复、消息展示、输入区、历史记录和运行状态反馈在内的日常使用流程更加顺畅。
- 智能体资料页、配置页和导入发布流程更加完整，新建、编辑、分享和同步智能体时会更直观。
- 搜索、筛选和导航能力得到增强，你可以更快地在会话、资源、技能和智能体中找到所需内容。
- 设置中心更加完善，现已提供外观、语言、通知、快捷键、隐私和权限等更完整的桌面端配置能力。
- 桌面应用的窗口、标题栏、通知提醒、截图和环境检查等基础体验已补齐，整体可用性更接近成熟桌面软件。
- 多语言内容与界面文案已全面覆盖，中文环境下的使用体验更加一致完整。
