# DesireCore v10.0.91

**发布日期**: 2026-07-20

![v10.0.91 更新概览](/img/changelog/v10.0.91.png)

## 新功能
- **推理深度选项**：为 GPT-5.6 系列模型新增可选的推理深度（快/中/高/极高/最大），可在输入区灵活切换，控制回复的思考深度。
- **技能安装版本门控**：安装市场技能时自动校验客户端版本，阻止不兼容的技能安装并引导升级，避免因版本不匹配导致的功能异常。

## 修复
- 修复继续生成时整篇重写、编辑重发后旧回复残留、历史分支内容无法正确恢复、消息重复等问题，确保多轮对话的连贯性。
- 修复服务重启后审批卡片仍显示为待处理状态、停止会话时误报为用户拒绝、委派任务审批模式意外降级为全局策略等问题。
- 修复应用安装状态无法持久化、技能详情页安装按钮状态不更新、无远程设备时缺少本机安装入口、技能计数重复显示等市场与安装体验问题。
- 修复后台分析任务（如话题摘要）因输出上限错误被吞没而反复失败、无法自愈的问题，提升内部处理稳定性。
- 修复本地模式下 Agent 通过技能调用本机 API 时出现 401 错误的问题，恢复本机服务的正常集成。
- 修复多个弹窗无法通过 Esc 键关闭、Esc 事件误触发其他面板、注册时接受无法收件的示例邮箱、工作记忆面板缺失用户私有记忆等界面与交互缺陷。

## 改进
- 应用安装与卸载过程现展示更细致的中间状态（如卸载中、重装中），服务在操作完成前不再被意外清理，状态反馈更精准。
- 优化智能体管理技能的提示词，去除硬性的提问次数限制，由 Agent 自主把握交互节奏，对话更自然。
- 模型配置更新：自动下线已过期的模型并可回退到同类型可用模型，同时新增 GPT-5.6 系列模型支持。
- 官方云账户的话题自动摘要改用更低成本模型，减少后台任务的算力消耗。
- 运行环境面板新增系统环境说明，明确区分本地工具与托管运行时，避免“未安装”提示引起误解。
- 增强本地服务安全性，默认仅开放必要的内部接口。
