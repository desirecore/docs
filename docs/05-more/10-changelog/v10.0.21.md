# DesireCore v10.0.21

**发布日期**: 2026-03-31

![v10.0.21 更新概览](/img/changelog/v10.0.21.png)

## 修复
- 修复了 Windows 上因用户目录路径解析异常导致的 Agent 转交、路由和 Hook 执行失败问题。
- 修复了 HttpRequest 工具在处理重定向时可能报错或请求行为异常的问题，使相关网络请求在 Windows 环境下更稳定可用。
- 修复了代码仓库、网页抓取、网页搜索、Ollama 同步、工具下载等多个场景中偶发的网络请求报错问题，减少“请求结束但未返回内容”一类异常。
