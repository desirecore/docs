# DesireCore v10.0.31

**发布日期**: 2026-04-14

![v10.0.31 更新概览](/img/changelog/v10.0.31.png)

## 新功能
- **用户消息卡片快捷操作**：用户消息卡片现在支持独立复制和朗读，复制带附件的消息时还会一并附上文件路径，便于复用内容和定位文件。

## 修复
- 图片上传与发送体验已修复，应用现在能更准确识别图片内容、正确判断当前模型是否支持识图，并避免纯图片消息在历史记录中出现多余提示文本。
- 导入智能体时，如果遇到相同 UUID 的现有智能体，现在可以直接选择覆盖导入，不再只能中断操作。
- 导出数据时，导出中的提示会在你选定保存位置后再出现，且成功、失败提示显示更稳定，不容易一闪而过。
- 运行环境检测已修复，打包后的应用在更多系统环境下也能正确识别 Node.js、npm 等运行时，减少环境明明已安装却显示异常的问题。
- 连接 MCP 服务前会先检查 Node.js 是否可用，缺失时会立即给出明确提示，避免长时间等待后才超时失败。
- MCP 工具发现失败后的重试机制已修复，临时失败不会被错误缓存，后续查询更容易自动恢复。

## 改进
- 运行环境页面优化了加载状态显示，版本信息加载过程中会明确展示加载中反馈，避免误以为当前环境没有可用版本。
