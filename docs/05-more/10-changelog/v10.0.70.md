# DesireCore v10.0.70

**发布日期**: 2026-06-02

![v10.0.70 更新概览](/img/changelog/v10.0.70.png)

## 新功能
- **服务目录治理与自助注册**：现在 Agent 可以注册外部服务并在应用内发起审批、上架或拒绝，已安装应用暴露的服务也会自动出现在服务目录中并显示健康状态。
- **隐藏式开发者模式**：在“设置 > 关于”连续点击版本号 7 次即可开启开发者模式，并通过独立入口管理调试功能、切换登录后端和发送测试通知。
- **模型规格自动补全**：新增和同步模型时会自动补全上下文长度、输出上限、能力标签等关键信息，手动添加模型时输入名称后也会自动预填可识别字段。
- **模型信息悬浮卡**：在聊天中的模型选择列表悬停模型后，可查看模型简介、上下文窗口和完整能力标签，选型更直观。
- **模型列表按厂商整理**：模型选择列表会按厂商聚合并按名称排序，查找同系列模型更方便。
- **全聊天区文件拖拽上传**：现在可以把文件拖到整个聊天区域而不只是输入框，并会看到明确的高亮提示。
- **回复中的 Markdown 文件路径可直接打开**：当 Agent 在回复里给出它实际生成过的 `.md` 文件路径时，点击即可直接在编辑器中打开。
- **技术图生成技能**：技能市场新增 tech-diagram 技能，可直接把自然语言描述生成架构图、流程图、时序图等多种 Mermaid 技术图，并支持多套视觉风格。
- **服务注册与使用元技能**：技能市场新增用于注册服务和使用服务的元技能，方便 Agent 更自然地接入外部服务能力。

## 修复
- 修复了多 Agent 并行协作时可能跨团队错误委派的问题；现在跨团队 fan-out 会自动拦截、过滤或向你发起审批，避免意外激活不相关 Agent。
- 修复了超长对话容易逼近模型上下文上限的问题；系统会自动压缩旧内容并保留摘要，降低因上下文过长导致响应异常的概率。
- 修复了 Windows 和 Linux 上通知中心顶部操作与窗口控制按钮重叠的问题，避免“全部已读/清除全部”难以点击。
- 修复了会话列表时间显示与聊天内时间不一致、刷新后首次进入会跳变的问题，日期展示现在更准确稳定。
- 修复了配置中心和服务目录同步时可能因本地冲突标记导致解析失败甚至服务启动异常的问题。
- 修复了浏览器能力在多 Chromium 浏览器并存时可能串到错误实例的问题，并补全了对 Edge 的支持及部分网页跳转异常。
- 修复了技能锁文件损坏时保护机制可能静默失效的问题，降低已修改技能被意外覆盖的风险。

## 改进
- 云端模型的上下文窗口、最大输出、能力标签等信息现在由统一规格库维护，模型参数更准确，且可随配置更新自动生效。
- 模型说明内容做了中文展示优化，减少中英混杂带来的阅读干扰。
- 所有按钮悬停时统一显示手型光标，交互反馈更加一致。
