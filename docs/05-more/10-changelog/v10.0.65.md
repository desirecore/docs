# DesireCore v10.0.65

**发布日期**: 2026-05-19

![v10.0.65 更新概览](/img/changelog/v10.0.65.png)

## 新功能
- **聊天公式渲染**：聊天消息中的 Markdown 现已支持行内公式和块级公式显示，数学内容会以更清晰易读的排版呈现。

## 修复
- 修复了部分智能体工具调用场景下会误报“工具并发问题”并导致请求失败的问题，复杂多轮工具交互现在更稳定。
- 修复了工具调用相关报错信息不准确的问题，遇到请求格式错误时现在会更真实地显示上游返回原因，便于排查。

## 改进
- 优化了聊天气泡内公式内容的显示效果，长公式在消息中浏览更顺畅，不易出现排版溢出。
