# DesireCore v10.0.12

**发布日期**: 2026-03-19

![v10.0.12 更新概览](/img/changelog/v10.0.12.png)

## 新功能
- **AI 图片展示与保存**：智能体或工具生成的图片现在会自动显示在对话中，支持悬停一键保存到本地，刷新后也能继续查看历史图片。
- **macOS 截图权限引导**：在 macOS 上首次使用截图功能时，如未授予屏幕录制权限，应用会主动提示并引导你前往系统设置完成授权。
- **Gateway Protocol v3.6 兼容**：连接新版 HostAgent/Gateway 后，可正常处理文本、图片和无返回值等不同类型的命令结果，远程能力兼容性更完整。

## 修复
- Windows 上智能体详情页中的 Markdown、JSON、YAML 等文件现在可以正常点击打开，不再因为路径分隔符问题无响应。
- 修复了暂停某个智能体生成后切换到其他对话仍显示“继续生成”按钮的问题，不同对话的生成状态现在彼此独立。
- 修复了验证图片生成、语音克隆等非对话类模型时容易报错的问题，这类模型现在可以正常完成连通性校验。
- 修复了资源管理器面板内容过多时无法正常滚动的问题，展开较多内容时也不会超出窗口难以操作。

## 改进
- 资源管理器中的工作目录在新增、删除或切换主目录后会立即刷新显示，“默认”标记也会准确跟随当前主目录。
- 模型选择器和资源面板在配置多个自定义模型服务时显示更稳定，减少异常提示并提升选择体验。
- Linux 版本新增了 deb 安装包，安装与桌面图标显示体验更完善。
