# DesireCore v10.0.25

**发布日期**: 2026-04-08

![v10.0.25 更新概览](/img/changelog/v10.0.25.png)

## 新功能
- **Agent 主动消息**：现在智能体可以在执行过程中主动向你发送结构化消息，并附带文件或图片附件，让任务进展、结果说明和补充材料传达得更清晰。
- **会话内等待能力**：智能体现在支持在当前会话中按秒级原地等待后继续执行任务，适合轮询、短暂等待外部结果或按你的要求稍后继续。

## 修复
- 修复了技能管理中“导入技能”在部分环境下无法使用的问题，且导入失败时会直接显示错误原因，不再出现“点了没反应”。
- 修复了 Windows 环境下技能导入的路径校验问题，提升了本地导入过程的兼容性与安全性。

## 改进
- **技能导入体验升级**：现在你可以直接通过 `.md`、文件夹或 `.zip` 包导入技能，导入流程更统一，也更适合批量选择和迁移技能。
- 对话流式输出更顺滑，在长文本生成过程中界面刷新更稳定，减少了卡顿和频繁重绘的感觉。
- 优化了应用空闲时的资源占用，在线状态等界面元素不再导致明显的 CPU 持续消耗。
- 智能体现在更容易理解自己可以扩展技能能力，因此当你询问安装新技能、从市场获取技能或通过 URL/Git 导入时，回应会更准确。
- 智能体在读取、搜索、编辑、写入文件和执行命令时的行为更稳健，能更合理地选择工具、减少误用，并在 Notebook 等内容处理上表现更好。
- 智能体对“等待”和“定时调度”两类能力的区分更清楚，在连续任务与异步提醒场景下会更合理地选择执行方式。
- 运行时引擎进一步统一，旧版本 Agent 配置会自动兼容迁移，整体运行体验更一致。
