# DesireCore v10.0.42

**发布日期**: 2026-04-28

![v10.0.42 更新概览](/img/changelog/v10.0.42.png)

## 修复
- 在桌面应用中已彻底禁用前端页面刷新快捷键，避免误触后导致当前界面被意外刷新或状态中断。
- 修复了长对话中消息列表滚动不稳定的问题，浏览历史消息、拖动滚动条或切回会话时不再容易跳动、错位或丢失当前位置。

## 改进
- **长会话性能优化**：针对长对话场景优化了消息列表渲染与虚拟化加载，打开大型会话、切换会话以及首屏展示会更快更流畅。
- **会话预览体验提升**：优化了长会话的历史预览与恢复逻辑，重新进入会话时能更顺畅地看到最近内容并延续之前的浏览位置。
- **消息内容渲染更稳定**：包含长文本、Markdown 和过程消息在内的复杂消息展示更加平稳，减少渲染卡顿对阅读体验的影响。
