# DesireCore v10.0.40

**发布日期**: 2026-04-28

![v10.0.40 更新概览](/img/changelog/v10.0.40.png)

## 修复
- 修复了多轮对话中追问可能一直卡在“思考中”无法继续的问题，后续回复现在可以更稳定地正常返回。
- 修复了断网、超时等流式请求异常时的重试处理，失败后的提示信息更准确，恢复网络后对话也更顺畅。
- 修复了自定义模型接入时的兼容性问题，使用部分国内模型服务商时，输出长度限制和推理模式现在能够更正确地生效。

## 改进
- 优化了模型请求的兼容处理，像 Kimi 等兼容接口在开启或关闭深度思考时表现更稳定、更符合对应服务商的实际能力。
