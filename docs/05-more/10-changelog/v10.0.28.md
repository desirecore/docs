# DesireCore v10.0.28

**发布日期**: 2026-04-13

![v10.0.28 更新概览](/img/changelog/v10.0.28.png)

## 新功能
- **结构化追问与确认交互**：当智能体需要你补充信息、做选择或确认高风险操作时，现在可以直接在对话中发起结构化提问，并通过专门的交互卡片完成回复、补充说明或跳过。
- **Git 运行环境检测**：在“资源管理器 → 算力资源 → 运行环境”中，新增了 Git 检测信息，可查看是否已安装 Git、版本与路径，以及当前使用的是原生 Git 还是内置兼容后端。

## 修复
- 修复了部分非 Anthropic 模型无法正确使用“向用户提问”能力的问题，现在智能体在更多模型下都能正常向你发起澄清和确认。
- 修复了“向用户发送消息”在多段落内容中重复显示的问题，长消息的展示更加准确、不再重复。
- 修复了 Git 环境卡片可能同时出现“原生 Git”和“未安装”等矛盾状态的问题，环境检测结果更加一致可靠。
- 修复了应用长时间持续运行时当天活跃统计可能漏记的问题，跨天使用时统计更加准确。

## 改进
- **对话交互体验优化**：智能体向你提问的内容现在支持从历史记录中恢复，并能把你的回答合并回原始卡片中，查看上下文更加连贯。
- 当智能体提问的后端流程异常或长时间无响应时，界面会在稍后提供“跳过”入口，避免你被卡在无法关闭的交互中。
- 优化了 Git 环境卡片的信息展示方式，未安装时会明确显示“未安装”，整体表达更清晰、与其他环境项保持一致。
- 优化了模型服务配置中的算力选项显示与映射补全，相关模型选择和默认服务展示更加完整。
- 优化了 Anthropic 模型下的工具调用缓存策略，在重复使用工具定义的场景中可带来更高效率与更低成本。
