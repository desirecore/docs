# DesireCore v10.0.83

**发布日期**: 2026-07-12

![v10.0.83 更新概览](/img/changelog/v10.0.83.png)

## 新功能
- **Claude 订阅技术适配**：新增通过官方 Claude Agent SDK 连接 Claude 会话的技术路径。使用官方 SDK 不等于获得第三方转发 Pro/Max 凭据的授权；按 Anthropic 当前规则，该入口在取得适用于 DesireCore 的明确授权前保持发布门控。
- **Claude SDK 后端优化**：对话历史可以续接已缓存的 Claude 会话并传递图片。当前能力边界不启用 SDK 内置工具，搜索、文件、Shell 等操作仍由 DesireCore 工具和审批链负责。
- **Windows GUI 自动化**：Windows 客户端内置 cua-driver，Agent 可直接操作原生应用界面（截图、点击、输入），零安装开箱即用。
- **三层提示词体系**：新增全局层（用户级）和团队层提示词，与 Agent 层叠加生效；提示词中心统一管理三层内容，支持结构化编辑与实时预览。
- **思考深度控制**：输入区新增思考深度选择按钮（自动/关闭/五档深度），支持会话级、Agent 级和全局默认三级优先级，适配 OpenAI、Anthropic 及国产模型方言。
- **算力预置更新通知**：用户自定义的 Provider 配置不会被上游更新静默覆盖，产生冲突时在审阅面板逐项对比决策，支持一键采用上游值。
- **数据迁移到新设备**：导出向导可打包智能体、配置和 API 密钥（加密保护）迁移到新设备，新机导入时自动识别并完成身份替换或数据归并。
- **文件"打开方式"菜单**：聊天中的文件条目新增"打开方式"下拉，可选择应用内打开、系统默认应用、其他应用或在文件夹中定位。
- **应用与服务手动刷新**：页面标题栏新增刷新按钮，可手动同步服务目录状态。
- **对话 PDF 导出**：改为客户端本地渲染，导出更稳定。
- **内置运行时环境**：Python 3.13 和 Node.js 24 LTS 随安装包内置，首启静默导入，完全离线可用。
- **AgentFS 回撤升级**：Agent 文件系统回撤改用 Git 全树快照，支持整体还原，且应用内置 Git 确保无系统依赖。
- **视频生成接入豆包 Seedance 2.0**：视频生成中转适配豆包 Seedance 模型，支持首尾帧和多参考图。
- **官方云算力接入快乐马视频**：官方云算力用户可直接使用 happyhorse-1.1 视频生成和 OpenAI 图像生成。
- **套餐折扣展示**：购买页面支持显示原价划线和折扣信息。

## 修复
- **智能体列表状态修复**：导入/克隆的智能体现在立即显示为在线，不再需要重启应用；聊天侧边栏在智能体改名后正确刷新名称。
- **委派工具多项修复**：并行委派不再跳过目标校验，无效目标会明确报错；委派卡片正确显示多目标名称；AgentFS 文件监听恢复即改即生效。
- **全局并发槽泄漏修复**：停止生成不再导致会话卡死在"排队中"，系统休眠唤醒后不再耗尽并发槽。
- **定时任务可靠性修复**：定时任务到期后真正执行完整流程，不再只回一句就被截断；失败任务正确退避重试，不再无限重刷；一次性任务杜绝重复触发。
- **上下文超限透明自愈**：云端模型实际上下文窗口与声明不符时，系统从报错中学习真实上限并自动调整，大输出一次成型不再分步续写。
- **思考深度档位修复**：最深档位的预算不再塌缩，各档位区分度恢复正常；选回默认档位后不再残留之前的深度设置。
- **同名模型路由修复**：官方云算力与本地 Provider 存在同名模型时，请求精确路由到目标 Provider，不再串台。
- **扫描件退化修复**：非视觉模型读取纯扫描件时直接提示切换模型，不再陷入无限重复输出。
- **Windows 路径显示修复**：系统提示中的 Windows 路径不再丢失反斜杠，复制后保持正确。
- **聊天消息区滚动条加宽**：方便鼠标拖拽。
- **消息列表闪烁修复**：滚到底部继续滑动不再导致整窗闪烁。
- **横向滚动条消除**：长 URL 折行显示，宽表格内部横滚，内容始终完整可见。
- **文件链接崩溃修复**：点击 Agent 回复中的本地文件链接不再导致窗口崩溃，文件交由系统默认应用打开。
- **任务板启动噪音消除**：启动和切换会话时不再产生 404 报错日志。
- **市场仓库源保存修复**：添加仓库源后不再卡在"保存中"或重复显示。
- **服务目录状态实时更新**：注册服务后目录自动刷新，不再需手动操作。
- **关系图谱补齐交付边**：Agent 成功交付结果的关系边现在正确显示。
- **输入框光标跟随**：换行、历史回溯等操作后光标自动滚入可视区域。
- **消息操作栏溢出修复**：短消息的操作按钮不再盖住时间戳。
- **多选模式修复**：多选模式下点击选择圆圈可靠响应，不再被视频播放器等元素拦截。
- **关系热度显示优化**：热度数值以千分位加一位小数展示，不再撑破卡片。
- **Windows 清除数据修复**：应用内清除数据不再因只读 Git 文件或保留设备名文件失败。
- **macOS 构建修复**：解决公证被拒和符号链接损坏导致的打包失败。
- **大文件发布修复**：大安装包上传改用分片上传，慢链路上不再超时卡死。

## 改进
- **Provider 并发限流加固**：支持配置单个 Provider 的最大并发请求数和排队策略，避免上游限流，并提供实时诊断端点。
- **排队中的查询可中止**：等待队列中的查询现在可以正常取消，交互式查询优先于后台任务执行。
- **全局并发上限可配置**：顶层并发查询上限默认 3，可通过环境变量调整。
- **Ls 工具免审批执行**：列出目录内容不再需要审批，权限判断交给操作系统；访问受限目录时提供清晰的授权引导。
- **委派子任务超时放宽**：同步等待超时从 5 分钟延长到 30 分钟，超时后不再自动中止子任务，交由调用方决定继续或结束。
- **API 密钥安全加固**：Agent 只能写入密钥不可读取，UI 中密钥默认掩码，文件工具精确拦截读取密钥文件。
- **预置配置版本可见**：算力页面头部显示当前预置数据版本号，便于排查配置同步问题。
- **算力页公告与模型分区**：官方算力模型按文本、图片生成、视频生成分区展示，生成类模型标注"对话不可用"。
- **模型价格展示**：按 Token 计价的模型显示相对倍率，其他计费模式显示单价与计价基准，顶部展示供应商信息。
- **AgentFS 写路径加固**：并发修改 Agent 配置时不再丢失更新，崩溃中断写入不留半写文件。
- **上下文裁剪优化**：裁剪预算更精确，可重取工具的结果被卸载时保留重取指引，自动压缩和记忆注入确保送达模型。
- **系统提示词管理优化**：支持 Agent 级能力声明开关，persona 编辑提供结构化表单，提示词预览按分区可视化占比。
- **SSRF 防护逐 Agent 可控**：每个 Agent 可独立配置外网访问限制，防止 DNS 重绑定攻击。
- **DNS pinning 防护**：Agent 外呼请求在真正发送前复核目标 IP，防止域名在注册后被篡改指向内网。
- **导入导出安全性增强**：防止恶意 Zip 包越界写入，导入失败后服务自动恢复。
- **凭据源兼容性加固**：新增凭据源类型不再导致老客户端崩溃或算力不可用，不支持的凭据源显示明确升级提示。
- **Provider 版本门槛**：上游 Provider 预设可声明最低客户端版本，不满足时展示升级提示而非静默失效。
- **默认模型路径修复**：历史数据中 Provider 标识缺失时自动修复，显式指定的 Provider 不再被错误路由到其他 Provider。
- **任务调度时间修正**：每轮对话自动注入当前时间，Agent 不再基于会话开始时的"旧时间"推算。
