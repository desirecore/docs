# DesireCore v10.0.83

**发布日期**: 2026-07-11

## 新功能
- **Claude 订阅接入**：支持通过 Claude 账号登录使用 Claude 算力，提供本机登录、应用内 OAuth、手动 token 三种凭证方式，模型请求经官方 SDK 后端履约，审计链路完整
- **Windows GUI 自动化**：Windows 客户端内置 cua-driver，Agent 可直接操作桌面应用（截图、点击、输入、读取窗口状态），开箱即用无需安装
- **思考深度控制**：聊天输入区新增思考档位按钮，支持 auto/off/浅/中/深/最深 档位选择，会话独立记忆，兼容 OpenAI/Anthropic 双格式与各大模型厂商
- **三层提示词体系**：新增全局层（用户级）、团队层、Agent 层三层叠加式系统提示词，支持集中式提示词中心统一编辑管理，团队提示词随团队 git 分发
- **迁移到新设备**：设置页新增「迁移到新设备」导出向导，新机导入时自动识别迁移包，支持身份替换或归并到当前身份，API 密钥可加密导出后在新设备恢复
- **运行环境离线安装**：安装包内置 Python 3.13 与 Node.js 24 LTS，首启后台静默导入，完全离线可用；版本列表支持离线兜底，联网失败不再空列表
- **算力预置更新通知**：config-center 预置配置更新时推送通知，可逐项审阅差异、决定采用上游或保留本地设置，用户自定义参数不被静默覆盖
- **文件打开方式**：聊天中的文件支持「打开方式」下拉菜单，可选择应用内打开、系统默认应用、其他应用打开或在文件夹中定位
- **套餐折扣展示**：付费套餐卡片支持原价划线与折扣标签，限免套餐显示限免标识
- **上下文超限透明自愈**：模型上下文超限时自动学习真实窗口上限并持久化，当次重试即可自愈；大输出动态预算一次成型，无需多步续写
- **Provider 并发控制**：支持为单个模型服务商设置并发请求上限，超出排队等待，解决多子 Agent 同时调用同一服务商导致超限失败的问题
- **扫描件遇非视觉模型自动提示切换**：非视觉模型处理纯扫描件 PDF 时直接提示用户切换模型，不再陷入无限重试循环
- **对话 PDF 导出改客户端渲染**：对话导出 PDF 改为客户端本地渲染，远程 standalone 部署下也可正常使用
- **算力服务页显示预置版本号**：算力配置页面头部显示当前预置数据版本号，方便排查配置同步问题
- **应用与服务页面手动刷新**：服务目录页新增手动刷新按钮，支持主动拉取最新服务列表
- **数据导出添加身份识别**：数据导出包含身份元数据，导入时检测来源身份与本机是否匹配并给出告警

## 修复

- 修复中止查询后全局并发槽泄漏，累计 3 次中止后所有 Agent 卡死的问题
- 修复系统休眠/会话重置两条路径同样导致并发槽泄漏
- 修复排队中的查询无法中止，点击停止后仍继续执行并可能产生不可逆副作用
- 修复 via-messaging 同一目标并发委派/消息/求助时，回复被错误匹配导致静默拿到错误结果
- 修复异步 send_message 回执缺少关联 ID，跨路径错配导致同步等待方被错误 resolve

- 修复定时任务 maxTurns 为 1 导致 Agent 只能回复一句随即被截断、从未完成任务

- 修复会话中模型相对时间漂移，长时间对话中「X分钟后」及调度时间持续偏离真实时间

## 改进
- 委派子任务同步等待超时从 5 分钟放宽为 30 分钟，超时不再中止子会话，改为提示调用方主动决策
- WorkflowRun 工具默认不设超时，改为可选参数由 Agent 控制
- 定时任务执行轮数从 1 轮放开至与普通聊天相同，真正边界由 Agent 自行决定
- 定时任务失败后自动指数退避（30s→1h），连续失败触发告警
- 深度思考档位持久化到 agent.json，重启后不丢失
- OpenAI/Anthropic 双格式思考参数按厂商方言自动适配，兼容国产网关
- compat 未声明方言的网关按识别方言全量注入思考开启参数
- Codex 响应模型 off 档显式发送关闭 reasoning 参数
- provider 级 requiredClientVersion 运行时门槛，不支持的 provider 提示升级客户端
- 上下文裁剪改为 cheapest-first 级联（截短→删轮→折叠→突破软地板），消除截完仍超的死锁
- 系统提示词底座精简去重，移除无效的 spawn_agent 能力声明
- Agent 配置算力能力增强：API 密钥只写不读（HTTP 响应掩码 + 文件工具保护 + Bash 黑名单）
- 新增 config-center reload 端点，前端可手动刷新校验算力配置
- Claude 订阅 provider 卡片显示订阅层级与连接状态，支持取消与退出登录
- 内置 git 随包安装，不需系统安装 git 即可使用 Agent 快照等 git 功能
- 运行时目录统一到 ~/.desirecore，老用户自动迁移
- 视频生成中转升级为 OpenAI Video API 格式，接入豆包 Seedance 2.0
- 算力服务模型列表按文本/图片/视频生成分区展示，标注计费模式与价格
- 聊天消息区滚动条加宽命中区，方便鼠标拖拽
- Ls 工具默认免审批直接执行，OS 权限拒绝时返回引导文案而非裸错误
- 市场仓库源保存不再等待整轮同步，避免卡「保存中」与仓库重复显示
- 全仓覆盖写统一改为原子写（临时文件+rename），消除崩溃半写损坏
- agent.json 读写统一收敛为 per-agent 锁，并发修改排队应用而不是报 409
