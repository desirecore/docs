# DesireCore v10.0.69

**发布日期**: 2026-05-29

![v10.0.69 更新概览](/img/changelog/v10.0.69.png)

## 新功能
- **智能体创建方式升级**：创建智能体时可直接手动填写名称、人设、原则和配色，或委派 DesireCore 在对话中可见地帮你完成创建。
- **消息编辑支持图片管理**：编辑已发送消息时，现在可以删除原有图片、补充新图片、粘贴图片，并支持全屏预览。
- **注册与找回密码体验增强**：注册时新增确认密码和密码强度提示，同时支持通过邮箱验证码完成忘记密码后的重置流程。
- **邮箱验证码验证登录**：注册后的邮箱验证改为输入 6 位验证码，验证成功后会自动登录，无需再返回登录页。
- **云端模型分组更直观**：登录后资源面板会立即显示云端合作伙伴分组并固定置顶，支持在分组标题处直接查看同步状态和手动刷新。
- **图片发送体积控制**：发送图片时会自动压缩并控制单次体积，超出限制时会明确提示，避免图片过大导致发送失败或上下文被图片挤占。

## 修复
- 修复了使用 DeepSeek 自定义网关时，部分推理模型可能因角色不兼容而无法正常对话的问题。
- 修复了没有更新日志时，版本更新提示框过窄导致底部操作按钮显示不全的问题。
- 修复了登录框和资料编辑框点到空白区域时会意外关闭、导致已填写内容丢失的问题。
- 修复了云端模型同步中的软状态被误判为失败，导致反复弹出错误提示或错误显示“加载失败”的问题。

## 改进
- 首页左上角“+”菜单中的“新对话”已改为“选择对话”，减少入口含义歧义。
- 心跳通知不再混入对话消息流，而是统一通过顶部横幅展示，查看更清爽且不同智能体之间互不干扰。
- 所有涉及“免费算力/额度”的界面现在都会补充合作伙伴说明，让算力来源更加透明。
- 云端合作伙伴相关文案与分组名称已同步优化，展示信息更准确一致。
