# DesireCore v10.0.3

**发布日期**：2026-03-07

![v10.0.3 更新概览](/img/changelog/v10.0.3.png)

## 新功能

- **技能导入**：支持从本地文件夹直接导入技能，降低接入门槛
- **文件引用传递**：followUp 路径现在支持图片与文件引用的传递，多轮对话中文件上下文更连贯

## 修复

- 修复了 Agent 详情页切换引擎后表单不即时更新的问题
- 修复了 InputArea 组件 setFileRefs 未定义导致的错误
- 修复了旧版 Agent 数据中 version 字段为字符串格式时的兼容性问题
- 修复了 V1 路径下用户上传图片被静默丢弃的问题
- 修复了刷新后用户消息显示原始 XML 标签的问题

## 改进

- 文件引用状态从组件本地状态迁移到全局 store，多处共享更一致
