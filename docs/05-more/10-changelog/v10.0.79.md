# DesireCore v10.0.79

**发布日期**: 2026-06-28

![v10.0.79 更新概览](/img/changelog/v10.0.79.png)

## 新功能
- **官方算力模型接入与消耗展示**：恢复官方算力模型接入，模型选择器现在可展示与默认模型对比的综合消耗倍率徽章和单价详情，能力标签过多时自动折叠。
- **对话模型智能过滤**：在对话模型选择列表和资源管理面板中，图像/视频生成类模型将被自动禁用并给出提示，避免因误选导致对话异常。

## 修复
- **流式响应稳定性**：修复了部分兼容代理流式对话因提前关闭连接导致应用崩溃的问题，现在遇到此类截断会正确提示错误并自动重试。
- **团队远程仓库弹窗**：修复了团队远程仓库连接弹窗背景显示异常、部分字段未翻译的问题，并完善了关闭交互体验。
