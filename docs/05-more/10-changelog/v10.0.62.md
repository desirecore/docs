# DesireCore v10.0.62

**发布日期**: 2026-05-19

![v10.0.62 更新概览](/img/changelog/v10.0.62.png)

## 修复
- 解决了 Agent 在处理 Git 冲突时“保留云端版本”在远程已删除文件的场景下失败的问题，现在这类冲突可以正确完成并继续提交。
- 修复了未跟踪文件被误判为合并冲突的问题，避免对话中弹出错误的冲突解决窗口。
- 修复了冲突解决后自动提交偶发失败的问题，避免出现文件已删除却仍被重复加入提交而导致报错。
- 修复了关闭冲突对比界面时偶发报错的问题，冲突处理流程更加稳定。

## 改进
- 优化了桌面端和 Web 模式下部分依赖的加载方式，减少启动或运行时因组件资源缺失导致的异常。
