# DesireCore v10.0.24

**发布日期**: 2026-04-06

![v10.0.24 更新概览](/img/changelog/v10.0.24.png)

## 新功能
- **错误日志面板**：活动记录中新增“错误日志”标签页，您可以集中查看系统运行中的报错统计、按级别筛选错误并分页浏览历史问题。
- **工具搜索**：智能体现可使用新的工具搜索能力，能更快查找可用工具并查看完整参数说明，提升复杂任务中的工具选择效率。

## 修复
- Agent 回复中的 `.md` 相对路径链接现在会按对应工作目录正确解析，点击文档链接不再容易出现 404。
- 修复了自动重试导致的重复重试、错误误判和异常中断问题，使对话在限流、网络波动或上游服务异常时表现更稳定。
- 修复了 WYSIWYG 编辑模式下拖拽宽度时可能触发异常分屏的问题，编辑器布局更加正常。

## 改进
- Grep 搜索工具现可搜索隐藏文件、自动排除版本控制目录、正确显示上下文内容，并提供更清晰的计数、排序和分页信息，让代码与文件检索结果更准确易读。
- 对话中的重试状态提示进行了细化，遇到限流、网络异常或服务暂时不可用时，您能看到更明确的等待原因。
- 系统底层依赖与安全性得到更新，整体运行稳定性、兼容性与安全防护有所提升。
