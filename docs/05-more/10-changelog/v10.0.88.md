# DesireCore v10.0.88

**发布日期**: 2026-07-16

![v10.0.88 更新概览](/img/changelog/v10.0.88.png)

## 新功能
- **团队身份感知**：Agent 现在能识别自己是组长还是成员，对话时自动获知角色与团队结构，协同任务更高效。
- **消息编辑支持增删文档引用**：编辑带文档的消息时，可以自由删除或更换引用的文档，与图片编辑能力对齐。
- **网络状态感知与自动恢复**：应用新增网络健康检测，断网时正在执行的任务将自动挂起重试、不再直接失败；界面会实时显示网络异常提示与恢复状态。
- **算力环境增强**：运行环境页面新增 macOS Homebrew 检测与展示；内置的 npm 包管理器现可被 Agent 识别和调用；Git 工具支持按版本自动择优选择或手动切换来源，且不会再因未装开发者工具而误弹安装窗口。
- **Agent 互操作开放**：团队队长可以直接让 Agent 修改队员的设定文件，无需硬性代码限制。
- **回撤体验升级**：检查点覆盖更完整，现在可以对历史的任意分段消息直接内联回退，操作更加灵活。
- **消息时间提示**：鼠标悬停在消息时间上会显示完整的日期、时、分、秒信息。

## 修复
- **应用升级重启后智能体列表空白**：修复了自动升级后首次启动列表永久为空且无法自动恢复的问题。
- **Agent 输出中途静默停止**：修复了因“承诺句”断流、对话流意外截断、向用户提问超时等原因导致回复毫无征兆终止的问题。
- **暂停与继续对话异常**：修复了暂停后历史对话被清空、未发出的消息丢失、继续时回答错话题等一系列逻辑错乱。
- **发送消息后工具被误清导致退化复读**：修复了委派消息后工具列表被清空，导致 Agent 陷入循环复读同一句话的问题。
- **委派审批永久卡死**：修复了委派 Agent 执行需审批操作时父会话一直等待、无法继续的问题。
- **工具执行错误修复**：修复了 macOS 上因文件监听耗尽系统资源导致 Bash/Grep 等集体报错、打包版应用中 grep 每次必失败、以及查找命令误报全部命令不存在等多处工具执行故障。
- **团队数据导入/导出**：修复了跨设备导入团队时成员私有记忆、偏好丢失以及运行记录归属错误的问题。
- **通知系统**：修复了点击系统通知后应用内依然显示未读、角标不消除的问题。
- **编辑器保存快捷键**：修复了使用 Ctrl+S / ⌘S 保存文件时仅更新界面状态、未真实写入磁盘的致命缺陷。
- **界面与提示**：修复了暗色模式下团队设置面板白底白字不可读、网关错误提示误判为“违反使用政策”、更新版本号带后缀时无法正常检测、以及回撤列表错误显示「手动」标签等多个体验问题。

## 改进
- **工具错误熔断策略**：工具连续多次失败时，不再粗暴终止，而是先引导 Agent 更换方案，若无效再向用户求助，最后以优雅收尾结束当前回合。
- **访问越界提示**：当 Agent 尝试访问工作目录范围外的文件时，拒绝提示会明确告知可以通过申请授权来临时获得访问权限。
- **审批模式尊重与增强**：Agent 的审批模式设置（如“全部放行”、“仅外部工具审批”）现在被严格遵循，不再全部弹出审批卡片。
- **邮箱服务启动稳定性**：修复了应用升级重启后邮件页面误报“服务未启动”、账号列表显示为空的问题。
- **回撤数据完整性**：修复了回撤后合并视图出现重复消息、聊天记录搜索命中重复条目以及记忆索引滞后的多项数据缺陷。
- **团队同步加固**：团队 Git 同步更加可靠，能及时感知远端新提交，并在合并冲突时自动备份本地改动，避免静默覆盖。
- **界面交互优化**：统一了模型编辑对话框的能力标签为完整的中文图标列表；技能安装位置弹窗支持滚动并优化了视觉样式；个人画像与原则编辑改回整页 Markdown 模式，避免复杂格式被意外丢弃。
