# DesireCore v10.0.11

**发布日期**: 2026-03-17

## 新功能
- **邮件规则自动处理**：邮件规则现在支持对新邮件自动回复，或直接触发智能体进行处理，让 Gmail、Outlook 和 IMAP 邮箱中的来信可以自动流转。
- **聊天图片文件化传输**：现在发送图片时会先上传为媒体文件再引用使用，聊天中的图片加载更稳定，也支持智能体继续识图和处理工具返回的图片。
- **流式 Token 实时显示**：在智能体回复过程中，你可以实时看到本轮对话的 Token 消耗变化，而不必等到回复结束后再查看。
- **聊天记录单条删除**：聊天记录弹窗现在支持按条删除消息，删除后对应内容也会同步从对话视图中移除。
- **SuperDoc 保存按钮**：SuperDoc 工具栏新增保存按钮，你可以更直接地将当前文档修改写入文件。
- **内测专用模型供应商**：新增内测专用供应商及一组预置模型，符合条件的环境下可直接使用更多测试模型。

## 修复
- 修复了聊天记录中"定位到聊天位置"有时跳转不准确的问题，现在可以更可靠地定位到对应消息。
- 修复了 SuperDoc 中的 `.md` 文件链接在应用重启或刷新后点击报错的问题，历史对话里的文档链接可正常打开。
- 修复了删除智能体后列表偶尔仍显示已删除项的问题，智能体列表会更快与实际状态保持一致。
- 修复了接口审计中输入 Token 经常显示为 0 的问题，现在启用缓存时的实际输入消耗也会被正确统计和展示。
- 修复了邮件规则触发智能体处理时的路由导入问题，避免部分自动处理流程异常。

## 改进
- 智能体资料读取方式已统一，相关配置内容的展示与读取更加一致稳定。
- Agent 版本信息与 AgentFS 规范版本已分离，版本显示和后续市场更新识别会更清晰准确。
- 输入框已恢复正确的平台识别逻辑，在 macOS 等系统上的快捷键提示表现更符合当前设备。
