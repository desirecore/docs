# DesireCore v10.0.41

**发布日期**: 2026-04-28

![v10.0.41 更新概览](/img/changelog/v10.0.41.png)

## 新功能
- **Alpha 内测通道**：设置中新增 Alpha 内测版更新通道，切换时会进行双重风险确认，帮助你在明确风险后体验更前沿的版本。

## 修复
- 修复了部分采用分离式接口地址的模型服务在媒体能力调用上的异常问题，现在像 MiniMax 这类聊天与图片、语音、视频接口分开的服务可正常使用相关生成能力。
- 修复了预置模型服务配置升级后部分基础参数不同步的问题，现在服务地址、接口格式、展示名称等更新能正确下发到本地配置。
- 修复了当模型服务配置包含媒体接口地址时可能导致配置读取失败并出现接口报错的问题。
- 修复了部分音乐生成服务在界面中显示异常的问题，现在相关能力会以正确名称和分类展示。
