# DesireCore v10.0.35

**发布日期**: 2026-04-20

![v10.0.35 更新概览](/img/changelog/v10.0.35.png)

## 新功能
- **Windows PowerShell 内置工具**：在 Windows 上，智能体现在可以直接调用 PowerShell 执行命令与脚本，支持审批、超时控制、后台运行和大输出处理等能力。
- **输入框继续生成按钮**：当回复中断或暂停后，你现在可以直接在输入框旁一键继续生成，无需只依赖消息气泡里的入口。

## 修复
- 修复了 Windows 上 PowerShell 工具无输出、中文乱码、只读命令误触发审批，以及不同 PowerShell 版本下提示不准确的问题。
- 修复了 PowerShell、Bash 等内置工具输出中多语言字符和 emoji 可能被误删或截断的问题，查看命令结果时更完整可靠。
- 修复了切换到会话后未读消息红点有时不会立即消失的问题。
- 修复了暂停生成或清空对话后，智能体状态仍可能错误显示为“工作中”的问题。
- 修复了在部分 Windows NAT 端口占用场景下，邮件中心和工作流服务无法正常连接、请求失败或静默失效的问题。
- 修复了部分长对话在上下文已接近上限时未能及时自动压缩的问题，减少因上下文估算不准导致的溢出风险。
- 修复了模型预置合并后，已下线模型仍出现在聊天模型列表、模型顺序异常的问题。

## 改进
- **对话流式性能优化**：聊天主区域和会话列表在流式输出时更加流畅，减少了其他会话消息变化带来的卡顿和无关刷新。
- 模型列表同步逻辑进一步优化，预置模型的默认顺序更准确，聊天场景下不再显示已弃用模型。
- 智谱 GLM 及相关接入点在开启思考能力后，现在可以更稳定地展示思考过程。
- 内测模型列表已更新，新增高速默认模型，并恢复 MiniMax-M2.5 作为备选模型。
