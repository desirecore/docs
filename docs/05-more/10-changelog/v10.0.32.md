# DesireCore v10.0.32

**发布日期**: 2026-04-16

![v10.0.32 更新概览](/img/changelog/v10.0.32.png)

## 修复
- 修复了委派转交（handoff）场景下消息显示不完整的问题，转交前源智能体的流式内容、工具调用卡片以及目标智能体收到的上下文信息现在都能完整保留并正确展示。
- 修复了委派转交后消息偶尔丢失或刷新后顺序错乱的问题，现在对话消息会基于真实消息时间稳定排序，历史缺失时间戳的消息也能正常显示在正确位置。
- 修复了算力服务中手动修改模型能力标签后显示和实际识别不一致的问题，现在自定义的能力配置会正确显示，并在发送图片等场景下被准确识别。
- 修复了通知中心部分文字颜色层级不一致的问题，现在通知时间和筛选标签在不同配色方案下的可读性更好。

## 改进
- 优化了邮件中心的整体配色与文字层级，邮件列表、详情、预览中的时间、地址、摘要以及已读头像等内容现在更清晰、更易读。
- 优化了邮件编辑与快捷回复区域的界面一致性，工具栏、引用块和错误提示的视觉风格现在更加统一。
