# DesireCore v10.0.32

**发布日期**: 2026-04-15

![v10.0.32 更新概览](/img/changelog/v10.0.32.png)

## 修复
- 修复了 Agent 委派转交（handoff）时目标对话中用户消息背景信息丢失、消息被吞掉或源 Agent 流式内容未完整保留的问题，现在转交前后的任务说明、上下文和已生成内容都能更稳定地显示出来。
- 修复了刷新或读取历史消息后对话顺序异常的问题，现在消息会按真实消息时间正确排序，长会话中的新内容也不会被排到旧位置。
- 修复了算力服务中模型能力标签读取和显示不准确的问题，手动修改后的能力配置会在模型详情、编辑弹窗和图片等能力校验中保持一致。
- 修复了通知中心部分文字颜色层级不一致的问题，时间信息和筛选标签在不同主题下更清晰易读。

## 改进
- 优化了邮件中心的整体配色与可读性，邮件列表、详情、预览、引用块和编辑工具栏的信息层级更清晰，已读头像与错误状态也更容易辨认。
