# DesireCore v10.0.26

**发布日期**: 2026-04-09

![v10.0.26 更新概览](/img/changelog/v10.0.26.png)

## 新功能
- **内置全局技能扩展**：新增 skill-creator、Word、PDF、Excel、PPT、网页访问和前端设计等 7 个内置全局技能，输入相关需求时智能体可更自然地自动调用对应能力完成文档处理、联网检索、页面设计与技能创建。
- **探索型委派预设**：智能体现在可以发起只读的“Explore”探索型子任务，更适合在大型项目中进行文件搜索、代码定位和内容分析。
- **SendUserMessage 专属消息卡片**：智能体主动发送给你的通知消息会以独立卡片展示，并支持 Markdown、附件和主动通知标签，阅读更直观。
- **运行时环境面板**：资源管理器新增运行时环境视图，你可以查看本机运行环境详情及可用工具，为使用相关能力提供更清晰的环境信息。

## 修复
- 修复了部分情况下 Token 消耗统计偏高的问题，历史活动记录、对话内统计和汇总数据现在会更准确。
- 修复了带缓存计费的模型在 Token 展示上的口径问题，缓存读写消耗现在会被单独显示，不再与普通输入重复计算。
- 修复了清除对话后活动记录消耗一并丢失的问题，现在清除聊天内容后仍可保留真实消耗记录用于审计查看。
- 修复了 SendUserMessage 消息只能在生成完成后才出现的问题，现在消息卡片会即时出现并随内容流式更新。
- 修复了部分历史消息与合并视图下 SendUserMessage 卡片渲染不完整的问题，回看对话时展示更稳定。

## 改进
- **全新品牌视觉**：应用更新了全新的品牌形象与图标资源，整体界面观感更加统一。
- 智能体配置中的模型默认项已统一收拢到更清晰的配置结构下，导入和兼容旧配置时会更顺滑。
- 对话内本次查询的消耗信息改为更清晰的分项展示，你可以更直观看到输入、输出、缓存和成本构成。
