# DesireCore v10.0.26

**发布日期**: 2026-04-09

![v10.0.26 更新概览](/img/changelog/v10.0.26.png)

## 新功能
- **内置全局技能扩充**：新增技能创建器、Word、PDF、Excel、PowerPoint、网页访问和前端设计等内置全局技能，智能体现在可以更自然地处理办公文档、联网检索、页面设计以及技能创建任务。
- **Explore 委派预设**：智能体现在可以一键委派只读型“探索”工作流，更适合进行文件搜索、代码阅读和内容分析等调研任务。
- **运行时环境管理**：资源面板新增运行时环境页签，您可以查看运行环境详情及可用工具，便于理解和管理智能体的执行环境。
- **SendUserMessage 专属消息卡片**：智能体主动发送给您的消息现在会以独立卡片展示，支持 Markdown、附件和主动通知标识，阅读更直观。
- **全新品牌视觉**：应用更新了新的品牌形象与图标，整体界面观感更统一。

## 修复
- 修复了部分模型下 Token 消耗统计不准确的问题，活动记录、会话状态和单次查询摘要中的用量显示现在更加真实可靠。
- 修复了历史对话中旧版累计式 Token 数据导致的过度统计问题，升级后旧数据也会按更合理的方式校正显示。
- 修复了清除对话后活动记录中的消耗数据一并丢失的问题，现在清除聊天内容后仍可保留真实的审计消耗记录。
- 修复了已清除的对话可能被同一智能体复用的问题，后续新对话将不会误用这些历史会话。
- 修复了 SendUserMessage 卡片必须等消息完整生成后才出现的问题，现在会在生成过程中即时显示并持续更新内容。
- 修复了中文场景下部分内置技能不易触发的问题，办公文档等相关需求现在更容易被正确识别并调用。

## 改进
- 查询摘要与运行状态面板优化了 Token 展示方式，输入、输出、缓存读写和成本信息现在区分得更清晰。
- Agent 模型配置结构进行了兼容升级，导入和读取旧版智能体配置时会更平滑，不易因配置字段变化而出错。
