# DesireCore v10.0.26

**发布日期**: 2026-04-09

![v10.0.26 更新概览](/img/changelog/v10.0.26.png)

## 新功能
- **内置全局技能扩充**：新增 skill-creator、docx、pdf、xlsx、pptx、web-access 和 frontend-design 等内置技能，用户现在可以更自然地创建技能、处理办公文档、访问网页内容和生成前端设计方案。
- **Explore 委派预设**：Agent 现在支持以 Explore 模式委派只读探索型子任务，更适合进行文件搜索、代码定位和内容分析。
- **SendUserMessage 专属消息卡片**：Agent 主动发送给用户的消息现在会以独立卡片展示，并支持 Markdown、附件和主动通知标记，阅读更清晰。
- **SendUserMessage 实时流式展示**：当 Agent 正在组织要发给你的消息时，消息卡片会立即出现并逐步实时更新内容，无需等到全部生成完毕。

## 修复
- 修复了部分场景下 Token 消耗统计被高估的问题，活动记录和查询摘要中的用量数据现在更准确。
- 修复了历史累计式 Token 凭据导致旧对话统计失真的问题，升级后旧数据也会按更合理的方式自动校正。
- 修复了清除对话后活动记录 Token 数据一并丢失的问题，现在删除聊天内容后仍会保留真实消耗用于审计。
- 修复了清除后的旧会话可能被同一 Agent 复用的问题，新的对话会正确从新会话开始。

## 改进
- **Token 展示升级**：查询摘要和运行状态面板现在会更清晰地区分输入、输出、缓存读取、缓存写入和成本，便于理解每次查询的实际消耗。
- **品牌视觉更新**：应用图标与部分界面视觉已更新为全新品牌样式，整体识别度更统一。
- **Agent 模型配置兼容升级**：Agent 的模型默认配置结构已优化，导入和读取旧配置时也能保持兼容，减少配置迁移带来的影响。
