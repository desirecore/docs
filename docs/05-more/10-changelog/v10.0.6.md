# DesireCore v10.0.6

**发布日期**: 2026-03-09

![v10.0.6 更新概览](/img/changelog/v10.0.6.png)

## 新功能
- **统一委派工具**：新增 `delegate` 统一工具替代原有三个独立委派工具，支持同步、异步和交接三种模式；异步委派在后台子会话中完成任务并自动回传结果到调用方下次对话
- **OpenClaw 导入**：支持从 OpenClaw 平台一键扫描并导入智能体和技能，提供双 Tab 选择界面和实时导入进度
- **多数据源心跳**：同一智能体支持追加多个数据源，新增数据源管理面板和独立开关控制
- **心跳监控面板**：智能体详情页新增心跳状态卡片、历史记录和级别筛选，安静时段结束后自动合并汇报
- **注册表统一**：Registry 目录结构统一为 `entries/` 格式，安装流程改为 Agent 驱动，支持设备感知卸载和安装进度显示

## 修复
- 修复心跳调度与设置面板联动问题：支持 UUID 调度查找、禁用改为暂停保留配置、开关变更后面板实时更新
- 修复 OpenClaw 导入的 CORS 跨域和缓存失效问题
- 修复 button 嵌套导致的 hydration 错误
- 修复 Registry 条目格式兼容：从新格式正确回退派生旧字段

## 改进
- 委派消息在目标智能体聊天中显示发送方头像和名称，区分人类用户与 Agent 委派
- CI release 创建改为幂等模式，避免多 job 并发时创建冲突
