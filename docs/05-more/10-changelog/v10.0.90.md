# DesireCore v10.0.90

**发布日期**: 2026-07-18

## 新功能
- **智能体管理工具**：新增内置的智能体管理工具，Agent 现在可以自主创建、更新、删除和查询智能体，无需依赖已被实例鉴权封堵的 HTTP API 调用路径；配套的市场技能也同步更新，使用新工具替代原有的 curl 调用方式

## 修复
- **沉浸式模式记忆面板显示虚假内容**：修复了"我记得你"卡片在全新 Agent 从未产生对话时仍显示虚构内容（如"你喜欢先听结论"）的问题，现在正确接入真实记忆数据，无记忆时显示"尚无工作记忆"
- **超大扫描 PDF 读取失败**：修复了读取大尺寸扫描 PDF 时，因图片体积超出发给模型网关的请求体字节上限（如 6MB）而导致失败的问题，现在会自动压缩图片、在超限时学习上限并自动重试
- **Anthropic 兼容网关下工具调用永久失败**：修复了使用 Anthropic 兼容网关时，因 thinking 块回传判断逻辑错误导致 Worker 子会话中的 Write、Edit 等工具调用始终返回 400 错误的问题
