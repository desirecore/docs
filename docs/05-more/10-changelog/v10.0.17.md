# DesireCore v10.0.17

**发布日期**: 2026-03-29

## 新功能
- **首次使用引导**：首次启动应用时会显示欢迎引导页，您可在开始使用前查看使用条款、隐私政策以及匿名统计说明。
- **智能体快速创建**：现在可以在首页通过弹窗填写名称和描述，直接由 AI 自动生成智能体配置并创建新智能体。
- **团队协作系统**：新增团队管理与组织关系能力，您可以创建团队、配置组长与成员，并在关系图谱中查看团队结构。
- **智能体消息通信**：智能体之间新增轻量级消息通信能力，除任务委派外还可进行通知、提问、汇报和求助。
- **本机接口请求工具**：新增内置 HttpRequest 工具，智能体调用本机 API 时无需依赖 Bash 或 curl，跨平台使用更稳定。
- **灰度更新通道**：设置中新增更新通道切换，您可以在稳定版与测试版之间切换，并自动选择更合适的下载源。
- **聊天 Mermaid 图表**：聊天中的 Mermaid 代码块现在可以直接渲染为图表，而不再只是显示源码。
- **文档原生导出**：SuperDoc 现已支持通过系统保存对话框导出 DOCX，并支持直接导出高质量 PDF。

## 修复
- 异步委派在对方仍在处理中时不再被提前显示为“已完成”，状态展示更准确。
- 修复了聊天中点击纯链接会让整个应用窗口跳转到外部网页的问题，现在会改为在系统浏览器中打开。
- 修复了 macOS 关闭窗口后重新打开可能崩溃的问题，也修复了窗口重建时协议重复注册导致的崩溃问题。
- 修复了 Windows 下数据导入失败、导入过程文件被占用以及删除智能体后目录残留的问题。
- 修复了活动记录页中不同智能体头像显示不正确的问题，并修正了部分统计数据与刷新时机不准确的情况。
- 修复了使用 API 代理工具时，非图片类工具结果偶尔错误显示为“正在分析图片”的问题。
- 修复了聊天中部分 Mermaid 图表在刷新历史消息后不显示、渲染失败或只显示源码的问题。
- 修复了 Markdown 渲染引擎在处理特定内容时可能导致消息区域异常的问题，异常情况下也能正常查看文本内容。
- 修复了 SuperDoc 导出 PDF 时可能导出整个应用界面或导错编辑器内容的问题，现在只会导出当前文档。
- 修复了保存大文件时可能出现失败的问题，导出和保存大体积内容更加稳定。
- 修复了文档编辑器全屏模式下工具栏空白区域无法拖拽窗口的问题。
- 修复了团队标识提示浮层被裁切、成员名称显示为 ID 等问题，团队结构信息更易读。
- 修复了模型选择器在默认模型解析失败时可能找不到可用聊天模型的问题。

## 改进
- **自动化模板体验**：优化了自动化模板页面的亮色模式可读性、分类横向滚动、预览图标显示以及返回导航体验。
- 文档编辑器优化了标题栏与工具栏布局，减少重复信息，并统一了全屏下的窗口拖拽体验。
- 模型选择器现在会直接显示当前默认模型名称，切换模型时更直观。
- 活动记录中的“API 调用”文案统一调整为“工具调用”，页面表述更清晰。
- 默认聊天模型与回退策略得到优化，在模型配置变化时更容易自动选中可用模型。
- 核心智能体内置技能名称和部分默认配置进行了本地化与更新，系统内置智能体更易理解。
