# DesireCore v10.0.78

**发布日期**: 2026-06-28

![v10.0.78 更新概览](/img/changelog/v10.0.78.png)

## 修复
- 修复了在 Windows 上通过 `dc://` 协议或快捷方式启动应用时可能出现的初始化失败问题。
- 修复了智能体编写文档时编辑器提前弹出并显示不完整内容的问题：现在文档会在整轮对话结束后再自动打开，确保打开时内容已完整。同时，流式生成期间差异对比的“接受/拒绝”按钮已被禁用，避免了中途操作导致的内容错乱。
