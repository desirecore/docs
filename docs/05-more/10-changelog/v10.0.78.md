# DesireCore v10.0.78

**发布日期**: 2026-06-28

![v10.0.78 更新概览](/img/changelog/v10.0.78.png)

## 修复
- 修复了 Windows 上通过 `dc://` 协议链接、桌面快捷方式或开始菜单打开应用时可能启动失败的问题。
- 修复了 AI 写文档时编辑器过早弹出并显示半成品内容的问题，现在会等到整轮回复完成后再自动打开；同时修复了在 AI 生成过程中，文档差异对比的接受/拒绝按钮可能提前结束编辑并导致内容错乱的问题，流式期间这些按钮将自动隐藏。
