# DesireCore v10.0.78

**发布日期**: 2026-06-16

![v10.0.78 更新概览](/img/changelog/v10.0.78.png)

## 修复
- Windows 上通过 `dc://` 深链、桌面/开始菜单快捷方式或关联文件启动应用时，启动失败的问题已修复，应用打开更稳定。
- 使用 Write 工具生成 Markdown 文档时，新文档会在整轮内容写完后再自动打开，避免先看到未完成的半成品。
- SuperDoc 在智能体流式编辑文档期间将暂时禁用 diff 的接受/拒绝操作，避免提前写回内容导致后续编辑错乱。

## 改进
- SuperDoc 在文档流式生成过程中仍会持续展示实时 diff 变化，让你在等待生成完成时也能清楚看到内容修改轨迹。
