# DesireCore v10.0.16

**发布日期**: 2026-03-24

## 新功能
- **云端发布**：现在可以直接在应用内把 Agent 一键发布到已绑定的 GitHub、Gitee 或 Gitea，并查看仓库链接、同步状态和版本历史。
- **所见即所得 Markdown 编辑**：Markdown 编辑器默认升级为更完整的所见即所得模式，支持更丰富的可视化编辑能力和 Ribbon 工具栏。
- **委派协作体验升级**：Agent 委派任务时现在支持更清晰的任务移交、异步结果回传和临时子任务 Worker，让多 Agent 协作过程更直观。

## 修复
- 发送图片后，模型现在能更稳定地识别图片内容，连续发图也能区分新旧图片，且刷新后图片不会再丢失。
- 修复了社会化关系页面中智能体在线状态显示不准确的问题，现在与会话列表保持一致并可实时更新。
- 修复了 Windows/Linux 下 Markdown 编辑器全屏时工具栏与窗口控制按钮重叠的问题，并补充了可拖拽区域。
- 修复了委派卡片在亮色主题下边框不明显的问题，同时头像显示和子任务状态展示也更加准确。
- 修复了云端发布与同步中的多种认证和连接兼容问题，包括首次发布未真正推送、旧连接失效或缺失时无法同步等情况。

## 改进
- Markdown 编辑器现已跟随应用全局主题自动同步，切换深浅色时界面表现更统一。
- 委派结果现在会自动写回原对话并实时更新状态，完成后还能收到桌面通知，跟进任务更省心。
- 云端同步按钮会根据当前状态智能选择发布更新或先同步再发布，减少不必要的操作步骤。
