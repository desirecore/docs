# DesireCore v10.0.29

**发布日期**: 2026-04-13

![v10.0.29 更新概览](/img/changelog/v10.0.29.png)

## 新功能
- **团队导入导出**：现在支持在多个入口直接导入或导出整个团队，并可按智能体、技能、记忆、工作区等分类选择需要传输的内容。
- **内置技能自动检查更新**：内置技能会每 10 分钟自动检查 Market 新版本，无需重启应用也能获取最新更新。
- **审计详情复制**：在审计详情页查看系统指令时，已可一键复制对应内容，便于排查、留档和复用。

## 修复
- 对话在重试、编辑后重发或消息合并场景下的消息丢失、重复分隔线和顺序异常问题已修复，聊天记录显示更稳定可靠。
- 当模型服务出现限流、网关异常或临时网络波动时，系统会更准确地自动重试，并向你展示更清晰的错误原因与重试状态。
- 修复了图片上传过程中仍可发送导致图片丢失的问题，发送时机会更安全。
- 修复了图片与文件引用在部分环境下识别不正确的问题，图片内容现在能更稳定地被作为视觉上下文传递给模型。
- 修复了 Windows 上导出 ZIP 可能损坏或失败后残留无效文件的问题，导出过程更可靠。
- 修复了导入入口对智能体包和团队包识别不一致的问题，选错入口时也会给出更明确的引导提示。
- 修复了团队导入后列表未及时刷新的问题，导入完成后内容可更快正确显示。
- 修复了更新技能时可能触发异常的界面问题，技能更新流程更稳定。
- 修复了话题完成与历史处理中的部分异常情况，单个话题失败不再影响其他内容继续处理。
- 修复了将 DesireCore 核心智能体设为团队组长的错误操作，避免团队结构被错误配置。

## 改进
- 用户消息卡片现已支持复制与朗读，你发送过的内容查看和复用更方便。
- 聊天中的文件附件现在可以直接点击打开，内置支持的格式会使用更合适的文档或代码查看方式展示。
- “需要你确认/回答”的卡片去掉了跳过按钮的等待时间，并提升了按钮可见性，交互更直接。
- 智能体主动通知和发送给你的用户消息会按更符合触发顺序的方式展示，对话流转更自然。
- 全局内置技能调整为按需参与调用，减少无关干扰，提升智能体执行任务时的聚焦度。
